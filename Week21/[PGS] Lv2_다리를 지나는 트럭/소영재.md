### 문제 설명 (요약)
- 일차선 다리를 여러 대의 트럭이 정해진 순서대로 건너려고 한다.
- 다리에는 최대 bride_length대의 트럭이 올라갈 수 있으며, 다리 위에 올라가 있는 트럭들의 무게 합은 최대 하중 weight를 넘을 수 없다.
- 트럭은 1초마다 다리에서 한 칸씩 이동한다.
- 모든 트럭이 다리를 완전히 건너는 데 필요한 최소 시간을 구하는 문제이다.

### 입력
- int bridge_length : 다리의 길이
- int weight : 다리가 견딜 수 있는 최대 무게
- int[] truck_weights : 다리를 건너야 하는 트럭들의 무게가 순서대로 담긴 배열

### 출력
- 모든 트럭이 다리를 건너는 데 걸리는 최소 시간을 반환

<hr>

### 핵심 풀이 전략
1. 다리 위의 상태를 Queue로 표시하고, 1초마다 다음 과정을 반복한다
2. 매초 큐의 맨 앞 값을 제거하고 현재 무게에서 뺀다.
3. 다음 트럭을 올릴 수 있으면 큐에 추가하고, 불가능하면 빈 칸을 의미하는 0을 추가한다.
4. 모든 트럭이 진입할 때까지 반복한 뒤, 마지막 트럭의 통과 시간인 bridge_length를 더한다.

---

```java
import java.util.*;
class Solution {
    public int solution(int bridge_length, int weight, int[] truck_weights) {
        int time = 0;
        int currentWeight = 0;
        int truckIndex = 0;
        
        Deque<Integer> queue = new ArrayDeque<>();
        
        for(int i = 0; i < bridge_length; i++){
            queue.offer(0);
        }
        
        while (truckIndex < truck_weights.length){
            time ++;
            
            currentWeight -= queue.poll();
            
            int nextTruck = truck_weights[truckIndex];
            
            if(currentWeight + nextTruck <= weight){
                queue.offer(nextTruck);
                currentWeight += nextTruck;
                truckIndex++;
            } else {
                queue.offer(0);
            }
        }
        
        return time + bridge_length;
    }
}
```

### 삽질 기록 🧠
- 마지막 트럭이 빠져나가기 위한 bridge_length를 더하지 않아서 오류가 났었다.
