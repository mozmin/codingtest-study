### 문제 설명 (요약)
- 각 칸에 색깔이 지정된 2차원 격자 board가 주어진다.
- 선택한 위치 (h,w)를 기준으로 위, 아래, 왼쪽, 오른쪽에 인접한 칸을 확인하여, 현재 칸 board[h][w]와 같은 색깔인 칸의 개수를 반환한다.

### 입력
- String[][] board : 각 칸의 색깔이 문자열로 저장된 정사각형 2차원 배열
- int h : 선택한 칸의 행
- int w : 선택한 칸의 열

### 출력
- board[h][w]의 위, 아래, 왼쪽, 오른쪽에 위치한 칸 중에서 현재 칸과 같은 색깔을 가진 칸의 개수를 반환한다.

<hr>

### 핵심 풀이 전략
1. 현재 위치 (h, w)의 상하좌우를 확인하기 위해 방향 배열을 사용한다.
2. 각 방향을 탐색하면서 다음 두 가지를 확인한다.
- 이동한 좌표가 배열의 범위 안에 있는지 확인한다.
- 이동한 칸의 색깔이 현재 칸의 색깔과 같은지 확인한다.
3. 범위 안에 있으면서 색깔까지 같다면 count를 1 증가시킨다.
4. 최종적으로 count를 반환한다.

---

```java
class Solution {
    int[] dx = {-1, 0, 1, 0};
    int[] dy = {0, 1, 0, -1};
    public int solution(String[][] board, int h, int w) {
        int count = 0;
        
        String color = board[h][w];
                
        for(int i = 0; i < 4; i++){
                int nextX = dx[i];
                int nextY = dy[i];
                    if (w + nextX >= 0 && w + nextX < board.length && h + nextY >= 0 && h + nextY < board[0].length) {
                        if(color.equals(board[h+nextY][w+nextX])){
                            count++;
                        }
                    }
                }
        return count;
    }
}
```
