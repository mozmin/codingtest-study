### 문제 설명 (요약)
각 칸마다 색이 칠해진 2차원 격자 보드판에서 특정 칸 `(h, w)`를 골랐을 때, 상하좌우로 인접한 칸 중 고른 칸과 같은 색깔로 칠해진 칸의 개수를 구하는 문제입니다. 

---

### 핵심 풀이 전략
1. **방향 벡터 설정**: 상, 하, 좌, 우 4방향을 탐색하기 위해 `dy` (또는 `dh`), `dx` (또는 `dw`) 배열을 선언합니다.
2. **경계 조건 확인 (Bounds Check)**: 인접한 칸의 좌표 `(ny, nx)`가 보드판의 범위를 벗어나지 않는지(`0 <= 좌표 < n`) 먼저 검사하여 `ArrayIndexOutOfBoundsException`을 방지합니다.
3. **문자열 동등성 비교**: 자바에서 색상 이름(문자열)을 비교할 때는 `==`가 아닌 `.equals()` 메서드를 사용하여 값 자체를 비교합니다.

---

### 내가 구현한 구현 코드
```java
class Solution {
    
    int[] dy = {1, -1, 0, 0};
    int[] dx = {0, 0, 1, -1};
    public int solution(String[][] board, int h, int w) {
        int answer = 0;
        
        int n = board.length;
        String color = board[h][w];
        
        for(int i = 0; i < 4; i++){
            int ny = h + dy[i];
            int nx = w + dx[i];
            
            if(ny >= 0 && nx >= 0 && ny < n && nx < n && board[ny][nx].equals(color)) answer++;
        }
        return answer;
    }
}
```
