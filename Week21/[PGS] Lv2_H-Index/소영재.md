### 문제 설명 (요약)
- 과학자가 발표한 논문의 인용 횟수가 담긴 배열 citations가 주어진다.
- 논문 n편 중에서 h번 이상 인용된 논문이 h편 이상일 때, 조건을 만족하는 h의 최댓값을 구한다.

### 입력
- int[] citations : 각 논문의 인용 횟수가 담긴 1차원 정수 배열

### 출력
- H-Index의 최댓값을 반환한다.

<hr>

### 핵심 풀이 전략
1. HashMap을 사용하여 인용 횟수별 논문 개수를 저장한다.
2. H-Index 후보 i를 0부터 전체 논문 수까지 검사한다.
3. Map.Entry를 순회하며 i회 이상 인용된 논문의 수를 계산한다.
4. i회 이상 인용된 논문이 i편 이상이면 H-Index의 후보가 된다.
5. 조건을 만족하는 값 중 가장 큰 값(answer)을 반환한다.

---

```java
import java.util.*;
class Solution {
    public int solution(int[] citations) {
        
        Map<Integer, Integer> map = new HashMap<>();
        int answer = 0;
        
        for(int i = 0; i < citations.length; i++){
            int index = citations[i];
            map.put(index, map.getOrDefault(index, 0) +1);
        }
        
        for(int i = 0; i <= citations.length; i++){
            int highCount = 0;
            
            for(Map.Entry<Integer, Integer> entry : map.entrySet()){
                int citation = entry.getKey();
                int count = entry.getValue();
                
                if(citation >= i){
                    highCount += count;
                }
            }
            
            if(highCount >= i){
                answer = Math.max(answer,i);
            }
        }
        
        return answer;
    }
}
```

### 배운 점 && 보완할 점
- Map.Entry를 사용하면 Key와 Value를 함께 꺼낼 수 있다는 점을 배웠다.
- 또한, Map을 순회할 때 다음과 같이 entrySet()을 사용할 수 있다.
- for(Map.Entry<Integer, Integer> entry : map.entrySet()){..}
- map.entrySet()은 Map에 저장된 각각의 key-value쌍을 하나의 Entry 형태로 반환한다.
