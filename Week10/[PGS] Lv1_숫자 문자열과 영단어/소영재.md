### 문제 설명 (요약)
- 숫자와 영단어가 섞인 문자열에서 영단어를 숫자로 변환하는 문제

### 입력
- String s : 숫자와 영단어가 섞인 문자열

### 출력
- 영단어를 숫자로 변환한 후, 최종 정수를 반환

<hr>

### 핵심 풀이 전략
- 1. 숫자에 해당하는 영단어를 담는 String[] 배열을 선언한다.
- 2. 반복문을 돌면서 해당 단어와 일치하는 값을 replace()메서드를 사용하여 대응하는 숫자로 치환한다.
- 3. 숫자로 이루어진 문자열을 정수로 변환하여 반환한다.
---

```java
class Solution {
    public int solution(String s) {
        
    String[] str = {"zero","one","two","three","four","five","six","seven","eight","nine"};

    for (int i = 0; i < str.length; i++) {
        s = s.replace(str[i], String.valueOf(i));
    }

    return Integer.parseInt(s);
        
    }
}
```
