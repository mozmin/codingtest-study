### 문제 설명
문자열 s가 주어질 때, 일부 숫자는 자릿수 그대로 들어 있고 일부는 영어 단어로 적혀 있다. <br>
이 문자열을 전부 숫자로 변환한 뒤, 최종 정수를 반환하는 문제이다. <br>

### 입력
- s: 숫자와 영단어가 섞여 있는 문자열

### 출력
- 변환된 정수값


### 핵심 풀이 전략
- 문자열을 앞에서부터 한 글자씩 확인했다.
- 현재 문자가 숫자라면 바로 StringBuilder에 추가했다.
- 숫자가 아니라면, 해당 문자를 기준으로 어떤 영단어가 시작되는지 판별해서 대응되는 숫자를 추가했다.
- 한 번 숫자 단어를 처리한 뒤에는, 그 단어 길이만큼 인덱스를 추가로 이동시켜 이미 확인한 문자를 다시 보지 않도록 했다.
- 최종적으로 만들어진 숫자 문자열을 Integer.parseInt()로 정수로 변환해 반환했다.


### 제출 코드
```java
import java.util.*;

class Solution {
    public int solution(String s) {
        StringBuilder sb = new StringBuilder();
        for(int i = 0; i < s.length(); i++){
            if(Character.isDigit(s.charAt(i))){
                sb.append(s.charAt(i));
            }
            switch(s.charAt(i)){
                case 'z' : sb.append("0");
                    i+=3;
                    break;
                case 'o' : sb.append("1");
                    i+=2;
                    break;
                case 't' :
                    if(i+1 <s.length() && s.charAt(i+1) == 'w') {
                        sb.append("2");
                        i+=2;
                        break;
                    }
                    else {
                        sb.append("3");
                        i+=4;
                        break;
                    }
                case 'f':
                    if(i+1 < s.length() && s.charAt(i+1) == 'o'){
                        sb.append("4");
                        i+=3;
                        break;
                    }
                    else {
                        sb.append("5");
                        i+=3;
                        break;
                    }
                case 's':
                    if(i+1 < s.length() && s.charAt(i+1) == 'i'){
                        sb.append("6");
                        i+=2;
                        break;
                    }
                    else {
                        sb.append("7");
                        i+=4;
                        break;
                    }
                case 'e' :
                    sb.append("8");
                    i+=4;
                    break;
                case 'n':
                    sb.append("9");
                    i+=3;
                    break;
            }
        }
        String string = sb.toString();
        int answer = Integer.parseInt(string);
        return answer;
    }
}

```
