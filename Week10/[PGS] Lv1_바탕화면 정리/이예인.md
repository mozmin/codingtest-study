### 문제 설명
문자열 배열 wallpaper가 주어지고, "."은 빈칸, "#"은 파일이 있는 칸을 의미한다.<br>
파일들을 한 번의 드래그로 모두 선택해 삭제하려고 할 때, 모든 파일을 포함하면서 이동 거리가 최소가 되는 드래그 시작점과 끝점을 구하는 문제이다. <br>
드래그는 왼쪽 위 점 (lux, luy)에서 오른쪽 아래 점 (rdx, rdy)로 이루어지며, 정답은 [lux, luy, rdx, rdy] 형태로 반환한다.<br>

### 입력
- wallpaper: 바탕화면 상태를 나타내는 문자열 배열
- "." : 빈칸
- "#" : 파일

### 출력
- 모든 파일을 한 번에 선택할 수 있는 최소 드래그 범위
- 형식: [lux, luy, rdx, rdy]


### 핵심 풀이 전략
- 바탕화면 전체를 순회하면서 #가 있는 모든 위치를 찾는다.
- 파일들의 위치 중에서
- 가장 작은 행 번호 / 가장 작은 열 번호 / 가장 큰 행 번호 / 가장 큰 열 번호
- 를 구하면, 모든 파일을 포함하는 최소 직사각형 범위를 만들 수 있다.
- 시작점은 (최소 행, 최소 열) 이고, 끝점은 (최대 행 + 1, 최대 열 + 1) 이다.
- 문제의 드래그 끝점은 파일이 있는 마지막 칸 자체가 아니라 그 바깥 점이어야 하기 때문이다.


### 제출 코드
```java
class Solution {
    public int[] solution(String[] wallpaper) {
        int[] answer = new int[4];
        int min_x = wallpaper.length;
        int min_y = wallpaper[0].length();
        int max_x = 0;
        int max_y = 0;
        for(int i = 0 ; i < wallpaper.length; i++){
            for(int j = 0 ; j < wallpaper[0].length(); j++){
                if(wallpaper[i].charAt(j) == '#'){
                    if(i < min_x) min_x = i;
                    if(i > max_x) max_x = i;
                    if(j < min_y) min_y = j;
                    if(j > max_y) max_y = j;
                }
            }
        }
        answer = new int[] {min_x, min_y, max_x+1, max_y+1};
        return answer;
    }
}

```

### 삽길 기록 🧠
- 헷갈렸던 부분은 +1 처리였다. maxRow, maxCol는 마지막 파일 칸의 위치이고, 문제에서 요구하는 끝점 (rdx, rdy)는 그 칸을 감싸는 오른쪽 아래 꼭짓점 좌표이므로 +1이 필요했다.
