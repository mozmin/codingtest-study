### 문제 설명 (요약)
- 바탕화면을 나타내는 문자열 배열 wallpaper가 주어진다.
- 각 칸은 빈칸('.'), 파일이 있는 위치('#')을 나타낸다.
- 이때 파일들이 있는 모든 위치를 한 번에 포함할 수 있도록 드래그해야 하는 가장 작은 직사각형의 범위를 구한다.

### 입력
- String[] wallpaper : 컴퓨터 바탕화면의 상태를 나타내는 문자열 배열

### 출력
- 파일을 한 번에 삭제하기 위해 최소한의 이동거리를 갖는 드래그의 시작점과 끝점을 담은 정수배열

<hr>

### 핵심 풀이 전략
1. 드래그의 시작과 끝 좌표를 구하기 위해 startX, startY, endX, endY를 초기화하고 선언한다.
2. 2중 반복문을 돌면서 현재 위치의 문자가 ('#')인지 확인한다.
3. Math.min과 Math.max로 끝 좌표를 갱신하면서 전체 영역의 최소 사각형 범위를 구한다.
4. 끝 좌표를 포함하지 않아야 함으로 최대 행과 최대 열에 각각 +1을 처리를 해준다.
---

```java
import java.util.*;
class Solution {
    public int[] solution(String[] wallpaper) {
        
        //최소, 최대값을 비교하기 위한 초기값 세팅
        int startX = Integer.MAX_VALUE;
        int startY = Integer.MAX_VALUE;
        int endX = 0;
        int endY = 0;
        
        int[] answer = new int[4];
        
        // #을 만날 때마다 최소, 최대값 비교 및 갱신
        for(int i = 0; i < wallpaper.length; i++){
            char[] arr = wallpaper[i].toCharArray();
            for(int j = 0; j < arr.length; j++){
                 if(arr[j] == '#'){
                    startX = Math.min(startX, i);
                    startY = Math.min(startY, j);
                    endX = Math.max(endX, i);
                    endY = Math.max(endY, j);
                 }
            }
        }
        answer[0] = startX;
        answer[1] = startY;
        answer[2] = endX + 1;
        answer[3] = endY + 1;

        return answer;
    }
}
```

### 삽질 기록 🧠
- endX, endY부분에서 +1을 처리하지 않아서 틀렸었다.

---

### 배운 점 && 보완할 점
- 처음에는 char[] arr = wallpaper[i].toCharArray(); split()메서드를 활용하여 처리하였다.
- 근데 split() 메서드가 시간이 오래걸린다는 점이 생각났다. 
- 해당 문제는 단일 문자 단위로만 비교하면 되므로, split()보다 효율적인 toCharArray()를 사용하여 char[]로 변환하는 방식으로 개선하였다.
