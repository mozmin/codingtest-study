### 문제 설명 (요약)
각 스테이지의 실패율을 계산한 뒤 실패율이 높은 스테이지부터 내림차순으로 정렬하는 문제.  
실패율은 특정 스테이지에 도달했으나 아직 클리어하지 못한 사용자 수를 해당 스테이지에 도달한 전체 사용자 수로 나눈 값으로 정의. 
실패율이 같을 경우 스테이지 번호가 작은 순으로 정렬.

### 입력
- N: 전체 스테이지 수  
- stages: 각 사용자가 현재 도달한 스테이지 번호가 담긴 배열

### 출력
실패율이 높은 스테이지부터 정렬된 스테이지 번호 배열을 반환
