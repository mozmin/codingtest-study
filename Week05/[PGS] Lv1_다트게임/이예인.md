### 문제 설명
다트 게임 결과 문자열이 주어졌을 때, 총 점수를 계산하는 문제이다.<br>
각 기회마다 점수와 보너스(S, D, T)가 주어지고, 옵션(*, #)에 따라 현재 점수와 이전 점수가 함께 변할 수 있다.<br>
- S : 1제곱
- D : 2제곱
- T : 3제곱
- \* : 해당 점수와 바로 이전 점수를 각각 2배
- \# : 해당 점수를 마이너스 처리

문자열을 앞에서부터 차례대로 읽으면서 현재 점수와 이전 점수를 잘 관리하는 것이 핵심이다.

### 입력
다트 결과를 나타내는 문자열 dartResult

### 출력
다트 게임의 최종 점수

### 핵심 풀이 전략
문자열을 한 글자씩 순회하면서 처리했다.<br>

숫자가 나오면 현재 점수로 저장하고, 문자가 S, D, T라면 현재 점수에 거듭제곱을 적용했다.<br>

이후 *, # 옵션이 나올 수 있으므로 현재 점수(curResult)와 이전 점수(prevResult)를 함께 관리했다. <br>

- 이전 점수 관리
* 옵션은 현재 점수뿐 아니라 이전 점수에도 영향을 준다.<br>
그래서 각 라운드가 끝날 때마다 이전 점수를 저장해두는 방식으로 처리했다.

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(String dartResult) {
        int answer = 0;
        int prevResult = 0;
        int curResult = 0;
        int dartResultInt = 0;
        for(int i = 0 ; i < dartResult.length(); i++){
            // 숫자
            if(Character.isDigit(dartResult.charAt(i))){
                dartResultInt = Character.getNumericValue(dartResult.charAt(i));
                if(dartResultInt == 1 && Character.isDigit(dartResult.charAt(i+1))){
                    curResult = 10;
                    i++;
                }
                else curResult = dartResultInt;
                continue;

            }
            // S, D, T
            if(dartResult.charAt(i) == 'S'){
                answer += curResult ;
            }
            if(dartResult.charAt(i) == 'D'){
                curResult *= curResult;
                answer += curResult;
            }
            else if(dartResult.charAt(i) == 'T'){
                curResult = curResult * curResult * curResult;
                answer += curResult;
            }
            // *, #
            else if(dartResult.charAt(i) == '*'){
                answer += curResult;
                answer += prevResult;
                curResult = curResult * 2;
            }
            else if(dartResult.charAt(i) == '#'){
                answer -= 2 * curResult;
                curResult = -curResult;
            }
            
            if(i!=dartResult.length()-1 && Character.isDigit(dartResult.charAt(i+1))){
                prevResult = curResult;
            }
        }       
        
        return answer;
    }
}
```

### 배운 점 && 보완할 점
- `Character.isDigit()`을 사용하면 특정 문자가 숫자인지 여부를 쉽게 확인할 수 있다. 문자열을 한 글자씩 순회하면서 숫자와 문자를 구분해야 하는 구현 문제에서 유용하게 활용할 수 있다.

- `Character.getNumericValue()`를 사용하면 `char` 형태의 숫자를 정수 값으로 변환할 수 있다. 문자열에서 숫자를 추출해 연산해야 할 때 `parseInt()`를 사용하지 않고 간단하게 정수로 변환할 수 있어 편리하다.
