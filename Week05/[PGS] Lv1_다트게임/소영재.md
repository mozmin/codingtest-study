### 문제 설명 (요약)
- 다트를 3번 던져 점수와 점수 영역, 옵션에 따라 점수를 계산한 뒤, 최종 점수를 반환한다.

### 입력
- String dartResult : 점수|보너스|옵션으로 이루어진 문자열 3세트

### 출력
- 다트를 3번 던져 점수와 점수 영역, 옵션에 따른 총 점수를 반환
<hr>

### 핵심 풀이 전략
- 다트를 던지는 횟수는 3번으로 고정되어 있으므로 3의 크기를 가진 score 정수 배열을 생성한다.
- 지금 몇 번째 다트를 던지고 있는지 확인해야 하므로 해당 다트의 위치를 가르키는 변수를 선언한다.
- dartResult 문자열을 돌면서 정수인지 판별한다. 이때, 10을 만날 경우 1, 0으로 판단될 수 있기 때문에 해당 조건을 처리해준다.
- 이후 조건문에 따라 score 값을 변경시킨다.
---

```java
import java.util.*;

class Solution {
    public int solution(String dartResult) {
        //다트를 던져 얻는 점수를 저장하는 배열
        int[] score = new int[3];
        //현재 몇 번째 다트인지 가르키는 변수 (-1부터 시작해야 배열에 접근 가능)
        int dart = -1;

        for (int i = 0; i < dartResult.length(); i++) {
            char c = dartResult.charAt(i);

            // 숫자를 만난다면 다트를 던진 것이므로 증가
            if (Character.isDigit(c)) {
                dart++;

                // 10 처리
                if (c == '1' && dartResult.charAt(i + 1) == '0') {
                    score[dart] = 10;
                    // 두 글자를 사용했기 때문에 i값을 증가
                    i++;
                } else {
                    //숫자를 넣어야 하기 때문에 char -> String -> Int로 변환
                    score[dart] = Integer.parseInt(String.valueOf(c));
                }
            }
            
            else if (c == 'S') {
                score[dart] = (int)Math.pow(score[dart], 1);
            }
            else if (c == 'D') {
                score[dart] = (int)Math.pow(score[dart], 2);
            }
            else if (c == 'T') {
                score[dart] = (int)Math.pow(score[dart], 3);
            }
            
            else if (c == '*') {
                score[dart] *= 2;
                if (dart > 0) {
                    score[dart - 1] *= 2;
                }
            }
            else if (c == '#') {
                score[dart] *= -1;
            }
        }

        return score[0] + score[1] + score[2];
    }
}
```

### 삽질 기록 🧠
- 문자열을 나누는데 정수 처리를 어떻게 해야할지 고민했다.
- 처음에는 Integer.parseInt(c); 형태로 사용했는데 계속 오류가 났다.
- Integer.parseInt()는 매개변수로 받는 형식이 String으로 변환해줘야 한다는 것을 AI를 통해 알게 되었다.
- 최근에 푸는 문제들 난이도가 내가 느끼기엔 어려워서 시간이 엄청 오래 걸리고 있는데 더 열심히 고민하고 문제를 풀다보면 나아지지 않을까.. 싶다

### 배운 점 && 보완할 점
- 문자를 정수인지 판별할 수 있는지 판단하는 isDigit()라는 메서드를 배우게 되었다.
- 제곱 계산을 해주는 편의 메서드가 있을 것 같아 찾아보니 Math.pow(a,b)형태로 제곱을 편하게 도와주는 메서드도 배우게 되었다.
