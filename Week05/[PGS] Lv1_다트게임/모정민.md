### 문제 설명 (요약)
* 다트 게임의 점수 계산 로직을 구현하는 문제.
* 총 3번의 기회마다 점수(0~10), 보너스(S, D, T), 옵션(*, #)이 주어짐.
* **보너스:** 점수를 각각 1제곱(S), 2제곱(D), 3제곱(T)으로 계산.
* **스타상(\*):** 해당 점수와 바로 직전 점수를 각각 2배로 만듦. (첫 번째 기회일 경우 해당 점수만 2배)
* **아차상(#):** 해당 점수를 마이너스로 만듦.

### 입력
* `String dartResult`: "점수|보너스|[옵션]"이 3세트 이어진 문자열 (예: `1S2D*3T`)

### 출력
* 3번의 기회에서 얻은 최종 점수의 총합 (int)

<hr>

### 핵심 풀이 전략

* **가변 길이 숫자 파싱**: 점수가 0~10점이므로, '10'을 두 글자가 아닌 하나의 숫자로 온전히 인식하기 위해 `String temp` 변수를 사용. 숫자가 아닐 때까지 문자를 누적함.
* **최근 데이터 관리 (Stack)**: 스타상(`*`) 처럼 "직전의 점수"를 다시 꺼내서 수정해야 하므로, 후입선출(LIFO) 특성을 가진 스택 자료구조(`ArrayDeque`)를 활용.
* **조건부 연산**: 보너스(S, D, T)가 등장하는 시점을 한 세트의 점수가 확정되는 시점으로 잡고 계산된 값을 스택에 삽입.

---

### 핵심 풀이 전략

```java
import java.util.*;

class Solution {
    
    int answer;
    Deque<Integer> stack;
    public int solution(String dartResult) {
        answer = 0;
        
        stack = new ArrayDeque<>();
        
        String temp = "";
        for(int i = 0; i < dartResult.length(); i++){
        
            char c = dartResult.charAt(i);
            
            if(Character.isDigit(c)){
                temp += c;
            }else if(c == 'S' || c == 'D' || c == 'T'){
                int point = Integer.parseInt(temp);
                bonus(c, point);
                
                temp = "";
            }
            // 옵션
            else{
                option(c);
            }
            
        }
        // 스택에 남은 모든 점수 합산
        while (!stack.isEmpty()) {
            answer += stack.pop();
        }
        
        
        return answer;
    }
    
    public void bonus(char c, int point){
        
        if(c == 'S'){
            stack.push((int) Math.pow(point, 1));
        }else if( c== 'D'){
            stack.push((int) Math.pow(point, 2));
        }else{
            stack.push((int) Math.pow(point, 3));
        }
    }
    
    public void option(char c){ 
        
        // 스타상
        if(c == '*'){
            
            int curPoint = stack.pop();
            if(!stack.isEmpty()){
                int exPoint = stack.pop();
                stack.push(exPoint * 2);
            }
            stack.push(curPoint * 2);        
        }
        else if(c == '#'){
            int curPoint = stack.pop();
            stack.push(-curPoint);
        }
    }
}
```

### 삽질 기록 🧠
- for문으로 1부터 하나씩 charAt() 쓰는 것까지는 했는데, 그 뒤로 해결법이 생각이 안나서 AI의 도움을 받았다.
- 1과 10을 구분하는 법을 이번에 확실히 알게 되었다.
- 스텍을 이전 값 저장용으로만 쓰려고 했다. 현재 값은 바로 저장 안함.
---

### 배운 점 && 보완할 점
- temp를 활용하여 1과 10을 구분하는 방법을 확실히 알게 되었다.
- 자바 문법 `Character.isDigit(c)` 를 알게 되었다.
- 옵션까지 모두 받은 뒤에 보너스와 점수를 계산하려고 했는데, 그것보다는 보너스와 점수는 바로바로 계산하고, 계산된 점수를 기반으로 옵션을 활용하는게 훨씬 좋은 방법이었다.
