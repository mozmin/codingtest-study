### 문제 설명
구매한 로또 번호 6개와 당첨 번호 6개가 주어진다.<br>
이때 알아볼 수 없는 번호는 0으로 표시되어 있으며, 0이 어떤 번호인지에 따라 최고 순위와 최저 순위가 달라질 수 있다.<br>
- 최고 순위: 0인 번호들이 모두 당첨 번호와 일치한다고 가정했을 때
- 최저 순위: 0인 번호들이 전부 불일치한다고 가정했을 때
최종적으로 [최고 순위, 최저 순위]를 반환하는 문제이다.<br>

### 입력
- lottos: 민우가 구매한 로또 번호 6개
- win_nums: 이번 주 당첨 번호 6개

### 출력
- 최고 순위와 최저 순위를 담은 정수 배열

### 핵심 풀이 전략
이 문제의 핵심은 두 가지를 세는 것이다. <br>

1. 확실히 맞은 번호 개수(minCount)
- lottos의 값이 0이 아니면서 win_nums와 같은 경우를 센다.
- 이 값은 곧 최저 순위를 결정하는 기준이 된다.

2. 알 수 없는 번호 개수(zeroCount)
- lottos에서 0의 개수를 센다.
- 이 값들을 모두 맞았다고 가정하면 최고 순위를 구할 수 있다.

최저 맞은 개수 = minCount <br>
최고 맞은 개수 = minCount + zeroCount <br>

그리고 로또 순위는 맞힌 개수에 따라 결정된다. <br>

코드에서는 7 - 맞힌 개수로 순위를 계산했고, <br>
0개 맞힌 경우 예외 처리를 위해 count == 0일 때 1로 바꿔서 최종적으로 6등이 나오도록 처리했다. <br>

### 제출 코드
```java
import java.util.*;

class Solution {
    public int[] solution(int[] lottos, int[] win_nums) {
        int[] answer = new int[2];
        int minCount = 0;
        int zeroCount = 0 ;
        for(int i = 0 ; i < 6; i++){
            // 번호가 0인 경우
            if(lottos[i] == 0){
                zeroCount++;
                continue;
            }
            for(int j = 0 ; j < 6; j++){
                // 번호를 맞춘 경우
                if(lottos[i] == win_nums[j]){
                    minCount++;
                }   
            }
        }
        
        int maxCount = minCount + zeroCount;
        
        if(minCount == 0) minCount = 1;
        if(maxCount == 0) maxCount = 1;
        
        int maxGrade = 7 - maxCount;
        int minGrade = 7 - minCount;
        
        answer[0] = maxGrade;
        answer[1] = minGrade;
        
        return answer;
    }
}
```
