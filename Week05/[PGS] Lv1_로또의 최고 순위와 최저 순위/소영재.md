### 문제 설명 (요약)
- 구매한 로또와 정답 번호를 비교하여 등수를 계산한다.
- 구매한 로또 중에 0이 있다면, 해당 번호를 당첨된 번호로 가정하고 계산한다.
- 0을 포함한 구매 로또에서 최대 등수와 최소 등수를 계산하여 반환한다.

### 입력
- int[] lottos : 로또 번호가 들어있는 정수 배열
- int[] win_nums : 당첨 번호가 들어있는 정수 배열

### 출력
- 최대 등수, 최소 등수가 들어있는 int[] 배열 반환
<hr>

### 핵심 풀이 전략
- 구매한 로또의 번호가 0인 경우는 당첨 된 것으로 간주하고 문제에 접근
- 리턴할 정수 배열에 실제 번호가 일치했던 횟수를 최소 등수로 가정, 일치했던 횟수와 0이 나왔던 횟수를 더해 최대 등수로 가정하였다.
---

```java
class Solution {
    public int[] solution(int[] lottos, int[] win_nums) {
        //정답을 담을 정수 배열
        int[] result = new int[2];
        //로또와 당첨번호가 일치하는지 저장하는 정수 배열
        int[] correctCount = new int[6];
        //맞은 횟수를 세는 변수
        int count = 0;
        
        //만약 구매한 로또의 번호가 0이라면 correctCount 배열에 2를 저장
        for(int i = 0; i < lottos.length; i++){
            if(lottos[i] == 0){
                correctCount[i] = 2;
            }
            //번호가 일치한다면 correctCount 배열에 1을 저장
            for(int j = 0; j < win_nums.length; j++){

                if(lottos[i] == win_nums[j]){
                    correctCount[i] = 1;
                    count++;
                }
            }
        }
        //최소 등수 반환
        result[1] = score(count);
        //0을 만났던 횟수만큼 count를 증가
        for(int i = 0; i < lottos.length; i++){
            if(correctCount[i] == 2){
                count++;
            }
        }
        //최대 등수 반환
        result[0] = score(count);
        return result;
    }
    // 등수 계산 메서드
    public int score(int count) {
        if(count == 6) return 1;
        if(count == 5) return 2;
        if(count == 4) return 3;
        if(count == 3) return 4;
        if(count == 2) return 5;
        return 6;
    }
}
```
