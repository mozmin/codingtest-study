### 제출 일자

### 문제 설명
여러 개의 신호등이 있고, 각 신호등은 초록불, 노란불, 빨간불이 일정한 시간만큼 반복된다.
모든 신호등이 동시에 노란불이 되는 가장 빠른 시간을 구하는 문제이다.
만약 동시에 노란불이 되는 시점이 없다면 -1을 반환한다.

### 입력
- signals[i][0]: i번째 신호등의 초록불 지속 시간
- signals[i][1]: i번째 신호등의 노란불 지속 시간
- signals[i][2]: i번째 신호등의 빨간불 지속 시간

### 출력
- 모든 신호등이 동시에 노란불이 되는 가장 빠른 시간
- 없으면 -1

### 핵심 풀이 전략
- 각 신호등의 한 주기 길이 (G + Y + R)를 먼저 구했다.
- 그리고 각 신호등마다 주기 길이만큼 배열을 만들어, 노란불인 구간만 1, 나머지 초록불/빨간불 구간은 0으로 표시했다.

- 이후 모든 신호등이 다시 같은 패턴으로 반복되는 시점은 각 주기의 최소공배수까지 확인하면 충분하다고 판단했다.
- 따라서 최소공배수 범위 안에서 시간을 1씩 증가시키며, 각 시간마다 모든 신호등이 노란불인지 확인했다.

- 각 시간 i에서 signals_color[j][i % signals_length[j]] 값을 확인하면 해당 신호등의 현재 상태를 알 수 있다.

- 모든 신호등의 값이 1이면 그 시점이 정답이다.

### 제출 코드
```java
class Solution {
    public int solution(int[][] signals) {
        
        // 각 신호등의 길이 합 (G+Y+R)
        int[] signals_length = new int[signals.length];
        for(int i = 0 ; i < signals.length; i++){
            for(int j = 0 ; j < 3; j++){
                signals_length[i] += signals[i][j];
            }
        }
        
        // 각 신호등 길이만큼의 2차원 배열 만들기
        int[][] signals_color = new int[signals.length][];
        for(int i = 0; i < signals.length; i++){
            signals_color[i] = new int[signals_length[i]];
        }
        
        // 각 신호등 값 넣기 Y -> 1, G/R -> 0
        for(int i = 0 ; i < signals.length; i++){
            int count = 0; 
            for(int j = 0 ; j < 3; j++){
                // Y
                if(j == 1) {
                    int colorNum = signals[i][j];
                    for(int k = count; k < count + colorNum; k++){
                        signals_color[i][k] = 1;
                    }
                    count += colorNum;
                }
                // G, R
                else {
                    int colorNum = signals[i][j];
                    for(int k = count; k < count + colorNum; k++){
                        signals_color[i][count] = 0;
                    }
                    count += colorNum;
                }
            }
        }       
              
        //최소공배수 구하기
        int max_num = signals_length[0];
        for(int i = 1 ; i < signals_length.length; i++){
            max_num = lcm(max_num, signals_length[i]);
        }
                
        // 최소공배수 만큼 신호 돌기
        for(int i = 0 ; i < max_num; i++){
            int kk = 1;
            for(int j = 0 ; j < signals.length; j++){
                kk *= signals_color[j][i%signals_length[j]];
                if(kk == 0) break;
            }
            if(kk == 1) return i+1;
        }
        
        return -1;
    }
    
    // 최대공약수 - 유클리드 호제법
    int gcd(int a, int b){
        while(b != 0){
            int r = a % b;
            a = b;
            b = r;
        }
        return a;
    }
    
    // 최소공배수
    int lcm(int a, int b){
        return a * b / gcd(a, b);
    }
}

```

### 배운 점 && 보완할 점
2차원 배열을 만들 때 각 행 길이가 서로 다를 수 있다는 점 때문에
- new int[signals.length][고정길이] 방식이 아니라
- new int[signals.length][]로 만든 뒤 각 행을 따로 생성해야 한다는 점도 다시 확인했다.

반복문 안의 break가 어디까지 종료되는지도 헷갈렸는데, 가장 안쪽 반복문 하나만 종료된다는 점을 정리할 수 있었다.
