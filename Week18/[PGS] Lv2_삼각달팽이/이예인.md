### 문제 설명
- 정수 n이 주어졌을 때, 높이가 n인 삼각형 모양에 숫자를 달팽이 모양으로 채우는 문제이다.

- 숫자는 1부터 시작해서 아래 방향, 오른쪽 방향, 왼쪽 위 대각선 방향 순서로 이동하며 채운다.
- 모든 숫자를 채운 뒤, 삼각형의 위쪽 행부터 아래쪽 행까지 순서대로 읽어 1차원 배열로 반환한다.

### 입력
- n: 삼각형의 높이

### 출력
- 삼각 달팽이 형태로 숫자를 채운 뒤, 위에서 아래로 읽은 1차원 정수 배열

---

### 핵심 풀이 전략
- 총 숫자의 개수는 1 + 2 + ... + n이므로 n * (n + 1) / 2이다.
- 숫자를 채우기 위해 n x n 크기의 2차원 배열을 사용한다.
- 이동 방향은 총 3가지이다.
  - 아래 방향
  - 오른쪽 방향
  - 왼쪽 위 대각선 방향
- 방향은 i % 3을 기준으로 결정한다.
- 각 방향으로 이동하는 횟수는 n, n-1, n-2, ... , 1처럼 점점 줄어든다.
- 시작 위치를 row = -1, col = 0으로 두면 첫 번째 아래 이동 시 (0, 0)에 숫자 1을 넣을 수 있다.
- 배열을 모두 채운 뒤, arr[i][j]에서 j <= i인 삼각형 영역만 answer 배열에 담는다.

---

### 제출 코드
```java
class Solution {
    public int[] solution(int n) {
        int size = n * (n + 1) /2;
        int[] answer = new int[size];
        
        int[][] arr = new int[n][n];
        
        int row = -1;
        int col = 0;
        int num = 1;
        
        for(int i = 0 ; i < n; i++){
            for(int j = i ; j < n; j++){
                if(i % 3 == 0){ // 아래로 이동
                    row++;
                } 
                else if (i % 3 == 1){ // 오른쪽으로 이동
                    col++;
                }
                else{ // 위로 이동
                    row--;
                    col--;
                }
                arr[row][col] = num;
                num++;
            }
        }
        
        int k = 0;
        for(int i = 0 ; i < n; i++){
            for(int j = 0 ; j <= i; j++){
                answer[k] = arr[i][j];
                k++;
            }
        } 
        return answer;
    }
}
```
