### 문제 설명 (요약)
- 각 직원은 출근 희망 시각을 정하고, 해당 시각으로부터 10분 이내에 출근하면 정상 출근으로 인정된다.
- 단, 토요일과 일요일의 출근 기록은 이벤트 판정에서 제외한다.
- 일주일 동안 평일에 모두 지각하지 않은 직원 수를 구한다.

### 입력
- int[] schedules : 직원별 출근 희망 시각이 담겨있는 1차원 정수 배열
- int[][] timelogs : 직원들이 일주일 동안 출근한 시각이 담긴 2차원 정수 배열
- int startday : 이벤트를 시작한 요일을 의미하는 정수

### 출력
- 이벤트 조건을 만족한 직원 수

<hr>

### 핵심 풀이 전략
1. 각 직원의 희망 출근 시각에 10분을 더해 허용 출근 시각을 계산한다.
2. 직원별로 7일간의 출근 기록을 확인한다.
3. staryday와 반복문의 j를 활용해 현재 날짜의 요일을 계산한다.
4. 토요일과 일요일이면 출근 시간을 검사하지 않고 건너뛴다.
5. 평일 출근 시간이 허용 시각보다 늦으면 해당 직원은 실패 처리한다.
6. 실패하지 않은 직원 수를 세어 반환한다.

```java
import java.util.*;
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        boolean[] fail = new boolean[schedules.length];
        
        Map<Integer, String> map = new HashMap<>();
        
        map.put(1, "Mon");
        map.put(2, "Tue");
        map.put(3, "Wed");
        map.put(4, "Thu");
        map.put(5, "Fri");
        map.put(6, "Sat");
        map.put(7, "Sun");
        
        for(int i = 0; i < schedules.length; i++){
            int target = changeTime(schedules[i]);
            
            for(int j = 0; j < timelogs[0].length; j++){
                int day = (startday + j -1) % 7 +1;
                
                if(map.get(day).equals("Sat") || map.get(day).equals("Sun")){
                    continue;
                }
                int conditionTarget = timelogs[i][j];
                if(conditionTarget<= target){
                    continue;
                } else {
                    fail[i] = true;
                    break;
                }
            }
        }
        
        int answer = 0;
        for(int i = 0; i < fail.length; i++){
            if(!fail[i]){
                answer++;
            }
        }
        return answer;
    }
    
    public int changeTime(int target){
        int hour = target/100;
        int min = (target%100) +10;
        
        if(min >= 60){
            hour += 1;
            min %= 60;
        }
        
        if(hour >= 24){
            hour %= 24;
        }
        
        return hour * 100 + min;

    }
}
```

### 배운 점 && 보완할 점
- 굳이 Map을 활용하지 않아도 된다.
```java
import java.util.*;

class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        int answer = 0;

        for (int i = 0; i < schedules.length; i++) {
            int target = changeTime(schedules[i]);
            boolean fail = false;

            for (int j = 0; j < timelogs[i].length; j++) {
                int day = (startday + j - 1) % 7 + 1;

                if (day == 6 || day == 7) {
                    continue;
                }

                if (timelogs[i][j] > target) {
                    fail = true;
                    break;
                }
            }

            if (!fail) {
                answer++;
            }
        }

        return answer;
    }

    public int changeTime(int target) {
        int hour = target / 100;
        int min = target % 100 + 10;

        if (min >= 60) {
            hour++;
            min -= 60;
        }

        return hour * 100 + min;
    }
}
```
