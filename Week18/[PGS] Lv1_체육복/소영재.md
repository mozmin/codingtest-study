### 문제 설명 (요약)
- 체육복을 도난당한 학생과 여벌 체육복을 가진 하갯ㅇ이 주어진다.
- 여벌 체육복은 바로 앞번호 또는 뒷번호 학생에게만 빌려줄 수 있다.
- 체육수업을 들을 수 있는 학생의 최댓값을 구한다.

### 입력
- int n : 전체 학생 수
- int[] lost : 체육복을 도난당한 학생 번호
- int[] reserve : 여벌 체육복을 가진 학생 번호 배열

### 출력
- 체육수업을 들을 수 있는 학생의 최댓값

<hr>

### 핵심 풀이 전략
1. lostSet에 체육복을 잃어버린 학생을 저장한다.
2. reserve를 순회하면서 도난도 당하고 여벌도 있는 학생은 lostSet에서 제거한다.
3. 실제로 빌려줄 수 있는 학생만 reserveSet에 저장한다.
4. lost를 정렬한 뒤, 작은 번호부터 확인한다.
5. 왼쪽 학생(-1)이 빌려줄 수 있으면 먼저 빌리고, 안 되면 오른쪽(+1) 학생에게 빌린다.
6. 마지막에 체육복을 빌리지 못한 (수업을 들을 수 없는) 학생을 뺀 값을 반환한다.

---

```java
import java.util.*;
class Solution {
    public int solution(int n, int[] lost, int[] reserve) {
        
        Set<Integer> lostSet = new HashSet<>();
        Set<Integer> reserveSet = new HashSet<>();
        
        for(int i = 0; i < lost.length; i++){
            lostSet.add(lost[i]);
        }
        
        for(int i = 0; i < reserve.length; i++){
            if(!lostSet.contains(reserve[i])){
                reserveSet.add(reserve[i]);
            } else {
                lostSet.remove(reserve[i]);
            }
        }
        
        Arrays.sort(lost);
        
        for(int i = 0; i < lost.length; i++){
            if (!lostSet.contains(lost[i])) {
                continue;
            }
            
            if(reserveSet.contains(lost[i]-1)){
                lostSet.remove(lost[i]);
                reserveSet.remove(lost[i]-1);
            } else if(reserveSet.contains(lost[i]+1)){
                lostSet.remove(lost[i]);
                reserveSet.remove(lost[i]+1);
            }
        }
               
        return n - lostSet.size();
    }
}
```

### 삽질 기록 🧠
- 2~3개의 테스트 케이스에서 계속 오류가 났다. lost 배열이 정렬되어 있지 않으면 순서가 꼬일 수 있어서 생긴 문제였다. (당연히 작은 순일 줄..)
- 반문 ex(입력 값으로 lost[4,2])

### 배운 점 && 보완할 점
1. 향상된 for문 활용
```java
import java.util.*;

class Solution {
public int solution(int n, int[] lost, int[] reserve) {

        Set<Integer> lostSet = new HashSet<>();
        Set<Integer> reserveSet = new HashSet<>();
        
        for (int l : lost) {
            lostSet.add(l);
        }
        
        for (int r : reserve) {
            if (!lostSet.contains(r)) {
                reserveSet.add(r);
            } else {
                lostSet.remove(r);
            }
        }
        
        Arrays.sort(lost);

        for (int l : lost) {
            if (!lostSet.contains(l)) {
                continue;
            }
            if (reserveSet.contains(l - 1)) {
                lostSet.remove(l);
                reserveSet.remove(l - 1);
            }
            else if (reserveSet.contains(l + 1)) {
                lostSet.remove(l);
                reserveSet.remove(l + 1);
            }
        }

        return n - lostSet.size();
    }
}
```
