### 문제 설명
- 전체 학생 수 n, 체육복을 도난당한 학생 번호 배열 lost, 여벌 체육복을 가진 학생 번호 배열 reserve가 주어진다.

- 체육복이 없는 학생은 바로 앞번호 또는 바로 뒷번호 학생에게만 체육복을 빌릴 수 있다.
- 단, 여벌 체육복을 가진 학생이 도난당한 경우에는 본인이 입어야 하므로 다른 학생에게 빌려줄 수 없다.

- 체육수업을 들을 수 있는 학생 수의 최댓값을 구하는 문제이다.

### 입력
- n: 전체 학생 수
- lost: 체육복을 도난당한 학생들의 번호 배열
- reserve: 여벌 체육복을 가진 학생들의 번호 배열

### 출력
- 체육수업을 들을 수 있는 학생 수의 최댓값

---

### 핵심 풀이 전략
- 학생별 체육복 개수를 배열로 관리한다.
- 처음에는 모든 학생이 체육복을 1벌 가지고 있다고 가정한다.
- lost에 포함된 학생은 체육복 개수를 1 감소시킨다.
- reserve에 포함된 학생은 체육복 개수를 1 증가시킨다.
- 이렇게 하면 도난당했지만 여벌도 있는 학생은 체육복 개수가 다시 1이 되므로 자연스럽게 처리된다.
- 체육복이 없는 학생을 왼쪽부터 확인하면서, 앞번호 학생에게 먼저 빌릴 수 있는지 확인한다.
- 앞번호 학생이 빌려줄 수 없다면 뒷번호 학생에게 빌릴 수 있는지 확인한다.
- 마지막으로 체육복을 1벌 이상 가진 학생 수를 세어 반환한다.

---

### 제출 코드
```java
class Solution {
    public int solution(int n, int[] lost, int[] reserve) {
        int answer = 0;
        int[] student = new int[n];
        for(int i = 0 ; i < n; i++){
            student[i] = 1;
        }
        for(int i = 0 ; i < lost.length; i++){
            student[lost[i] - 1]--;
        }
        for(int i = 0 ; i < reserve.length; i++){
            student[reserve[i] - 1]++;
        }
        
        for(int i = 0 ; i < n; i++){
            if(student[i] == 0) {
                if(i > 0 && student[i-1] == 2){
                    student[i]++;
                    student[i-1]--;
                }
                
                else if(i < n-1 && student[i+1] == 2){
                    student[i]++;
                    student[i+1]--;
                }
            }
        }
        
        for(int i = 0 ; i < n; i++){
            if(student[i] >= 1) answer++;
        }
        return answer;
    }
}
```
