### 문제 설명 (요약)
- 원판 n개를 1번 기둥에서 3번 기둥으로 옮기는 문제이다.
- 단, 한 번에 하나의 원판만 옮길 수 있고, 큰 원판은 작은 원판 위에 올릴 수 없다.
- 이동한 과정을 [출발기둥, 도착기둥] 형태로 순서대로 반환해야 한다.

### 입력
int n : 원판의 개수

### 출력
- 원판을 옮긴 순서가 담긴 2차원 정수 배열

<hr>

### 핵심 풀이 전략
1. 위에 있는 n-1개의 원판을 from에서 via로 옮긴다.
2. 가장 큰 원판 1개를 from에서 to로 옮긴다.
3. via에 있던 n-1개의 원판을 다시 to로 옮긴다.
4. 재귀 종료 조건은 n == 1 일 때이다, 원판이 1개라면 바로 from에서 to로 옮기면 된다.
---

### 구현 코드 (정답 코드 참고)
```java
import java.util.*;

class Solution {
    //원판을 옮긴 기록을 저장하는 리스트
    List<int[]> list = new ArrayList<>();

    public int[][] solution(int n) {
        
        hanoi(n, 1, 2, 3);

        int[][] answer = new int[list.size()][2];

        for (int i = 0; i < list.size(); i++) {
            answer[i] = list.get(i);
        }

        return answer;
    }

    public void hanoi(int n, int from, int via, int to) {
        if (n == 1) {
            list.add(new int[]{from, to});
            return;
        }

        hanoi(n - 1, from, to, via);

        list.add(new int[]{from, to});

        hanoi(n - 1, via, from, to);
    }
}
```
