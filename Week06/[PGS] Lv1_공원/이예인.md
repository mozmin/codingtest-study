# [Lv1] 공원산책


`#구현`

### 성능 요약

메모리:  KB, 시간:  ms <br>
시간복잡도: O(R × M)
- R: routes의 개수
- M: 한 명령에서 이동하는 최대 칸 수

### 제출 일자
2026년 3월 17일 

### 문제 설명
공원은 문자열 배열 park로 주어지며, <br>
- S : 시작 위치
- O : 이동 가능한 칸
- X : 장애물

이동 명령은 routes로 주어지고, "E 2"처럼 방향과 이동 칸 수로 이루어져 있습니다. <br>

각 명령마다 이동 후 공원 범위를 벗어나지 않아야 하고, 이동 경로 중 장애물 X가 없어야 합니다. <br>

조건을 모두 만족하는 경우에만 이동하며, 모든 명령 수행 후 최종 위치를 [행, 열] 형태로 반환하는 문제입니다. <br>

### 입력
- String[] park : 공원 정보
- String[] routes : 이동 명령 정보

### 출력
- 최종 위치를 담은 int[] 

### 핵심 풀이 전략
1. 먼저 park 전체를 탐색해서 시작점 S의 좌표를 찾습니다.
2. 각 이동 명령에 대해 방향과 이동 칸 수를 파싱합니다.
3. 이동 결과가 범위를 벗어나는지 먼저 검사합니다.
4. 범위를 벗어나지 않는다면, 실제 이동 경로를 한 칸씩 확인하면서 장애물 X가 있는지 검사합니다.
5. 중간에 장애물이 없을 때만 현재 위치를 갱신합니다.
6. 모든 명령을 처리한 뒤 최종 좌표를 반환합니다.

### 제출 코드
```java
import java.util.*;

class Solution {
    public int[] solution(String[] park, String[] routes) {

        int cur_x = -1;
        int cur_y = -1;
        String park_x;
        // S 위치 찾기
        for(int i = 0 ; i < park.length; i++){
            if(cur_x != -1) break;
            for(int j = 0 ; j < park[0].length(); j++){
                park_x = park[i];
                if(park_x.charAt(j) == 'S'){
                    cur_x = i;
                    cur_y = j;
                    break;
                }
            }
        }
        String route;
        Character dir;
        int num = 0;
        for(int i = 0 ; i < routes.length; i++){
            route = routes[i];
            dir = route.charAt(0);
            num = Character.getNumericValue(route.charAt(2));
            if(dir == 'E' && cur_y + num >= park[0].length()){
                continue;
            }
            if(dir == 'W' && cur_y - num < 0){
                continue;
            }
            if(dir == 'S' && cur_x + num >= park.length){
                continue;
            }
            if(dir == 'N' && cur_x - num < 0){
                continue;
            }
            
            boolean k = true;
            if(dir == 'E'){
                for(int j = 1 ; j < num+1; j++){
                    if(park[cur_x].charAt(cur_y+j) == 'X') k = false;
                }
                if(k == true) cur_y += num;
            }
            k = true;

            if(dir == 'W'){
                for(int j = 1 ; j < num+1; j++){
                    if(park[cur_x].charAt(cur_y-j) == 'X') k = false;
                }
                if(k == true) cur_y -= num;
            }
            k = true;
            
            if(dir == 'S'){
                for(int j = 1 ; j < num+1; j++){
                    if(park[cur_x + j].charAt(cur_y) == 'X') k = false;
                }
                if(k == true) cur_x += num;
            }
            k = true;
            
            if(dir == 'N'){
                for(int j = 1 ; j < num+1; j++){
                    if(park[cur_x - j].charAt(cur_y) == 'X') k = false;
                }
                if(k == true) cur_x -= num;
            }
            k = true;         
            
        }
        int[] answer = {cur_x, cur_y};
        return answer;
    }
}

```
