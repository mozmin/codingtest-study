### 문제 설명
주차장 요금표와 차량의 입·출차 기록이 주어질 때, <br>
각 차량별 누적 주차 시간을 계산한 뒤 요금표에 따라 최종 주차 요금을 구하는 문제이다. <br>

주의할 점은 다음과 같다. <br>
- 차량 번호가 작은 순서대로 결과를 반환해야 한다.
- 출차 기록이 없는 차량은 23:59에 출차한 것으로 간주해야 한다.
- 요금은 입출차마다 계산하는 것이 아니라, 차량별 누적 주차 시간 기준으로 한 번 계산해야 한다.

### 입력
- fees : [기본 시간, 기본 요금, 단위 시간, 단위 요금]

- records : [시각 차량번호 IN/OUT] 형식의 문자열 배열

### 출력
- 차량 번호가 오름차순인 차량들의 주차 요금을 담은 int[]

### 핵심 풀이 전략
1. 입차 시간 저장용 HashMap <br>
map에는 현재 주차 중인 차량의 입차 시간을 저장했다. <br>
- key: 차량 번호
- value: 입차 시각(분 단위)
차량이 IN이면 입차 시간을 저장하고, OUT이면 저장된 입차 시간과의 차이를 계산해 주차 시간을 구했다. <br>

2. 누적 주차 시간 저장용 HashMap <br>
timeMap에는 차량별 누적 주차 시간을 저장했다. <br>
- 이번 주차 시간 = 출차 시각 - 입차 시각
- 누적 시간 += 이번 주차 시간

3. 23:59 출차 처리
기록을 모두 확인한 뒤에도 map에 남아 있는 차량은 출차 기록이 없는 차량이므로 23:59에 출차한 것으로 간주했다. <br>

4. 차량 번호 오름차순 정렬
문제에서 차량 번호가 작은 순서대로 결과를 반환해야 하므로 timeMap.keySet()을 배열로 옮긴 뒤 Arrays.sort()로 정렬했다. <br>
그 후 정렬된 차량 번호 순서대로 요금을 계산해서 answer 배열에 담았다. <br>

### 제출 코드
```java
import java.util.*;
class Solution {
    public int[] solution(int[] fees, String[] records) {
        
        HashMap<String, Integer> map = new HashMap<>(); // 차번호와 입차시간 저장
        HashMap<String, Integer> timeMap = new HashMap<>(); // 차번호와 누적 시간 저장
        
        for(int i = 0 ; i < records.length; i++){
            String record = records[i];
            int hour = Integer.parseInt(record.substring(0, 2));
            int min = Integer.parseInt(record.substring(3, 5));
            int time = hour * 60 + min; // 입출차 시간을 분으로 변경
            String carNum = record.substring(6,10);
            String inOut = record.substring(11);
            
            if(inOut.equals("IN")){
                map.put(carNum, time);
            } 
            // OUT인 경우
            else{
                int inTime = map.get(carNum); // 입차시간
                int totalTime = time - inTime; // 주차시간
                if(timeMap.containsKey(carNum)){
                    int prevTime = timeMap.get(carNum);
                    timeMap.put(carNum, totalTime+prevTime);
                }
                else timeMap.put(carNum, totalTime);
                map.remove(carNum);
            }         
        }
        // 23시 59분 출차 처리
        if(!map.isEmpty()){
            for(String key : map.keySet()){
                int time = 23*60+59 - map.get(key); 
                if(timeMap.containsKey(key)){
                    int prevTime = timeMap.get(key);
                    timeMap.put(key, prevTime + time);
                }
                else timeMap.put(key, time);
            }
        }
        
        String[] carArray = new String[timeMap.size()];
        int idx = 0;
        for(String key : timeMap.keySet()){
            carArray[idx++] = key;
        }
        Arrays.sort(carArray); // 차번호 오름차순 정렬
        int[] answer = new int[timeMap.size()];
        for(int i = 0 ; i < timeMap.size(); i++){
            answer[i] = calFee(timeMap.get(carArray[i]), fees);
        }
        
        return answer;
    }
    
    // 요금 계산
    int calFee(int time, int[] fees){        
        if(time <= fees[0]) return fees[1];
        else{
            time -= fees[0];
            return fees[1] + fees[3] * (int)(Math.ceil((double)time/fees[2]));
        }        
    }
}
```

### 삽길 기록 🧠
1. 처음에는 하루 누적 시간으로 계산한다는걸 못읽어서, 차량이 나갈 때마다 바로 요금을 계산해서 누적 금액을 구했다.하지만 문제는 총 주차 시간을 기준으로 요금을 계산해야했다.  <br>

2. HashMap 순회 순서를 믿으면 안된다. <br>
처음에는 keySet()으로 key를 2차원 배열의 [0]에 넣고,<br>
values()로 value를 2차원 배열의 [1]에 넣으려고 했는데 <br>
HashMap은 순서를 보장하지 않기 때문에 key와 value를 따로 순회하면 짝이 어긋날 수 있다. <br>
그래서 key와 value를 함께 다뤄야 할 때는 `entrySet()`을 쓰거나 <br>
내가 이번 문제에 사용한거처럼 아예 차량 번호를 정렬한 뒤 `map.get(key)`로 꺼내는 방식이 더 안전하다.  <br>
