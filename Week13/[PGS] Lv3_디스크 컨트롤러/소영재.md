### 문제 설명 (요약)
- 디스크 컨트롤러는 여러 작업 요청이 들어왔을 때, 우선 순위에 따라 처리한다.
- 각 작업은 [요청 시각, 작업 소요 시간] 형태로 주어지며, 디스크 컨트롤러는 현재 처리 가능한 작업들 중 우선순위가 가장 높은 작업을 선택해 실행한다.
- 작업의 우선순위는 다음 순서로 결정된다.
- 1. 작업 소요 시간이 짧은 작업
- 2. 요청 시각이 빠른 작업
- 3. 작업 번호가 작은 작업
- 우선순위에 따라 작업을 처리한 뒤, 모든 작업의 반환 시간 평균의 정수 부분을 구하는 문제이다.

### 입력
- int[][] jobs : 작업 정보가 담긴 2차원 배열

### 출력
- 모든 작업의 요청부터 종료까지 걸린 시간의 평균값을 정수로 반환

<hr>

### 핵심 풀이 전략
1. 작업 배열을 우선순위에 맞게 정렬한다 (소요시간이 짧은 작업, 작업의 요청 시각이 빠른 순)
2. 각 작업이 처리되었는지를 저장하는 boolean[] visited 배열을 선언한다.
3. 반복문을 돌면서 현재 시간까지 요청된 작업 중 작업 시간이 가장 짧은 작업을 찾는다.
4. 처리할 수 있는 작업이 없다면 디스크의 시간(time)만 증가시킨다.
5. 작업 완료 시간 - 요청 시간을 total에 누적한다.
6. 해당 작업을 처리 완료로 표시한다.
7. 최종적ㅇ로 모든 작업의 평균 반환 시간을 반환한다.

---

```java
import java.util.*;

class Solution {
    public int solution(int[][] jobs) {

        Arrays.sort(jobs, (a, b) -> {
            if (a[1] == b[1]) {
                return a[0] - b[0];
            }
            return a[1] - b[1];
        });

        boolean[] visited = new boolean[jobs.length];

        //현재 시간
        int time = 0;
        //모든 작업의 합계 (대기 시간 + 작업 시간)
        int total = 0;
        //작업을 완료한 개수
        int count = 0;

        while (count < jobs.length) {
            //현재 선택된 작업 위치
            int currentJob = -1;
            //현재까지 찾은 작업 중 가장 짧은 작업 시간
            int minWorkTime = Integer.MAX_VALUE;

            for (int i = 0; i < jobs.length; i++) {

                if (visited[i]) {
                    continue;
                }

                if (jobs[i][0] <= time && jobs[i][1] < minWorkTime) {
                    currentJob = i;
                    minWorkTime = jobs[i][1];
                }
            }

            // 현재 시간에 처리할 수 있는 작업이 없는 경우
            if (currentJob == -1) {
                time++;
                continue;
            }

            // 선택한 작업 처리
            time += jobs[currentJob][1];

            // 작업 완료 시간 - 요청 시간
            total += time - jobs[currentJob][0];

            visited[currentJob] = true;
            count++;
        }

        return total / jobs.length;
    }
}
```
