### 성능 요약
메모리: KB, 시간: ms  
시간복잡도: O(M × K log K)

### 제출 일자

### 문제 설명
배열 `array`와 여러 개의 명령 `commands`가 주어진다.<br>
각 명령은 `[i, j, k]` 형태이며, 배열의 `i`번째 숫자부터 `j`번째 숫자까지 자른 뒤 정렬했을 때 `k`번째에 있는 수를 구해야 한다.<br>
<br>
배열의 인덱스는 문제 설명상 1부터 시작하지만, Java 배열 인덱스는 0부터 시작하므로 인덱스 변환이 필요하다.<br>
<br>

### 입력
- array: 원본 정수 배열<br>
- commands: 수행할 명령 배열<br>
- commands[i][0]: 자르기 시작 위치<br>
- commands[i][1]: 자르기 끝 위치<br>
- commands[i][2]: 정렬 후 선택할 위치<br>

### 출력
- 각 command마다 구한 K번째 수를 담은 배열<br> 

---

### 핵심 풀이 전략
- commands 배열을 순회하면서 각 명령을 하나씩 처리한다.<br>
- `commands[i][0]`부터 `commands[i][1]`까지의 구간 크기를 계산한다.<br>
- 해당 크기만큼 새로운 배열 `newArray`를 만든다.<br>
- 원본 배열에서 지정된 구간의 값을 `newArray`에 복사한다.<br>
- `newArray`를 오름차순 정렬한다.<br>
- 정렬된 배열에서 `commands[i][2] - 1`번째 값을 answer에 저장한다.<br>
- 문제의 위치는 1부터 시작하므로 실제 배열 접근 시에는 `-1`을 해줘야 한다.<br>

---

### 제출 코드
```java
import java.util.*;

class Solution {
    public int[] solution(int[] array, int[][] commands) {
        int[] answer = new int[commands.length];
        for(int i = 0; i < commands.length; i++){
            int size = commands[i][1] - commands[i][0] + 1;
            int[] newArray = new int [size];
            int k = 0;
            for(int j = commands[i][0] - 1; j <= commands[i][1]-1; j++){
                newArray[k] = array[j];
                k++;
            }
            Arrays.sort(newArray);
            answer[i] = newArray[commands[i][2]-1];
        }
        return answer;
    }
}
```
