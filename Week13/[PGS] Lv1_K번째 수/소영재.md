### 문제 설명 (요약)
- 주어진 배열에서 각 commmand[i, j, k]에 대해 i번째부터 j번째까지 자르고 정렬한 뒤 k번째 수를 구하는 문제

### 입력
- int[] array : 정수가 담겨있는 1차원 정수 배열
- int[][] commands : [i,j,k] 형태의 2차원 정수 배열

### 출력
- 각 command의 결과를 담은 int[]를 반환

<hr>

### 핵심 풀이 전략
1. commands 배열을 순회하며 i, j, k 값을 각각 꺼낸다.
2. i번째부터 j번째까지 자르고 들어갈 숫자만큼 임시 배열 temporaryArray를 선언한다.
3. temporaryArray 배열에 해당하는 값들을 대입한 뒤, sort() 메서드로 정렬한다.
4. 정렬된 값을 리스트에 담는다.
5. 리스트에 있는 값들을 다시 int[]로 변환한 뒤, 반환한다.

---

```java
import java.util.*;

class Solution {
    public int[] solution(int[] array, int[][] commands) {

        List<Integer> answer = new ArrayList<>();

        for (int i = 0; i < commands.length; i++) {

            int firstNumber = commands[i][0];
            int secondNumber = commands[i][1];
            int thirdNumber = commands[i][2];

            int[] temporaryArray = new int[secondNumber - firstNumber + 1];

            int index = 0;
            for (int j = firstNumber - 1; j <= secondNumber - 1; j++) {
                temporaryArray[index] = array[j];
                index++;
            }

            Arrays.sort(temporaryArray);

            answer.add(temporaryArray[thirdNumber - 1]);
        }

        int[] result = new int[answer.size()];

        for (int i = 0; i < answer.size(); i++) {
            result[i] = answer.get(i);
        }

        return result;
    }
}
```
