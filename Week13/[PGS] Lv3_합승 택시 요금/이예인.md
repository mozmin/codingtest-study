### 문제 설명
출발 지점 `s`에서 A의 도착 지점 `a`, B의 도착 지점 `b`까지 이동할 때 필요한 최소 택시 요금을 구하는 문제이다.<br>
<br>
A와 B는 일정 지점까지 택시를 함께 타고 이동할 수 있다.<br>
이후 특정 지점에서 각자의 목적지로 따로 이동할 수 있다.<br>
<br>
따라서 모든 지점을 합승 종료 지점으로 가정해보고, 다음 비용의 최솟값을 구하면 된다.<br>
<br>
`s → 합승 종료 지점` 비용 + `합승 종료 지점 → a` 비용 + `합승 종료 지점 → b` 비용<br>

### 입력
- n: 지점의 개수<br>
- s: 출발 지점<br>
- a: A의 도착 지점<br>
- b: B의 도착 지점<br>
- fares: 지점 사이의 택시 요금 정보<br>
- fares[i][0]: 출발 지점<br>
- fares[i][1]: 도착 지점<br>
- fares[i][2]: 두 지점 사이의 요금<br> 

### 출력
- A와 B가 목적지까지 이동하는 데 필요한 최소 택시 요금<br>

---

### 핵심 풀이 전략
- 그래프를 인접 행렬 `table`로 구성한다.<br>
- 택시 요금은 양방향이므로 `table[aa][bb]`, `table[bb][aa]`를 모두 채운다.<br>
- 출발지 `s`에서 모든 지점까지의 최단 거리를 구한다.<br>
- A의 도착지 `a`에서 모든 지점까지의 최단 거리를 구한다.<br>
- B의 도착지 `b`에서 모든 지점까지의 최단 거리를 구한다.<br>
- 특정 지점 `i`를 합승 종료 지점이라고 생각하면 전체 비용은 `distS[i] + distA[i] + distB[i]`이다.<br>
- 모든 지점 `i`에 대해 위 값을 계산하고 최솟값을 answer로 갱신한다.<br>

---

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(int n, int s, int a, int b, int[][] fares) {
        int max = 200 * 100 * 100000;
        int answer = max;
        int[][] table = new int [n+1][n+1];
        for(int i = 0; i < fares.length; i++){
            int aa = fares[i][0];
            int bb = fares[i][1];
            int cc = fares[i][2];
            
            table[aa][bb] = cc;
            table[bb][aa] = cc;
        }
        
        int[] distS = check(s, table, n, max);
        int[] distA = check(a, table, n, max);
        int[] distB = check(b, table, n, max);
        
        for(int i = 1; i <= n; i++){
            int total = distS[i] + distA[i] + distB[i];
            answer = Math.min(answer, total);
        }
        
        
        return answer;
    }
    
    int[] check(int start, int[][] table, int n, int max){
        int[] dist = new int[n+1];
        Arrays.fill(dist, max);
        
        dist[start] = 0;
        
        PriorityQueue<int[]> pq = new PriorityQueue<>(
            Comparator.comparingInt(o -> o[1])
        );
        
        pq.offer(new int[] {start, 0});
        
        while(!pq.isEmpty()){
            int[] cur = pq.poll();
            
            int curNode = cur[0];
            int curCost = cur[1];
            
            if(dist[curNode] < curCost) continue;
            
            for(int i = 1; i <= n; i++){
                if(table[curNode][i] == 0) continue;
                
                int newCost = curCost + table[curNode][i];
                
                if(newCost < dist[i]){
                    dist[i] = newCost;
                    pq.offer(new int[]{i, newCost});
                }
            }
            
        }
        return dist;
        
    }
}
```

### 배운 점 && 보완할 점 
- 다익스트라를 구현하는 코드 공부를 더 많이 해야겠다.
- Priority Queue도 익숙하지 않아서.. Comparator.comparingInt에 익숙해도록 반복 연습을 해야할거 같다.
