# [Lv1] 키패드 누르기

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/67256)

`#구현` `#문자열` `#BFS`

### 제출 일자

2026년 5월 6일 (4.13ms, 90.9MB)

### 문제 설명 (요약)
- 주어진 숫자 배열을 순서대로 눌러야 한다.
- 숫자 1,4,7은 왼손, 3,6,9는 오른손으로 누른다.
- 숫자 2,5,8,0은 현재 왼손과 오른손 위치에서 더 가까운 손으로 누르며, 거리가 같으면 주어진 주 손잡이(hand)를 따른다.

### 입력
- int [] numbers : 눌러야 하는 숫자 배열
- String hand : 사용자의 주 손잡이 ("left"), ("right")

### 출력
- 각 숫자를 누른 손을 문자열로 반환 (왼손은 "L", 오른손은 "R")

<hr>

### 핵심 풀이 전략
1. 키패드를 4 x 3 2차원 배열로 만든다.
2. 왼손 시작 위치는 "*", 오른손 시작 위치는 "#"으로 둔다.
3. 숫자가 1,4,7이면 왼손을 사용하고 3,6,9이면 오른손을 사용한다.
4. 숫자가 2,5,8,0이면 BFS로 각 손가락 위치에서 목표 숫자까지의 최단 거리를 구한 뒤, 더 짧은 손을 사용한다.
5. 만약 거리가 같으면 hand 값에 따라 사용할 손을 결정한다.
6. 숫자를 누른 뒤에는 해당 손의 현재 위치를 갱신한다.
7. 최종적으로 각 숫자를 누른 손을 문자열로 반환한다.

---

```java
import java.util.*;
class Solution {
    int [] dx = {-1, 0, 1, 0};
    int [] dy = {0, 1, 0, -1};
    
    public String solution(int[] numbers, String hand) {
        StringBuilder sb = new StringBuilder();
        String[][] phoneArray = new String[4][3];
        //왼손의 현재 위치를 나타내는 포인터(시작은 "*")
        String leftPointer = "*";
        //오른손의 현재 위치를 나타내는 포인터(시작은 "#")
        String rightPointer ="#";
        
        //핸드폰을 2차원 배열로 만들기
        int index = 1;
        for(int i = 0; i < 3; i++){
            for(int j = 0; j <3; j++){
                phoneArray[i][j] = String.valueOf(index);
                index++;
            }
        }
        
        //1~9 제외한 나머지 처리
        phoneArray[3][0] = "*";
        phoneArray[3][1] = "0";
        phoneArray[3][2] = "#";
        
        for(int i = 0; i < numbers.length; i++){
            
            //1,4,7일 경우에는 왼손 사용
            if(numbers[i] == 1 || numbers[i] == 4 || numbers[i] == 7){
                sb.append("L");
                leftPointer = String.valueOf(numbers[i]);
            }
            
            //3,6,9일 경우에는 오른손 사용
            else if(numbers[i] == 3 || numbers[i] == 6 || numbers[i] == 9){
                sb.append("R");
                rightPointer = String.valueOf(numbers[i]);
            }
            
            //2,5,8,0일 경우에는 현재 위치에서 가까운 손가락을 사용
            else {
                //rightLength, leftLength = 각 손가락 위치에서 현재 숫자까지의 이동 거리
                int rightLength = checkFinger(phoneArray, rightPointer, numbers[i]);
                int leftLength = checkFinger(phoneArray, leftPointer, numbers[i]);
                
                //만약, 길이가 동일하다면 어떤 손잡이인지에 따라 결정
                if(rightLength == leftLength){
                    if(hand.equals("right")){
                        sb.append("R");
                        rightPointer = String.valueOf(numbers[i]);
                    } else {
                        sb.append("L");
                        leftPointer = String.valueOf(numbers[i]);
                    }
                    
                //길이가 다를 경우 짧은 손가락을 사용    
                } else if(rightLength < leftLength){
                    sb.append("R");
                    rightPointer = String.valueOf(numbers[i]);
                } else {
                    sb.append("L");
                    leftPointer = String.valueOf(numbers[i]);
                }
            }
        }
        return sb.toString();
    }
    // 시작 위치에서 endPoint까지 최단 거리 탐색
    public int checkFinger(String[][]phoneArray, String startPoint, int endPoint){
        
        Deque<int[]> queue = new ArrayDeque<>();
        boolean[][] visited = new boolean[4][3];
        
        int startX = 0;
        int startY = 0;
        
        //시작점 탐색
        for(int i = 0; i < 4; i++){
            for(int j =0; j < 3; j++){
                if(phoneArray[i][j].equals(startPoint)){
                    startX = i;
                    startY = j;
                }
            }
        }
        
        queue.offer(new int[]{startX, startY, 0});
        visited[startX][startY] = true;
        
        while(!queue.isEmpty()){
            int [] current = queue.poll();
            
            int x = current[0];
            int y = current[1];
            int dist = current[2];
            
            if(phoneArray[x][y].equals(String.valueOf(endPoint))){
                return dist;
            }
            
            for(int i = 0; i < 4; i++){
                int nx = x + dx[i];
                int ny = y + dy[i];
                
                if(nx < 0 || ny < 0 || nx >=4 || ny >= 3){
                    continue;
                }
                
                if(visited[nx][ny]){
                    continue;
                }
                visited[nx][ny] = true;
                queue.offer(new int[]{nx, ny, dist +1});
            }
        }
        return -1;
    } 
}
```

### 삽질 기록 🧠
- "*", "#"의 존재 때문에 계속 String.valueOf() 메서드를 사용하게 되어서 자료형에서 실수가 많았다.

### 배운 점 && 보완할 점
- StringBuilder 클래스 적용 : 문자열이 계속 수정된다는 점을 파악해서 이전에 배운 StringBuilder 클래스를 적용하였다.
- phoneArray를 만들 때 1~9까지 반복문을 돌리고, "*", "#", "0"을 별도로 대입하고 BFS 내에서 시작점을 반복문을 통해 구하는 방식이다.
- 비효율적인 것 같아 문제를 푼 뒤, GPT에게 물어보니 Map자료형을 사용해서 쉽게 풀 수 있었다.
```java
Map<String, int[]> map = new HashMap<>();
map.put("1", new int[]{0,0});
```
