### 문제 설명
스마트폰 키패드에서 주어진 숫자들을 순서대로 눌러야 한다.<br>
규칙은 다음과 같다.
- 1, 4, 7은 항상 왼손으로 누른다.
- 3, 6, 9는 항상 오른손으로 누른다.
- 2, 5, 8, 0은 현재 왼손과 오른손 위치에서 더 가까운 손으로 누른다.
- 거리가 같으면 사용자의 주 손잡이(hand)를 따른다.
- 처음 왼손은 *, 오른손은 #에 위치한다.

### 입력
- numbers: 순서대로 눌러야 하는 숫자 배열
- hand: 왼손잡이인지 오른손잡이인지 나타내는 문자열

### 출력
- 각 숫자를 누른 손을 순서대로 이어 붙인 문자열을 반환한다.

---

### 핵심 풀이 전략
- 왼쪽 열 숫자 1, 4, 7은 바로 왼손 처리한다.
- 오른쪽 열 숫자 3, 6, 9는 바로 오른손 처리한다.
- 가운데 열 숫자 2, 5, 8, 0은 거리 비교 후 처리한다.
- 선택한 손의 현재 위치를 반드시 갱신한다.
- *는 10, 0은 11, #은 12로 치환하면 계산하기 쉽다.
- 키패드의 위치를 좌표로 변환해서 생각한다.
```
x = (11 - 1) / 3 = 3
y = (11 - 1) % 3 = 1
```
- 현재 숫자와 각 손 위치 사이의 거리는 맨해튼 거리로 계산한다.
```
거리 = Math.abs(x1 - x2) + Math.abs(y1 - y2)
```
---

### 제출 코드
```java
import java.util.*;
class Solution {
    public String solution(int[] numbers, String hand) {
        String answer = "";
        StringBuilder sb = new StringBuilder();
        int leftHand = 10;
        int rightHand = 12;
        for(int i = 0 ; i < numbers.length; i++){
            if(numbers[i] % 3 == 1){
                sb.append("L");
                leftHand = numbers[i];
            }
            else if (numbers[i] != 0 && numbers[i] % 3 == 0){
                sb.append("R");
                rightHand = numbers[i];
            }
            else{
                int num = numbers[i];
                int numLeft = leftHand;
                int numRight = rightHand;
                if(num == 0) num = 11;
                if(numLeft == 0) numLeft = 11;
                if(numRight == 0) numRight = 11;
                
                int x = (num-1) / 3; 
                int y = (num-1) % 3; 
                
                int left_x = (numLeft-1) / 3; 
                int left_y = (numLeft-1) % 3; 
                
                int right_x = (numRight -1) / 3; 
                int right_y = (numRight-1) % 3; 
                
                int left = Math.abs(x - left_x) + Math.abs(y - left_y); 
                int right = Math.abs(x - right_x) + Math.abs(y - right_y); 
                

                if(left == right){
                    if(hand.equals("right")){
                        sb.append("R");
                        rightHand = numbers[i];
                    }
                    else{
                        sb.append("L");
                        leftHand = numbers[i];
                    }
                }
                else if(left > right){
                    sb.append("R");
                    rightHand = numbers[i];
                }
                else{
                    sb.append("L");
                    leftHand = numbers[i];
                }  
            }        
        }
        answer = sb.toString();
        
        return answer;
    }
}
```
