# [Lv2] 무인도 여행
https://school.programmers.co.kr/learn/courses/30/lessons/154540

`#BFS` `#그래프탐색` `#완전탐색` `#정렬`

### 성능 요약
메모리: KB, 시간: ms  
시간복잡도: O(N × M)
- `N`: maps의 행 개수
- `M`: maps의 열 개수
- 모든 칸을 최대 한 번씩 방문하므로 전체 탐색 시간은 `O(N × M)`

### 제출 일자
2026년 4월 28일

### 문제 설명
지도 `maps`가 주어진다.  <br>
각 칸은 바다 또는 무인도를 의미한다.<br>

- `'X'`: 바다
- 숫자 문자: 무인도이며, 해당 칸에서 얻을 수 있는 식량의 양
 
상하좌우로 연결된 숫자 칸들은 하나의 무인도로 본다.  <br>
각 무인도에서 얻을 수 있는 총 식량의 양을 구한 뒤, 오름차순으로 정렬하여 반환한다. <br>

만약 갈 수 있는 무인도가 없다면 `[-1]`을 반환한다. <br>

### 입력
- `maps`: 문자열 배열
  - 각 문자열은 지도의 한 행을 의미한다.
  - 각 문자는 `'X'` 또는 숫자로 구성된다.


### 출력
- 각 무인도에서 머물 수 있는 날짜 수를 오름차순으로 정렬한 배열
- 무인도가 없다면 `[-1]`

---

### 핵심 풀이 전략
- 문자열 배열 `maps`를 탐색하기 쉽도록 `char[][] map`으로 변환한다.
- 방문 여부를 저장하기 위해 `boolean[][] visited` 배열을 사용한다.
- 전체 지도를 순회하면서 아직 방문하지 않은 숫자 칸을 찾는다.
- 숫자 칸을 발견하면 BFS를 실행하여 연결된 무인도 전체를 탐색한다.
- BFS 중 방문한 숫자 칸의 값을 식량 합계에 더한다.
- 하나의 BFS가 끝나면 해당 무인도의 총 식량 값을 리스트에 저장한다.
- 모든 탐색이 끝난 뒤 리스트가 비어 있으면 `[-1]`을 반환한다.
- 리스트에 값이 있다면 배열로 변환한 뒤 오름차순 정렬하여 반환한다.

---

### 제출 코드
```java
import java.util.*;
class Solution {
    public int[] solution(String[] maps) {
        char[][] map = new char[maps.length][maps[0].length()];
        for(int i = 0 ; i < maps.length; i++){
            for(int j = 0 ; j < maps[0].length(); j++){
                map[i][j] = maps[i].charAt(j);
            }
        }
        // 무인도 개수
        List<Integer> list = new ArrayList<>();
        boolean[][] visited = new boolean[map.length][map[0].length];

        
        for(int i = 0 ; i < map.length; i++){
            for(int j = 0 ; j < map[0].length; j++){
                if(map[i][j] != 'X' && visited[i][j] == false){
                    int food = bfs(map, i, j, visited);
                    list.add(food);
                }
            }
        }
        if(list.isEmpty()) return new int[]{-1};
        
        int[] answer = new int[list.size()];
        for(int i = 0 ; i < answer.length; i++){
            answer[i] = list.get(i);
        }
        Arrays.sort(answer);
        return answer;
    }
    
    int bfs(char[][] map, int x, int y, boolean[][] visited){
        Queue<int[]> queue = new LinkedList<>();
        int[] dx = {1, -1, 0, 0};
        int[] dy = {0, 0, 1, -1};
        int food = 0; // 식량
        queue.offer(new int[]{x, y});
        visited[x][y] = true;
        while(!queue.isEmpty()){
            int[] cur = queue.poll();
            int xx = cur[0];
            int yy = cur[1];
            food += map[xx][yy]-'0';
            for(int k = 0; k < 4; k++){
                int nx = xx + dx[k];
                int ny = yy + dy[k];
                if(nx >= 0 && ny >= 0 && nx < map.length && ny < map[0].length){
                    if(visited[nx][ny] == false && map[nx][ny] != 'X'){
                        visited[nx][ny] = true;
                        queue.offer(new int[]{nx, ny});
                        
                    }
                }

            }
        }
        
        return food;
        
    }
}
```
