### 문제 설명 (요약)
- 문자열 배열 (maps)는 무인도 지도를 의미한다.
- ('X')는 바다, ('1' ~ '9')는 땅이며, 숫자는 식량의 양이다.
- 상,하,좌,우로 연결된 땅들은 하나의 무인도로 본다.
- 각 무인도별 최대 체류 일수를 구한 뒤, 오름차순으로 정렬해서 반환한다.
- 만약 무인도가 하나도 없다면 -1을 반환한다.

### 입력
- String[] maps : 지도의 한 행이 담겨있는 1차원 String 배열

### 출력
- 각 무인도에서 머무를 수 있는 최대 일수를 오름차순으로 담은 배열을 반환
<hr>

### 핵심 풀이 전략
1. maps를 2차원 char 배열로 변환한다.
2. 전체 지도를 돌면서 아직 방문하지 않은 땅을 찾는다.
3. 해당 위치에서 BFS를 통해 하나의 섬 전체를 탐색한다.
4. 섬을 탐색하면서 숫자를 모두 더한다.
5. 각 섬의 합을 answer 리스트에 저장한 뒤, int[] 배열로 변환하여 반환한다. 만약 섬이 없다면 -1을 반환한다.
---

```java
import java.util.*;
class Solution {
    int dx[] = {-1, 0, 1, 0};
    int dy[] = {0, 1, 0, -1};
    public int[] solution(String[] maps) {
        
        char arrayMap [][] = new char[maps.length][maps[0].length()]; 
        boolean visited[][] = new boolean[maps.length][maps[0].length()];
        
        ArrayList <Integer> answer = new ArrayList<>();
        
        for(int i = 0; i < maps.length; i++){
            for(int j = 0; j < maps[0].length(); j++){
                arrayMap[i][j] = maps[i].charAt(j);
            }
        }
        
        for(int i = 0; i < maps.length; i++){
            for(int j = 0; j < maps[0].length(); j++){
                if(arrayMap[i][j] != 'X' && visited[i][j] == false){
                    int count = bfs(arrayMap, i, j, visited);
                    answer.add(count);
                }
            }
        }
        if(answer.size() == 0){
            return new int[]{-1};
        }
        
        int[] answer2 = new int[answer.size()];
        for(int i = 0; i < answer.size(); i++){
            answer2[i] = answer.get(i);
        }
        Arrays.sort(answer2);
        return answer2;
    }
   
    public int bfs(char[][] arrayMap, int x, int y, boolean[][] visited){
        Deque<int[]> queue = new ArrayDeque<>();
        queue.offer(new int[] {x,y});
        visited[x][y] = true;
        
        int sum = arrayMap[x][y] - '0';
        
        while(!queue.isEmpty()){
            int[] cur = queue.poll();
            int cx = cur[0];
            int cy = cur[1];
            
            for(int i = 0; i < 4; i++){
                int nx = cx + dx[i];
                int ny = cy + dy[i];
                
                if(nx < 0 || ny < 0 || nx >= arrayMap.length || ny >= arrayMap[0].length){
                    continue;
                }
                
                if(visited[nx][ny]){
                    continue;
                }
                
                if(arrayMap[nx][ny] == 'X'){
                    continue;
                }
                
                queue.offer(new int[]{nx, ny});
                visited[nx][ny] = true;
                
                sum += (int)arrayMap[nx][ny] -48;
            }
        }
        return sum;
    }
}
```

### 배운 점 && 보완할 점
- Arrays.sort()는 배열을 정렬한 뒤, 반환값이 void 이기 때문에 return Arrays.sort(answer2); 와 같이 사용할 수 없다는 점을 알게 되었다.
- char 타입의 값을 int로 변환할 때 크게 두 가지 방법이 있다.
- 1. 연산된 결과의 -'0' 처리
- 2. Character.getNumericValue();
- gpt 피셜 결과는 동일하지만 -'0'이 속도가 미세하게(?) 빠르다고 한다.
