### 문제 설명 (요약)
- 일렬로 늘어선 n개의 아파트가 있고 기존에 설치된 기지국의 위치가 stations으로 주어진다.
- 하나의 기지국은 자신의 위치를 기준으로 왼쪽 w칸, 오른쪽 w칸까지 전파를 전달할 수 있다.
- 모든 아파트에 전파가 전달되도록 하기 위해 추가로 설치해야 하는 기지국의 최소 개수를 구하는 문제이다.

### 입력
- int n : 아파트의 개수
- int[] stations : 이미 설치된 기지국의 위치 1차원 정수 배열
- int w : 기지국 하나가 좌우로 전달할 수 있는 거리
- 
### 출력
- 모든 아파트에 전파가 닿도록 추가 설치해야 하는 기지국의 최소 개수

<hr>

### 핵심 풀이 전략
1. 기지국이 이미 커버하고 있는 구간을 제외하고, 전파가 닿지 않는 빈 구간의 길이를 계산한다.
2. 빈 구간의 길이에 따라 필요한 기지국 수를 구한다. (몫, 나머지 계산)
3. 반복문을 전부 순회한 뒤, 마지막 구간의 필요한 기지국 개수를 계산한다.
4. 최종적으로 구해진 필요 기지국 개수를 반환한다. 

---

```java
class Solution {
    public int solution(int n, int[] stations, int w) {
        int spread = (w * 2) + 1;
        
        int answer = 0;
        
        int start = 0;
        
        for(int i = 0; i < stations.length; i++){
            int leftSpread = stations[i] - w;
            
            int rightSpread = stations[i] + w;
            
            if(start < leftSpread){
                int emptyApt = leftSpread - start-1;
                
                answer += emptyApt / spread;
                
                if(emptyApt % spread != 0){
                    answer++;
                }
            }
            start = rightSpread;
        }
        if(start < n){
            int emptyApt = n - start;
            
            answer += emptyApt / spread;
            
            if(emptyApt % spread != 0){
                    answer++;
                }
        }
        
        return answer;
    }
}
```
