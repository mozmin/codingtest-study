### 문제 설명
일렬로 `1번`부터 `n번`까지의 아파트가 있다.  <br>
이미 일부 위치에는 기지국이 설치되어 있고, 하나의 기지국은 설치 위치를 기준으로 좌우 `w`칸까지 전파를 전달할 수 있다.<br>
기존 기지국들이 커버하지 못하는 아파트 구간에 새 기지국을 최소 개수로 설치해야 한다.<br>

### 입력
- n: 아파트의 개수
- stations: 이미 설치된 기지국의 위치 배열
- w: 기지국 하나가 좌우로 커버할 수 있는 거리

### 출력
- 모든 아파트에 전파가 닿도록 추가로 설치해야 하는 기지국의 최소 개수

---

### 핵심 풀이 전략
- 기지국 하나가 커버할 수 있는 범위는 w * 2 + 1이다.
- 기존 기지국의 커버 구간을 순서대로 확인한다.
- 이전 기지국이 커버한 마지막 위치를 prev_end로 관리한다.
- 현재 기지국의 커버 시작 위치 start와 이전 커버 끝 위치 prev_end 사이에 전파가 닿지 않는 구간이 있는지 확인한다.
- 빈 구간의 길이 num이 존재하면, 해당 구간을 커버하기 위해 필요한 기지국 수를 계산한다.
- 필요한 기지국 수는 다음과 같이 올림 나눗셈으로 구한다.
- num / (w * 2 + 1) 나머지가 있다면 기지국이 하나 더 필요하다.
- 모든 기존 기지국을 확인한 뒤, 마지막 기지국 이후 n번 아파트까지 남은 구간도 추가로 확인한다.
  
---

### 제출 코드
```java
class Solution {
    public int solution(int n, int[] stations, int w) {
        int count = 0;
        int w_size = w * 2 + 1;
        int start = 0;
        int end = 0;
        int prev_end = 0;
        for(int i = 0; i < stations.length; i++){
            start = stations[i] - w;
            end = stations[i] + w;
            
            if(start < 1) start = 1;
            if(end > n) end = n;
 
            int num = start - prev_end - 1;
            
           if (num > 0) {
                count += num / w_size;
                if (num % w_size != 0) count++;
            }
            prev_end = end;
            
        }
        
        if(prev_end < n){
            int num = n - prev_end;
            count += num / w_size;
            if(num % w_size != 0) count++;
            
        }
        
        return count;
    }
}
```

### 삽길 기록 🧠
- 처음에 int배열로 풀었다가 시간초과가 나서 실패했다. 입력값을 보고 이렇게 풀면 안될거 같다는 생각을 하긴 햇는데,, 그래도 일단 풀어봤다.
- 이후에는 위의 코드로 풀었다가, 출력문을 안지워서 효율성 테스트에서 실패가 떴었다. 
- 앞으로 코테를 치게 되면 출력문은 무조건 지우고 제출하는게 좋을거 같다.
  
```java
import java.util.*;
class Solution {
    public int solution(int n, int[] stations, int w) {
        int count = 0;
        int w_size = w * 2 + 1;
        int[] table = new int[n+1];
        for(int i = 0 ; i < stations.length; i++){
            int ind = stations[i];
            for(int j = ind - w; j <= ind + w; j++){
                if(j < 0 || j > n) continue;
                table[j] = 1;
            }
        }     
        
        int nn = 0;
        for(int i = 1 ; i < n+1; i++){
            if(table[i] == 1){
                if(nn == 0) continue;
                int num = nn / w_size;
                if(nn % w_size != 0) num += 1;
                count += num;
                nn = 0;
            }
            else if(table[i] == 0){
                nn++;
            }
        }
        if(nn != 0){
            int num = nn / w_size;
            if(nn % w_size != 0) num += 1;
            count += num;
        }
        

        return count;
    }
}
```
