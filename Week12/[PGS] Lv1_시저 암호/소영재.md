### 문제 설명 (요약)
- 문자열 ('s')의 각 알파벳을 n만큼 뒤로 밀어 암호문을 만드는 문제이다.
- 대문자는 대문자 범위 안에서만 이동하고, 소문자는 소문자 범위 안에서만 이동한다. (z 다음은 a, Z 다믐은 A)
- 단 문자열 ('s')에 공백이 있다면 이동하지 않고 그대로 둔다.
- 최종적으로 만들어진 암호문을 반환한다.

### 입력
- String s : 문자열이 담겨있는 변수
- int n : 문자열을 이동시킬 거리

### 출력
- 문자열 ('s')의 각 알파벳을 n만큼 민 암호문을 반환한다.

<hr>

### 핵심 풀이 전략
1. 현재 문자를 꺼낸다.
2. 공백이면 그대로 결과에 추가한다.
3. 조건문으로 소문자와 대문자일 때 상황을 분리하고 n만큼 이동시킨 문자가 'Z', 'z'를 넘는지 검사한다.
4. 이동한 문자열을 answer에 변환하여 추가한다.
---

```java
import java.util.*;
class Solution {
    public String solution(String s, int n) {
        String answer = "";
        
        int word = s.length();
        
        for(int i = 0; i < word; i++){
            
            char key = s.charAt(i);
            
            if(key == ' '){
                answer += key;
                continue;
            }
              if(key>= 'A' && key <= 'Z' && key+n > 'Z'){
                answer += (char)(key+n-26);
                continue;
            }
            
             if(key >= 'a' && key <= 'z' && key+n > 'z'){
                answer += (char)(key+n-26);
                continue;
            }   
            
            answer += (char)(key+n);          
        }
        
        return answer;
    }
}
```
