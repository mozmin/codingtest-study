### 문제 설명
문자열 `s`와 정수 `n`이 주어진다.  <br>
문자열의 각 알파벳을 알파벳 순서대로 `n`칸 뒤의 문자로 바꾼 문자열을 반환한다.<br>

단, 알파벳이 `z` 또는 `Z`를 넘어가면 다시 `a` 또는 `A`부터 이어서 변환한다.  <br>
공백은 아무리 밀어도 공백 그대로 유지한다.<br>

### 입력
- 문자열 `s`
- 정수 `n`

### 출력
- 문자열 `s`의 각 알파벳을 `n`만큼 민 결과 문자열

---

### 핵심 풀이 전략
- 문자열을 한 글자씩 순회한다.
- 공백이면 그대로 결과 문자열에 추가한다.
- 대문자와 소문자를 ASCII 범위로 구분한다.
  - 대문자: `A(65)` ~ `Z(90)`
  - 소문자: `a(97)` ~ `z(122)`
- 현재 문자에 `n`을 더한 뒤, 알파벳 범위를 벗어나면 `26`을 빼서 다시 앞쪽 알파벳으로 순환시킨다.
- 문자열 누적은 `StringBuilder`를 사용해 효율적으로 처리한다.

---

### 제출 코드
```java
import java.util.*;

class Solution {
    public String solution(String s, int n) {
        StringBuilder sb = new StringBuilder();
        //a: 97 A:65 // z : 122 Z : 90 
        
        for(int i = 0; i < s.length(); i++){
            if(s.charAt(i) == ' ') {
                sb.append(" ");
                continue;
            }
            int num = s.charAt(i);
            // 대문자
            if(num >= 65 && num <= 90){
                num += n;
                if(num > 90) num -= 26;
                char c = (char) num;
                sb.append(c);
            } 
            else{
                num += n;
                if(num > 122) num -= 26;
                char c = (char) num;
                sb.append(c);
            }
        }
        return sb.toString();
    }
}
```
