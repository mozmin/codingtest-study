### 문제 설명
양의 정수 `n`을 `k`진수로 바꾼 뒤, 변환된 수 안에서 조건에 맞는 소수가 몇 개인지 구하는 문제이다. <br>
`k`진수로 변환된 문자열에서 `0`을 기준으로 숫자를 나누었을 때, 각 숫자 조각이 소수라면 개수를 증가시킨다.<br>
이 중 소수인 숫자의 개수를 구하면 된다.

### 입력
- n: 변환할 10진수 양의 정수
- k: 변환할 진법

### 출력
- n을 k진수로 변환한 뒤, 조건에 맞는 소수의 개수

---

### 핵심 풀이 전략
- 먼저 n을 k진수로 변환한다.
- 변환 과정에서 나머지를 리스트에 저장하면 숫자가 거꾸로 저장되므로, 다시 뒤집어 올바른 순서의 배열을 만든다.
- 변환된 수를 앞에서부터 순회하면서 0을 기준으로 숫자 조각을 나눈다.
- 0을 만나면 지금까지 저장한 숫자 조각이 소수인지 판별한다.
- 마지막 숫자 조각은 뒤에 0이 없을 수 있으므로 반복문이 끝난 뒤 한 번 더 검사한다.
- 소수 판별은 2부터 √n까지 나누어 떨어지는 수가 있는지 확인한다.

---

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(int n, int k) {
        int answer = 0;
        List<Integer> list = new ArrayList<>();
        while(n >= k){
            int kk = n % k;
            n = n / k;
            list.add(kk);
        }
        
        list.add(n % k);
        
        int[] arr = new int[list.size()];
        for(int i = list.size()-1 ; i >= 0; i--){
            arr[list.size()-1-i] = list.get(i);
        }
        
        for(int i = 0 ; i < arr.length; i++){
            System.out.print(arr[i]);
        }
        System.out.println("---");
        
        List<Integer> list2 = new ArrayList<>();
        for(int i = 0 ; i < arr.length; i++){
            if(arr[i] == 0){
                if(check(list2)) answer++;
                list2.clear();
            }
            else{
                list2.add(arr[i]);
            }
        }
        if(!list2.isEmpty()){
            if(check(list2)) answer++;
        }
        
        
        return answer;
    }
    
    boolean check(List<Integer> list2){
        long n = 0;
        for(int i = list2.size()-1 ; i >= 0; i--){
            n += list2.get(i) * Math.pow(10, list2.size()-1-i);
        }
        System.out.println(n);
        if(n == 1 || n == 0) return false;
        for(int i = 2 ; i <= Math.sqrt(n); i++){
            if(n % i == 0) return false;
        }
        return true;
    }
    
}
```
