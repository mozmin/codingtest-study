### 문제 설명 (요약)
- 정수 n을 k진수로 변환한 후, 변환된 숫자를 0을 기준으로 분리하여 나온 숫자들 중에서 소수인 개수를 구하는 문제

### 입력
- int n : 변환할 10진수 정수
- int k : 변환할 진수

### 출력
- K진수로 변환 후 0으로 나눈 각 숫자 중 소수의 개수 반환
<hr>

### 핵심 풀이 전략
1. 10진수를 K진수 문자열로 변환한다.
2. 0을 기준으로 분리한다.
3. 빈 문자열은 건너뛰고 숫자로 변환한다.
4. 소수를 판별하여 소수의 개수를 구한 뒤, 반환한다.

---

```java
import java.util.*;
class Solution {
    public int solution(int n, int k) {
        String result = "";
        
        while(n > 0){
            int exNumber = n % k;
        
            result = exNumber + result;
            
            n = n / k;
        }
        
        String [] arr =  result.split("0");
        int count = 0;
        
        //NumberFormatException 발생
        for(String s : arr){
            if(s.equals("")){
                continue;
            }
            
            long conditionNumber = Long.parseLong(s);
            
            if(conditionNumber < 2){
                continue;
            }
            
            boolean check = true;
            int numberCount = 0;
            
            for(int i = 2; i <= Math.sqrt(conditionNumber); i++){
                if (conditionNumber % i == 0){
                    check = false;
                    break;
                }
            }
            
            if(check){
                count++;
            } 
        }
        
        return count;
    }
}
```
