### 문제 설명
- 모든 음식의 스코빌 지수를 K 이상으로 만들기 위해, 스코빌 지수가 가장 낮은 두 음식을 섞는 과정을 반복하는 문제이다.
- 두 음식을 섞으면 새로운 음식의 스코빌 지수는 다음과 같이 계산된다.
- 섞은 음식의 스코빌 지수 = 가장 맵지 않은 음식 + 두 번째로 맵지 않은 음식 × 2
- 모든 음식의 스코빌 지수가 K 이상이 되면 섞은 횟수를 반환하고, 아무리 섞어도 모든 음식의 스코빌 지수를 K 이상으로 만들 수 없다면 -1을 반환한다.

### 입력
- 음식들의 스코빌 지수를 담은 배열 scoville
- 목표 스코빌 지수 K

### 출력
- 모든 음식의 스코빌 지수를 K 이상으로 만들기 위해 섞어야 하는 최소 횟수
- 만들 수 없는 경우 -1

---

### 핵심 풀이 전략
- 매번 가장 스코빌 지수가 낮은 음식 두 개를 골라야 하므로 최소 힙을 사용한다.
- 사용하면 가장 작은 값을 빠르게 꺼낼 수 있다.
- 모든 스코빌 지수를 우선순위 큐에 넣는다.
- 큐의 최솟값이 K보다 작다면 음식을 섞어야 한다.
- 이때 큐에 음식이 2개 미만이면 더 이상 섞을 수 없으므로 -1을 반환한다.
- 가장 작은 값 두 개를 꺼내 새로운 스코빌 지수를 계산한 뒤 다시 큐에 넣는다.
- 섞는 과정을 한 번 진행할 때마다 횟수를 증가시킨다.
- 큐의 최솟값이 K 이상이 되는 순간 모든 음식의 스코빌 지수가 조건을 만족하므로 섞은 횟수를 반환한다.

---

### 제출 코드
```java
import java.util.*;

class Solution {
    public int solution(int[] scoville, int K) {
        int count = 0;
        PriorityQueue<Long> pq = new PriorityQueue<>();
        for(int s : scoville){
            pq.add((long)s);
        }
        while(!pq.isEmpty() && pq.peek() < K){
            if(pq.size() < 2) return -1;
            Long first = pq.poll();
            Long second = pq.poll();
            Long mixed = first + second * 2L;
            pq.offer(mixed);
            count++;
        }
        
        
        return count;
    }
}
```
