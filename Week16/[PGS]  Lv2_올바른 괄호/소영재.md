### 문제 설명 (요약)
- 괄모 훈자열이 주어졌을 때, 모든 괄호의 짝이 올바르게 맞는지 확인하는 문제이다.'

### 입력
- String s : '(' 와 ')'로만 이루어진 문자열

### 출력
- 올바른 괄호 문자열이면 'true', 올바르지 않은 괄호 문자열이면 'false'를 반환

<hr>

### 핵심 풀이 전략
1. '('를 만난다면 스택에 저장한다.
2. ')'를 만난다면 짝이 될 수 있는 '('가 있는지 확인한다.
    - 만약 스택이 비어있다면 짝을 찾을 수 없으므로 false
    - 스택이 비어있지 않다면 짝이 있으므로 '('를 제거한다.
3. 최종적으로 스택이 비어있다면 모두 짝을 찾은 것이므로 true를 반환한다.
4. 스택이 비어있지 않다면 '('이 남아있다는 것이므로 false를 반환한다.

---

```java
import java.util.*;
class Solution {
    boolean solution(String s) {
        Deque<Character> stack = new ArrayDeque<>();
        
        for(int i = 0; i < s.length(); i++){
            char a = s.charAt(i);
            if(a == '('){
                stack.push(a);
            } else {
                if(stack.isEmpty()){
                    return false;
                }
                stack.pop();
            }
        }
        if(stack.isEmpty()){
            return true;
        } else {
            return false;
        }
    }
}
```

### 삽질 기록 🧠
- 처음에는 '('과 ')'을 각각 담아서 계산하는 방식으로 풀이했는데, 조건문을 작성하다가 순서를 보장하지 못한다는 점을 알게 되어 다시 풀게 되었다.
