### 문제 설명
- 문자열 s가 주어졌을 때, 괄호가 올바르게 짝지어져 있는지 판단하는 문제이다.
- 올바른 괄호란 여는 괄호 (가 있으면 반드시 이후에 닫는 괄호 )가 나와야 하며, 닫는 괄호가 여는 괄호보다 먼저 등장하면 안 된다.
- 예를 들어 ()() 또는 (())()는 올바른 괄호이지만, )()( 또는 (()(는 올바르지 않은 괄호이다.

### 입력
- 괄호로만 이루어진 문자열 s

### 출력
- 문자열 s가 올바른 괄호이면 true
- 올바르지 않은 괄호이면 false

---

### 핵심 풀이 전략
- 문자열을 왼쪽부터 한 글자씩 확인한다.
- 여는 괄호 (가 나오면 스택에 넣는다.
- 닫는 괄호 )가 나오면 스택에서 여는 괄호 하나를 제거한다.
- 만약 닫는 괄호가 나왔는데 스택이 비어 있다면, 짝지을 여는 괄호가 없으므로 올바르지 않은 괄호이다.
- 문자열을 모두 확인한 뒤 스택이 비어 있다면 모든 괄호가 올바르게 짝지어진 것이다.
- 반대로 스택에 여는 괄호가 남아 있다면 닫히지 않은 괄호가 존재하므로 올바르지 않은 괄호이다. 

---

### 제출 코드
```java
import java.util.*; 

class Solution {
    boolean solution(String s) {
        
        Stack<Character> stack = new Stack<>();
        for(int i = 0; i < s.length(); i++){
            char c = s.charAt(i);
            if(c == '(') stack.push(c);
            else{
                if(stack.isEmpty()) return false;
                else stack.pop();
            }
        }

        if(stack.isEmpty()) return true;
        else return false;
        
    }
}
```
