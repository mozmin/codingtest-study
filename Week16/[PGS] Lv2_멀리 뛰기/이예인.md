### 문제 설명
- 효진이는 한 번에 1칸 또는 2칸을 뛸 수 있다.
- 총 n칸을 이동해야 할 때, 끝까지 도달하는 방법의 수를 구하는 문제이다.

### 입력
- 멀리뛰기에 사용될 칸의 수 n

### 출력
- 끝까지 도달하는 방법의 수를 1234567로 나눈 나머지

---

### 핵심 풀이 전략
- 마지막에 도달하는 방법을 기준으로 생각한다.
- n번째 칸에 도달하는 방법은 두 가지로 나눌 수 있다.
- n-1번째 칸에서 1칸을 뛰어오는 경우
- n-2번째 칸에서 2칸을 뛰어오는 경우
- 따라서 dp[n] = dp[n-1] + dp[n-2] 점화식을 세울 수 있다.
- dp[1] = 1, dp[2] = 2로 초기화한다.
- 이후 3부터 n까지 반복하면서 이전 두 값의 합을 저장한다.
- 결과값이 커질 수 있으므로 문제 조건에 따라 매번 1234567로 나눈 나머지를 저장한다.

---

### 제출 코드
```java
class Solution {
    public long solution(int n) {
        if(n==1) return 1;
        if(n==2) return 2;
        
        // dp[n] = dp[n-1] + dp[n-2]
        long[] dp = new long[n+1];
        dp[1] = 1;
        dp[2] = 2;
        
        for(int i = 3; i <= n; i++){
            dp[i] = (dp[i-1] + dp[i-2])%1234567;
        }
        
        return dp[n];
    }
}
```
