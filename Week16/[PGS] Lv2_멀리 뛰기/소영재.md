### 문제 설명 (요약)
- 한 번에 1칸 또는 2칸씩 뛸 수 있을 때, 정확히 n칸에 도달하는 모든 방법의 수를 구하는 문제이다.

### 입력
- int n : 도달해야 하는 거리

### 출력
- 1칸 또는 2칸씩 이동하여 정확히 n칸에 도달할 수 있는 방법의 수를 1234567로 나눈 나머지

<hr>

### 핵심 풀이 전략
1. dp[i]를 i칸까지 도달하는 방법의 수로 정의한다.
2. 초기 값 (dp[1], dp[2])을 바탕으로 dp[n]까지 도달하는 방법의 수를 계산한다.
3. 최종적으로 dp[n]이 n칸까지 이동할 수 있는 모든 방법의 수를 저장하고 있으므로 반환한다.

---

```java
class Solution {
    public long solution(int n) {
        if(n == 1){
            return 1;
        }
        if(n == 2){
            return 2;
        }
        
        int [] dp = new int [n+1];
        dp[1] = 1;
        dp[2] = 2;
        
        for(int i = 3; i <= n; i++){
            dp[i] = (dp[i-1] + dp[i-2]) % 1234567;
        }      
        
        return dp[n];
    }
}
```

### 삽질 기록 🧠
```java
class Solution {
    public long solution(int n) {
        if(n == 1){
            return 1;
        }
        if(n == 2){
            return 2;
        }
        
        int [] dp = new int [n+1];
        dp[1] = 1;
        dp[2] = 2;
        
        for(int i = 3; i <= n; i++){
            dp[i] = dp[i-1] + dp[i-2];
        }      
        
        return dp[n] % 1234567;
    }
}
```
- n이 커지게 된다면 반복문 내부에서 오버플로우가 발생하여 에러가 났었다.
- 또한 n이 1일 때와 2일 때 return 처리를 하지 않아서 몇몇 테스트 케이스에서 실패가 나왔었다.
