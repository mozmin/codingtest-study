`#다이나믹프로그래밍(DP)`

### 제출 일자
2026년 5월 31일 0.52ms

### 문제 설명 (요약)
`N`행 4열의 배열이 주어질 때, 첫 행부터 마지막 행까지 한 행씩 내려오며 각 행에서 하나의 열을 밟습니다.
단, 같은 열을 연속해서 밟을 수 없다는 규칙이 있습니다. 마지막 행까지 내려왔을 때 얻을 수 있는 점수의 최댓값을 구합니다.

---

### 핵심 풀이 전략
1. **다이나믹 프로그래밍(DP) 활용**: 이전 행까지의 최적해를 누적하여 다음 행의 최적해를 구하는 바텀업(Bottom-up) 방식을 사용합니다.
2. **연속된 열 제한 처리**: 현재 행의 `i`번째 열을 선택할 때, 이전 행의 `i`번째 열을 제외한 나머지 3개의 열 중 가장 큰 값을 찾아 현재 값에 더해줍니다.
3. **최종 결과**: 배열의 마지막 행에 누적된 4개의 값 중 가장 큰 값을 반환합니다.

---

### 내가 구현한 구현 코드
```java
import java.util.*;

class Solution {
    int solution(int[][] land) {
        int answer = 0;

        int[][] dist = new int[land.length][4];
        for(int i = 0; i < 4; i++){
            dist[0][i] = land[0][i];
        }
        
        for(int i = 1; i < land.length; i++){
            
            for(int j = 0; j < 4; j++){
                dist[i][j] = land[i][j] + maxValue(i, j, dist);
            }
        }
        
        for(int i = 0; i < 4; i++){
            answer = Math.max(answer, dist[land.length-1][i]);
        }
        
        return answer;
    }
    
    public int maxValue(int n, int index, int[][] dist){
        
        int max = 0;
        
        for(int i = 0; i < 4; i++){
            if(i == index) continue;
            
            max = Math.max(max, dist[n-1][i]); 
        }
        
        return max;
    }
    
}
```

### 정답 코드 (리팩토링 - 공간 복잡도 최소화 및 로직 단순화)
1) 주어진 `land` 배열을 그대로 재활용하여 메모리 낭비를 줄이고,
2) 열의 개수가 4개로 고정되어 있다는 점을 이용해 `Math.max`를 중첩 사용하는 방식으로 속도와 코드를 더 최적화할 수 있다.

```java
class Solution {
    int solution(int[][] land) {
        // 별도의 dist 배열 없이 원본 land 배열에 바로 누적 (공간 복잡도 최적화)
        for (int i = 1; i < land.length; i++) {
            // 열이 4개로 고정되어 있으므로 반복문이나 별도 메서드 호출 없이 하드코딩하는 것이 연산 속도에 유리함
            land[i][0] += Math.max(Math.max(land[i-1][1], land[i-1][2]), land[i-1][3]);
            land[i][1] += Math.max(Math.max(land[i-1][0], land[i-1][2]), land[i-1][3]);
            land[i][2] += Math.max(Math.max(land[i-1][0], land[i-1][1]), land[i-1][3]);
            land[i][3] += Math.max(Math.max(land[i-1][0], land[i-1][1]), land[i-1][2]);
        }

        int answer = 0;
        for (int i = 0; i < 4; i++) {
            answer = Math.max(answer, land[land.length-1][i]);
        }

        return answer;
    }
}
```
