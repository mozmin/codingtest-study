### 문제 설명
- 땅따먹기는 각 행에서 하나의 칸을 선택해 점수를 얻는 문제이다.
- 단, 바로 이전 행에서 선택한 열과 같은 열은 다음 행에서 다시 선택할 수 없다.
- 각 행마다 4개의 칸이 주어지며, 마지막 행까지 내려갔을 때 얻을 수 있는 점수의 최댓값을 구해야 한다.

### 입력
- 각 행마다 4개의 점수가 담긴 2차원 배열 land

### 출력
- 같은 열을 연속해서 선택하지 않으면서 얻을 수 있는 최대 점수

---

### 핵심 풀이 전략
- 각 칸에 도달했을 때 얻을 수 있는 최대 점수를 누적해서 저장한다.
- 현재 행의 특정 열을 선택하려면, 바로 이전 행에서는 같은 열을 선택할 수 없다.
- 따라서 현재 칸의 값에 이전 행의 다른 세 열 중 최댓값을 더한다.
- 이 과정을 두 번째 행부터 마지막 행까지 반복한다.
- 마지막 행에는 각 열을 마지막으로 선택했을 때의 최대 누적 점수가 저장된다.
- 마지막 행의 4개 값 중 가장 큰 값을 정답으로 반환한다.
- 별도의 DP 배열을 만들지 않고, 주어진 land 배열을 직접 갱신하여 공간을 절약할 수 있다.

---

### 제출 코드
```java
import java.util.*;

class Solution {
    int solution(int[][] land) {
        int answer = 0;
        for(int i = 1; i < land.length; i++){
            land[i][0] += Math.max(land[i-1][1], Math.max(land[i-1][2], land[i-1][3]));
            land[i][1] += Math.max(land[i-1][0], Math.max(land[i-1][2], land[i-1][3]));
            land[i][2] += Math.max(land[i-1][0], Math.max(land[i-1][1], land[i-1][3]));
            land[i][3] += Math.max(land[i-1][0], Math.max(land[i-1][1], land[i-1][2]));
        }
        for(int i = 0 ; i < 4; i++){
            if(answer < land[land.length-1][i]) answer = land[land.length-1][i];
        }
        return answer;
    }
}
```
