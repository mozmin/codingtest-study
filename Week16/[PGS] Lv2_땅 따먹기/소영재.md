### 문제 설명 (요약)
- 땅은 여러 행으로 이루어져 있고, 각 행에는 4개의 칸이 있다. 각 행에서는 반드시 하나의 칸을 선택해 점수를 얻어야 한다.
- 단, 바로 이전 행에서 선택한 칸과 같은 열의 칸은 선택할 수 없다.

### 입력
- int[][] land : 각 칸의 점수가 담긴 2차원 정수 배열

### 출력
- 같은 열을 연속해서 선택하지 않으면서 얻을 수 있는 최대 점수를 반환

<hr>

### 핵심 풀이 전략
1. i행 j열을 선택해서 얻을 수 있는 최대 점수를 저장하는 dp 2차원 정수 배열을 선언한다.
2. 반복문을 통해 dp 2차원 정수 배열 안에 i행j열을 선택했을 때 얻을 수 있는 최대 점수를 저장한다.
3. 마지막 행의 4개 값 중 최대값을 계산하여 반환한다.
---

```java (해설 코드 참고)
class Solution {
    int solution(int[][] land) {
        int[][] dp = new int[land.length][4];
        
        dp[0][0] = land[0][0];
        dp[0][1] = land[0][1];
        dp[0][2] = land[0][2];
        dp[0][3] = land[0][3];
        
        for (int i = 1; i < land.length; i++) {
            dp[i][0] = land[i][0] + Math.max(dp[i - 1][1], Math.max(dp[i - 1][2], dp[i - 1][3]));
            dp[i][1] = land[i][1] + Math.max(dp[i - 1][0], Math.max(dp[i - 1][2], dp[i - 1][3]));
            dp[i][2] = land[i][2] + Math.max(dp[i - 1][0], Math.max(dp[i - 1][1], dp[i - 1][3]));
            dp[i][3] = land[i][3] + Math.max(dp[i - 1][0], Math.max(dp[i - 1][1], dp[i - 1][2]));
        }

        int last = land.length - 1;

        return Math.max(
                Math.max(dp[last][0], dp[last][1]),
                Math.max(dp[last][2], dp[last][3])
        );
    }
}
```

### 삽질 기록 🧠
```java
class Solution {
    int score = 0;
    int solution(int[][] land) {
        
        for(int i = 0; i < land.length; i++){
            dfs(0, i, land, 0);
        }
        return score;
    }
    public void dfs(int depth, int startPoint, int[][] land, int sum){
        if(depth == land.length){
            score = Math.max(score, sum);
            return;
        }
        
        for(int i = 0; i < land[0].length; i++){
            if(i == startPoint){
                continue;
            }
            dfs(depth + 1, i, land, sum + land[depth][i]);
        }    
    }
}
```
- 처음에는 DFS로 모든 경우를 탐색하려고 했지만, 시간 복잡도가  4^N 즉 4^100,000 이라는 것을 알게 되었다.

### 배운 점 && 보완할 점
- dp[i][j]는 i행 j열을 선택해서 끝났을 때 얻을 수 있는 최대 누적 점수를 의미한다.
- 앞으로 비슷한 문제를 만나면 DFS를 활용하는 것을 고려하되, 시간 복잡도가 크거나 이전 단계의 값을 저장해서 재사용할 수 있다면 DP를 떠올려보는 생각을 가져야겠다.
