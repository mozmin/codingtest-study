### 문제 설명 (요약)
불량 이용자를 신고하고 신고당한 유저가 정지된다면 결과를 메일로 받는 문제
단 한 유저가 동일한 유저를 계속 신고할 수 없다(중복 x)
최종적으로 유저들이 각각 받게 되는 메일 횟수를 구하는 문제

### 입력
- String[] id_list : 유저 목록이 담긴 String[] 배열
- String[] report : 신고자 신고된 유저 목록이 담긴 String[] 배열
- int k : 정지 신고 기준 정수

### 출력
유저들이 신고 후 정지된 결과를 받는 메일 수
<hr>

### 핵심 풀이 전략
1. Set 자료형을 활용하여 report 내의 중복을 제거한다.
2. 신고를 몇 번 당했는지와, 신고를 한 뒤 메일을 몇 번 받을지를 Map 자료구조를 통해 선언한다.
3. 중복이 제거되었으므로 해당 신고목록을 돌면서 신고 당한 사람의 value 값을 증가시킨다. (초기값은 0)
4. 신고목록을 다 돈 후, 만약 유저가 신고 당한 횟수가 k값보다 크거나 같으면 해당 유저를 신고한 유저가 받을 메일 횟수를 증가시킨다. (초기값은 0)
5. 메일 목록 (map)을 돌면서 value 값을 정답에 추가한다
---

```java
import java.util.*;
class Solution {
    public int[] solution(String[] id_list, String[] report, int k) {
    
        int[] result = new int[id_list.length];
        
        //몇 번 신고당했는지 매핑하는 map 자료구조 선언
        Map<String, Integer> map = new HashMap<>();
        
        //메일을 몇 통 받았는지 매핑하는 map 자료구조 선언
        Map<String, Integer> mail = new HashMap<>();
        
        //신고 목록에서 중복을 제거하기 위한 set 자료구조 선언 
        Set<String> set = new HashSet<>();
         
        //초기값 세팅
        for(int i = 0; i < id_list.length; i++){
            map.put(id_list[i], 0);
            mail.put(id_list[i], 0);
        }
        
        //신고한 조합이 중복으로 저장되지 않게 변환
        for(int i = 0; i < report.length; i++){
            set.add(report[i]);
        }
        
        //중복 제거된 신고 기준으로 피신고자가 신고당한 횟수를 누적
        for(String str : set){
            String [] split = str.split(" ");
            String to = split[1];
            
            map.put(to, map.get(to) +1);
        }
        
        //신고를 당한 횟수가 다 저장되었으니 돌면서 정지 대상을 신고한 사람의 메일 수를 증가
        for(String str : set){
            String [] split = str.split(" ");
            String from = split[0];
            String to = split[1];
            
            //만약 신고 당한 횟수가 k보다 같거나 많다면 해당 사람을 신고한 사람이 받을 메일을 추가
            if(map.get(to) >= k){
                mail.put(from, mail.get(from) +1);
            }
        }
        //반복문을 돌면서 받을 메일 횟수를 꺼냄
        for(int i = 0; i < id_list.length; i++){
            result[i] = mail.get(id_list[i]);
        }     
        return result;
    }
}


```

### 삽질 기록 🧠
- 처음 for(String s : set)에서 신고 당한 횟수가 k보다 같거나 많다면 해당 사람을 신고한 사람이 받을 메일을 추가하는 로직을 함께 돌렸었는데
- 이렇게 되면 신고 당한 횟수가 k번을 넘게되면 메일 횟수가 중복으로 계속 추가될 수 있기 때문에 오류가 났었다.
