### 문제 설명 (요약)
고정된 찍기 정답이 정해져있는 수포자 3명과 정답이 순서대로 들어있는 배열 answers[]가 주어져있을 때
가장 많은 문제를 맞힌 사람이 누구인지 리턴

### 입력
answers[] : 정답이 들어있는 int 배열

### 출력
가장 많이 정답을 맞춘 수포자의 번호가 담긴 배열 (높은 점수를 받은 사람이 여럿일 경우, 오름차순으로 정렬)
<hr>

### 핵심 풀이 전략
1. 수포자들의 찍기는 고정되어 있어서 각각 수포자들의 찍기 정답이 담긴 배열을 선언
2. 반복문을 돌면서 answers[]배열과 수포자들의 정답 배열의 길이가 다르기 때문에 %연산을 통해 계산
3. 만약 수포자들이 찍은 번호와 정답이 일치한다면 해당 수포자의 score 값을 증가
4. 반복문이 끝난 후 Math.max()를 활용하여 어떤 수포자가 제일 맞췄는지 계산
---

```java

import java.util.*;
class Solution {
    public int[] solution(int[] answers) {
        
        //수포자들의 찍기는 고정되어 있어서 고정된 값을 가진 배열 선언
        int [] math1 = {1,2,3,4,5};
        int [] math2 = {2,1,2,3,2,4,2,5};
        int [] math3 = {3,3,1,1,2,2,4,4,5,5};
        
        //각 수포자들의 맞춘 횟수를 저장
        int[] score = new int[3];
        
        //가장 많이 맞춘 수포자들의 번호 리스트
        List<Integer> result = new ArrayList<>();
        
        for(int i = 0; i < answers.length; i++){
           
        int answer = answers[i];
        
        if (math1[i % math1.length] == answer){
            score[0]++; }
        if (math2[i % math2.length] == answer){
            score[1]++; }
        if (math3[i % math3.length] == answer){
            score[2]++; }
        }
        //수포자 중 가장 많이 맞춘 점수 찾기 
        int max = Math.max(score[0], Math.max(score[1], score[2]));
        
        //최고 점수를 가진 수포자의 번호를 결과 리스트에 추가
         for(int i = 0; i < 3; i++){
            if(score[i] == max){
                result.add(i + 1);
            }
        }
         
        return result.stream().mapToInt(Integer::intValue).toArray();
    }
}


```

### 삽질 기록 🧠

- Math.max() 메서드를 활용할 때 처음에 Math.max(score[0], score[1], score[2])와 같은 형태로 사용했는데
- Math.max() 메서드는 인수를 2개를 받는 메서드여서 오류가 났었다.
