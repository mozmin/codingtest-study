### 문제 설명
수포자 3명이 각각 정해진 규칙대로 문제의 답을 찍는다.
실제 정답 배열 `answers`가 주어졌을 때, 가장 많은 문제를 맞힌 사람들의 번호를 오름차순으로 반환한다.
(동점이면 여러 명을 함께 반환)

### 입력
- `answers`: 1번 문제부터 마지막 문제까지의 정답이 담긴 int 배열

### 출력
- 가장 높은 점수를 받은 수포자 번호를 담은 int 배열
- 정답자는 1,2,3 중 하나 이상이며 오름차순 반환

### 핵심 풀이 전략
1. 각 수포자의 찍기 패턴을 배열로 저장한다.
2. `answers`를 처음부터 끝까지 순회하면서, `i%패턴길이`로 패턴을 반복 적용해 정답과 비교
3. 맞히면 해당 수포자의 점수(`ans`)를 증가
4. 점수 배열 `ans`에서 최댓값 `max`를 구한다.
5. `max`와 같은 점수를 가진 사람만 골라 결과 배열에 담아 반환한다.

### 제출 코드
```java
import java.util.*;

class Solution {
    public int[] solution(int[] answers) {

        // 수포자들이 찍는 방식
        int[] math1 = {1, 2, 3, 4, 5};
        int[] math2 = {2, 1, 2, 3, 2, 4, 2, 5};
        int[] math3 = {3, 3, 1, 1, 2, 2, 4, 4, 5, 5};
        
        int[] ans = new int[3];

        // 정답 개수 구하기
        for(int i = 0 ; i < answers.length; i++){
            if(math1[i%5] == answers[i]) ans[0]++;
            if(math2[i%8] == answers[i]) ans[1]++;
            if(math3[i%10] == answers[i]) ans[2]++;
        }

        // 가장 높은 점수 구하기 
        int max = 0;
        for(int i = 0 ; i < 3; i++){
            if(ans[i] > max) max = ans[i];
        }

        // 가장 높은 점수 받은 학생 수 구하기
        int count = 0; 
        for(int i = 0 ; i < 3; i++){
            if(ans[i] == max) count++;
        }

        // 가장 높은 점수 받은 학생 answer 배열에 넣기
        int[] answer = new int[count];
        int k = 0;
        for(int i = 0 ; i < 3; i++){
            if(ans[i] == max){
                answer[k] = i+1;
                k++;
            }
        }
        
        return answer;
    }
}

```
