`#DFS`

### 제출 일자

2026년 3월 11일

### 문제 설명 (요약)
음이 아닌 정수들을 적절히 더하거나 빼서 타겟 넘버를 몇 번 만들 수 있는지 횟수를 새는 문제

### 입력
- int[] numbers : n개의 음이 아닌 정수들이 담겨있는 정수 배열
- int target : 만들어야 하는 목표 숫자

### 출력
- numbers 배열 안에 있는 정수들을 더하거나 빼서 target과 같은 숫자를 만든 횟수
<hr>

### 핵심 풀이 전략
- 최단거리가 아닌, 모든 경우의 수를 찾는 문제이기 때문에 DFS로 접근한다.
- 재귀함수를 통해 숫자를 더하거나 뺄 때를 모두 검사한다.
- dfs 메서드 호출 도중 numbers 배열의 길이 (숫자의 끝)과 깊이(메서드를 호출한 수)가 같아지면 정답 횟수를 증가시키고 아니라면 증가시키지 않고 리턴한다. 
---

```java
class Solution {
    
    int answer = 0;

    public int solution(int[] numbers, int target) {
        dfs(numbers, target, 0, 0);
        return answer;
    
    }
    
    public void dfs(int[] numbers, int target, int sum , int depth){
        if(depth == numbers.length && sum == target){
            answer++;
            return;
        } else if (depth == numbers.length) {
            return;
        }
        
        dfs(numbers, target, sum + numbers[depth], depth +1);
        dfs(numbers, target, sum - numbers[depth], depth +1);
    }
}
```
