### 문제 설명 (요약)
검사지와 선택지 결과 바탕으로 가장 높은 점수를 얻은 성격 유형을 구하기

### 입력
String[] survey : 검사지가 들어있는 배열 <br>
int[] choices : 선택지가 들어있는 배열

### 출력
가장 높은 점수를 얻은 성격 유형
<hr>

### 핵심 풀이 전략
1. 사전 순으로 빠른 유형을 제출해야 하는데, 어짜피 성격 유형의 조합은 정해져있으니 미리 사전 순으로 배열을 만들어 정리해둔다.
2. 조건문 기준을 4로 세팅한 뒤, 해당 기준에 따라 조건식을 검사한다.
---

```java
import java.util.*;
class Solution {
    public String solution(String[] survey, int[] choices) {
        int number = survey.length;
        
        Map<String, Integer> map = new HashMap<>();
        
        String [] str = {"RT", "CF", "JM", "AN"};
        
        String answer = "";
        
        for(int i = 0; i < number; i++){
            String[] array = survey[i].split("");
            
            int choice = choices[i];
                
            if(choice > 4){
                map.put(array[1], map.getOrDefault(array[1], 0)+(choice - 4));
            } else if(choice <= 3){
                map.put(array[0], map.getOrDefault(array[0], 0)+(4 - choice));
            } else {
                continue;
            }
        }
        
        for(String s : str){
            String [] ex = s.split("");
            String first = ex[0];
            String second = ex[1];
            
            if(map.getOrDefault(first, 0) >= map.getOrDefault(second, 0)){
                answer += first;
            } else {
                answer += second;
            }
        }
        return answer;
    }
}
```

### 배운 점 && 보완할 점
map.get()을 통해 접근할 때 해당 map 안에 없는 키를 찾는다면 NullPointerException이 발생한다는 점을 인지하게 되었다.
이를 해결하기 위해서는 map의 키에 해당하는 값들을 0으로 초기화 시키고 시작하는 방법을 쓰면 좋겠다라는 생각이 들었다.
