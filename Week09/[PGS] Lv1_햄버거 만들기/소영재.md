### 문제 설명 (요약)
- 정해진 순서 (빵 - 야채 - 고기 - 빵)으로 햄버거를 만든다.
- 최종적으로 햄버거를 만든 개수를 반환한다.

### 입력
- int [] ingredient : 햄버거에 들어갈 재료가 순차적으로 담긴 정수형 배열

### 출력
- 햄버거를 만든 개수

<hr>

### 핵심 풀이 전략
1. 재료를 순서대로 하나씩 리스트에 추가한다.
2. 재료를 추가할 때마다 리스트의 마지막 4개의 값이 1,2,3,1인지 확인한다.
3. 만약 해당 패턴이 발견되면, 마지막 4개의 요소를 제거하고 햄버거 개수를 증가시킨다.
4. 모든 재료를 처리할 때까지 반복한다.
---

```java
import java.util.*;
class Solution {
    public int solution(int[] ingredient) {
        int answer= 0;

        List<Integer> list = new ArrayList<>();


        for(int i = 0; i < ingredient.length; i++){
            list.add(ingredient[i]);

            int size = list.size() -1;

            if(size >= 3){
                if(list.get(size) == 1 && list.get(size - 1) == 3 && list.get(size - 2 ) == 2 && list.get(size - 3) == 1){
                    list.remove(size);
                    list.remove(size -1);
                    list.remove(size -2);
                    list.remove(size -3);
                    answer ++;

                }
            }
        }
        return answer;
    }
}
```

### 삽질 기록 🧠
```java
import java.util.*;
class Solution {
public int solution(int[] ingredient) {
        int answer= 0;

        List<Integer> list = new ArrayList<>();
        
        for(int i : ingredient){
            list.add(i);
        }
        
        for(int i = 0; i <= list.size() -4 ; i++){
            if(list.get(i) == 1 && list.get(i+1) == 2 && list.get(i+2) == 3 && list.get(i+3) == 1){
                list.remove(i+3);
                list.remove(i+2);
                list.remove(i+1);
                list.remove(i);
                answer ++;
                i = -1;
            }
        }
        return answer;
    }
}
```
- 시간 초과가 났다.
- 의도 : 햄버거가 만들어지면 1,2,3,1 순차적으로 만들어져야 하니, 해당 하는 값을 찾게되면 삭제 후 다시 리스트를 정렬하여 처음부터 또다른 햄버거를 찾는다.
- GPT에게 물어보니 list를 지울 때마다 리스트를 다시 앞으로 땡겨오는 로직이 시간이 오래 걸린다고 하였다. 햄버거가 한 개 만들어질 때마다 리스트를 4번씩 땡겨야 하니 더 오래 걸렸던 것 같다.
- GPT에게 힌트만 얻어서(중간이 아닌, 뒤에서 부터 찾으면 시간을 단축시킬 수 있다.) 코드를 변경했다.

### 배운 점 && 보완할 점
- 틀린 정답이지만 list.remove(i), list.remove(i+1)..순으로 지웠는데 앞부터 지우게 되면 리스트 순서가 꼬여 원하는대로 다시 리스트를 정렬할 수 없다는 점을 알게 되었다.
