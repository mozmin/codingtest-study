# [LV2] 보물 지도

[문제 링크](https://school.programmers.co.kr/learn/courses/15009/lessons/121690)

`#BFS`

### 제출 일자

### 문제 설명 (요약)
- n x m 크기의 격자에서 (1,1)에서 출발하여 (n,m)까지 이동해야 한다.
- 격자에는 함정이 존재하며, 해당 칸은 지나갈 수 없다.
- 단, 한 번만 사용할 수 있는 마법 신발을 이용하면 한 번에 두 칸을 이동할 수 있고, 이때 중간에 있는 함정은 무시할 수 있다.
- 보물 위치(n,m)까지 도달하는 최소 시간을 구하며, 도달할 수 없다면 -1을 반환한다.

### 입력
- int n : 보물 지도의 가로 길이
- int m : 보물 지도의 세로 길이
- int[][] hole : 함정의 위치가 담겨있는 2차원 정수 배열

### 출력
- 보물이 있는 칸으로 이동하는데 필요한 최소 시간을 반환, 단 보물이 있는 칸으로 이동할 수 없다면 -1을 반환
<hr>

### 핵심 풀이 전략
1. 최단 시간을 구하는 문제이므로 BFS를 사용한다.
2. 같은 위치라도 신발 사용 여부에 따라 상태가 달라지므로 visited[x][y][신발 사용 여부] 형태의 3차원 방문 배열을 사용한다.
3. 이동을 일반 이동, 점프 이동으로 나눠서 계산한다.
4. BFS 탐색 시 현재 위치에서 일반 이동 경로, 점프 이동 경로를 모두 queue에 넣어 모든 가능한 경로를 동시에 탐색한다.
5. n,m에 처음 도달하는 순간이 최소 시간이므로 반환한다.
6. 모든 경우를 탐색했음에도 도달하지 못하면 -1을 반환한다.
---

```java
import java.util.*;

class Solution {
    public int solution(int n, int m, int[][] hole) {

        boolean[][][] visited = new boolean[n][m][2];
        //지도를 담는 2차원 배열
        int[][] map = new int[n][m];

        Deque<int[]> queue = new ArrayDeque<>();

        //방향을 이동하기 위한 배열 변수 선언
        int[] nx = {1, 0, -1, 0};
        int[] ny = {0, -1, 0, 1};

        //지도 내에 함정을 표시
        for (int i = 0; i < hole.length; i++) {
            int a = hole[i][0] - 1;
            int b = hole[i][1] - 1;
            map[a][b] = 1;
        }

        //시작점 위치를 queue에 저장
        queue.offer(new int[]{0, 0, 0, 1});

        //0,0을 신발을 아직 사용하지 않은 상태로 방문
        visited[0][0][1] = true;

        while (!queue.isEmpty()) {
            int[] cur = queue.poll();

            int x = cur[0];
            int y = cur[1];
            int dist = cur[2];
            int shoes = cur[3];

            //종료조건, 보물의 위치에 도달했다면 해당 거리를 리턴
            if (x == n - 1 && y == m - 1) {
                return dist;
            }
            // 상하좌우 각각에 대해 탐색 시작
            for (int i = 0; i < 4; i++) {
                //혀재 위치에서 한 칸 이동한 좌표를 계산
                int nextX = x + nx[i];
                int nextY = y + ny[i];

                //일반적으로 이동하는 조건
                //1. 격자 범위를 벗어나면 안됨
                //2. 함정이 아니어야 함
                //3. 같은 신발 상태로 방문한 적이 없어야 함
                if (nextX >= 0 && nextX < n && nextY >= 0 && nextY < m && map[nextX][nextY] == 0 && !visited[nextX][nextY][shoes]) {
                    visited[nextX][nextY][shoes] = true;
                    queue.offer(new int[]{nextX, nextY, dist + 1, shoes});
                }
                //마법의 신발이 있을 때 이동하는 조건
                //1. 신발이 있어야 함
                //2. 점프를 뛴 곳이 격자 범위를 벗어나면 안됨, 함정이 아니어야 함
                //3. 방문한적이 없어야 함
                if (shoes == 1) {
                    int jumpX = x + nx[i] * 2;
                    int jumpY = y + ny[i] * 2;

                    if (jumpX >= 0 && jumpX < n && jumpY >= 0 && jumpY < m && map[jumpX][jumpY] == 0 && !visited[jumpX][jumpY][0]) {
                        //방문체크와 동시에 신발 여부 변경
                        visited[jumpX][jumpY][0] = true;
                        queue.offer(new int[]{jumpX, jumpY, dist + 1, 0});

                    }
                }
            }
        }

        return -1;
    }
}
```
