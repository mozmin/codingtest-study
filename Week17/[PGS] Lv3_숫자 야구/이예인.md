### 문제 설명
- 1000부터 9999까지의 네 자리 정수 중에서, 주어진 제출 함수 submit에 넣었을 때 특정 결과를 반환하는 숫자를 찾는 문제이다.
- submit 함수는 숫자를 입력받아 문자열 결과를 반환한다.
- 이때 결과가 "4S 0B"라면 정답 숫자를 정확히 맞힌 상태를 의미하므로, 해당 숫자를 찾아 반환하면 된다.

### 입력
- 정수 n
- 숫자를 입력하면 판정 결과 문자열을 반환하는 함수 submit

### 출력
- submit 함수에 넣었을 때 "4S 0B"를 반환하는 네 자리 정수

---

### 핵심 풀이 전략
- 가능한 네 자리 수의 범위는 1000부터 9999까지이다.
- 범위가 크지 않기 때문에 모든 숫자를 하나씩 확인하는 완전탐색으로 접근할 수 있다.
- 각 숫자를 submit 함수에 넣어 결과 문자열을 확인한다.
- 결과가 "4S 0B"와 같다면 해당 숫자가 정답이므로 바로 반환한다.
- 모든 숫자를 확인했는데도 조건을 만족하는 값이 없다면 기본값으로 0을 반환한다.

---

### 제출 코드
```java
import java.util.function.Function;

class Solution {
    public int solution(int n, Function<Integer, String> submit) {
        for(int i = 1000; i <= 9999; i++)
            if (submit.apply(i).equals("4S 0B")) return i;
        return 0;
    }
}
```
