### 문제 설명
- 동영상 재생 위치와 명령어 목록이 주어졌을 때, 모든 명령어를 수행한 뒤 최종 재생 위치를 구하는 문제이다.
- 명령어는 다음 두 가지가 있다.
  - next: 현재 위치에서 10초 뒤로 이동
  - prev: 현재 위치에서 10초 앞으로 이동

- 단, 이동 과정에서 다음 조건을 만족해야 한다.
- 현재 위치가 오프닝 구간 안에 있다면 오프닝이 끝나는 위치로 이동한다.
- prev 명령어를 수행했을 때 0초보다 작아지면 0초로 이동한다.
- next 명령어를 수행했을 때 동영상 길이를 넘어가면 동영상의 마지막 위치로 이동한다.
- 모든 명령어 수행 후에도 위치가 오프닝 구간 안에 있다면 오프닝이 끝나는 위치로 이동한다.

### 입력
- 동영상 길이 video_len
- 현재 재생 위치 pos
- 오프닝 시작 시각 op_start
- 오프닝 종료 시각 op_end
- 명령어 배열 commands

### 출력
- 모든 명령어를 수행한 뒤의 최종 재생 위치
- 형식은 mm:ss

---

### 핵심 풀이 전략
- mm:ss 형식의 시간을 계산하기 쉽게 초 단위 정수로 변환한다.
- 동영상 길이, 현재 위치, 오프닝 시작 시간, 오프닝 종료 시간을 모두 초 단위로 바꾼다.
- 각 명령어를 수행하기 전에 현재 위치가 오프닝 구간 안에 있는지 확인한다.
- 현재 위치가 오프닝 구간에 포함되면 오프닝 종료 시각으로 이동시킨다.
- 명령어가 next이면 현재 위치에 10초를 더한다.
- 명령어가 prev이면 현재 위치에서 10초를 뺀다.
- 이동 결과가 동영상 범위를 벗어나지 않도록 0초와 동영상 길이 사이로 보정한다.
- 모든 명령어를 처리한 뒤 마지막으로 한 번 더 오프닝 구간 여부를 확인한다.
- 최종 위치를 다시 mm:ss 형식으로 변환해 반환한다.

### 제출 코드
```java
import java.util.*;

class Solution {
    public String solution(String video_len, String pos, String op_start, String op_end, String[] commands) {
        int video_len_int = changeToSecond(video_len);
        int pos_int = changeToSecond(pos);
        int op_start_int = changeToSecond(op_start);
        int op_end_int = changeToSecond(op_end);
        
        for(int i = 0; i < commands.length; i++){
            if(pos_int >= op_start_int && pos_int <= op_end_int) pos_int = op_end_int;
            if(commands[i].equals("next")){
                pos_int += 10;
                if(pos_int >= video_len_int) pos_int = video_len_int;
            }
            else if(commands[i].equals("prev")){
                pos_int -= 10;
                if(pos_int <= 0) pos_int = 0;
            }
        }
        if(pos_int >= op_start_int && pos_int <= op_end_int) pos_int = op_end_int;
        String min_string = String.valueOf(pos_int / 60);
        String sec_string = String.valueOf(pos_int % 60);
        if(min_string.length() <= 1) {
            String temp = "0";
            min_string = temp.concat(min_string);
        }
        if(sec_string.length() <= 1){
            String temp = "0";
            sec_string = temp.concat(sec_string);
        }
        
        String temp2 = ":";
        String answer = min_string.concat(temp2);
        answer = answer.concat(sec_string);
        return answer;
        
    }
    
    int changeToSecond(String time){
        String minute = time.substring(0,2);
        String second = time.substring(3,5);
        
        int minute_i = Integer.parseInt(minute);
        int second_i = Integer.parseInt(second);
        
        return minute_i * 60 + second_i;
    }
}
```
