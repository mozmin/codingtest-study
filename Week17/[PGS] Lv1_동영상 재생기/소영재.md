### 문제 설명 (요약)
- 동영상의 현재 재생 위치에서 "next" 명령이 들어오면 10초 앞으로 이동하고, "prev" 명령이 들어오면 10초 뒤로 이동한다.
- 단, 동영상의 범위를 벗어날 수 없으며 현재 위치가 오프닝 구간에 포함되어 있다면 자동으로 오프닝 종료 시점으로 이동해야 한다.
- 모든 명령을 수행한 뒤 최종 재생 위치를 "mm:ss" 형식으로 반환하는 문제이다.

### 입력
- String video_len : 동영상 전체 길이
- String pos : 현재 재생 위치
- String op_start : 오프닝 시작 위치
- String op_end : 오프닝 종료 위치
- String[] commands : "next" 또는 "prev" 명령이 들어있는 배열 

### 출력
- 모든 명령을 수행 후 최종 재생 위치를 "mm:ss" 형식으로 변환

<hr>

### 핵심 풀이 전략
1. 문자열 형태의 시간을 모두 초(second) 단위 정수로 변환한다.
2. 현재 위치가 오프닝 구간에 포함되어 있다면 즉시 오프닝 종료 지점으로 이동한다.
3. 명령 배열을 순회하면서 처리한다.
4. 최종 초 단위를 "mm:ss" 형식으로 변환하여 반환한다.
---

```java
class Solution {
    public String solution(String video_len, String pos, String op_start, String op_end, String[] commands) {
        
        int videoLen = changeSecond(video_len);
        int currentPos = changeSecond(pos);
        int opStart = changeSecond(op_start);
        int opEnd = changeSecond(op_end);
        
        if (currentPos >= opStart && currentPos <= opEnd) {
            currentPos = opEnd;
        }
        
        for(int i = 0; i < commands.length; i++){
            if(commands[i].equals("next")){
                currentPos = Math.min(currentPos + 10, videoLen);
            }
            if(commands[i].equals("prev")){
                currentPos = Math.max(currentPos - 10, 0);
            }
            
            if (currentPos >= opStart && currentPos <= opEnd) {
                currentPos = opEnd;
            }
        }
        
        String min = String.valueOf(currentPos / 60);
        String second = String.valueOf(currentPos % 60);
            
        if(currentPos / 60 < 10){
            min = "0" + String.valueOf(currentPos / 60);
        }
        
        if(currentPos % 60 < 10){
            second = "0" + String.valueOf(currentPos % 60);
        }
        return min + ":" + second;
    }
    
    public int changeSecond(String time){
        String[] target = time.split(":");
        return Integer.parseInt(target[0])*60  + Integer.parseInt(target[1]); 
    }
}
```
