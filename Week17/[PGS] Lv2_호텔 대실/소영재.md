### 문제 설명 (요약)
- 호텔 객실 예약 시간이 주어졌을 때, 각 예약 종료 후 청소 시간 10분을 고려하여 모든 예약을 처리하는 데 필요한 최소 객실 수를 구하는 문제

### 입력
- String[][] book_time : 예약 시작 시간과 종료 시간이 담긴 2차원 문자 배열

### 출력
- 모든 예약을 처리하기 위해 필요한 최소한의 객실 수

<hr>

### 핵심 풀이 전략
1. 각 시간을 분 단위 정수로 변환한다.
2. 예약 시작 시간에는 객실 사용이 시작되므로 +1, 종료 시간 + 청소 10분이 끝나는 시점에는 -1을 기록한다.
3. 누적된 최댓값이 모든 예약을 처리하기 위해 필요한 최소 객실 수가 되므로 이를 반환한다.

---

```java (정답 코드 참고)
import java.util.*;
class Solution {
    public int solution(String[][] book_time) {
        int[] time = new int[24 * 60 +10];
        
        int answer = 0;
        
        for(String[] book : book_time){
            int startTime = changeMinute(book[0]);
            int endTime = changeMinute(book[1]) + 10;
            
            time[startTime] ++;
            time[endTime]--;
        }
        
        int room = 0;
        
        for(int i = 0; i < time.length; i++){
            room += time[i];
            answer = Math.max(answer, room);
        }
         return answer;
        
    }
    
    public int changeMinute(String target){
        String [] time = target.split(":"); 
        return Integer.parseInt(time[0]) * 60 + Integer.parseInt(time[1]);
    }
}
```

### 배운 점 && 보완할 점
- 누적합(Difference Array) 기법을 사용하면 구간 전체를 직접 순회하지 않고도 겹치는 구간의 개수를 구할 수 있다.
- 처음에는 예약 구간 전체를 직접 time[] 배열 안에 표시해야 한다고 생각했지만, 누적합 기법을 이용하면 시작점과 종료점만 기록해도 동일한 결과를 얻을 수 있다는 점을 알게 되었다.
