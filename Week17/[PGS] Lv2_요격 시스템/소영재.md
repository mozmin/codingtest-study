### 문제 설명 (요약)
- 각 미사일은 [s,e] 구간을 지나가며, 요격 미사일을 특정 x좌표에 발사하면 해당 좌표를 지나는 모든 미사일을 한 번에 요격할 수 있다.
- 주어진 모든 미사일을 요격하기 위해 필요한 요격 미사일의 최소 개수를 구하는 문제이다.

### 입력
- int[][] targets : 각 미사일의 시작 지점과 끝 지점이 담긴 2차원 정수 배열

### 출력
- 모든 미사일을 요격하기 위해 필요한 최소 요격 미사일 개수

<hr>

### 핵심 풀이 전략
1. 미사일을 끝 지점을 기준으로 오름차순 정렬하낟.
2. 가장 먼저 끝나는 미사일부터 확인한다.
3. 현재 요격 위치(currentPoint)가 다음 미사일 구간에 포함되지 않는다면 새로운 요격 미사일을 발사한다.
4. 현재 요격 위치가 다음 미사일 구간에 포함된다면 기존 요격 미사일로 처리한다.
5. 모든 미사일을 순회하며 필요한 요격 미사일 개수를 반환한다.

---

```java
import java.util.*;
class Solution {
    int currentPoint = 0;
    
    public int solution(int[][] targets) {
        Arrays.sort(targets, (a,b) -> a[1] - b[1]);
        
        int answer = 0;
        for(int[] target : targets){
            
            int startPoint = target[0];
            int endPoint = target[1];
            
            if (currentPoint == 0) {
                answer++;
                currentPoint = endPoint;
                continue;
            }

            if(currentPoint <= startPoint){
                answer++;
                currentPoint = endPoint;
            }
            
        }
        return answer;
    }
}
```
