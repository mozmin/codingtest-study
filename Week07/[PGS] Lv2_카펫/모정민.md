### 문제 설명 (요약)
중앙에는 노란색 격자가 칠해져 있고, 테두리 1줄은 갈색 격자로 칠해진 직사각형 형태의 카펫이 있다.

갈색 격자의 수와 노란색 격자의 수가 주어질 때, 전체 카펫의 가로, 세로 크기를 구해야 한다.

- 전체 카펫의 넓이는 `brown + yellow`와 같다.
- 전체 넓이의 약수들을 이용하여 가로, 세로 길이를 유추한다. (가로 길이는 세로 길이보다 같거나 길다.)
- 유추한 가로, 세로 길이를 바탕으로 **노란색 영역의 크기**가 `(가로 - 2) * (세로 - 2)`와 일치하는지 확인한다.

### 입력
- brown: 갈색 격자의 수 (8 이상 5,000 이하)
- yellow: 노란색 격자의 수 (1 이상 2,000,000 이하)

### 출력
카펫의 가로, 세로 크기를 순서대로 배열에 담아 `[가로, 세로]` 형태로 반환

### 핵심 풀이 전략
1. i: 1 ~ yellow/2 + 1 까지 반복을 돌며 yellow의 약수인지 확인
2. yellow / i 를 통해 가로 길이 계산
3. (가로 x 2) + (세로 x 2) + 4 = brown 인지 확인 -> true 라면, brown의 각 가로 세로는 yellow의 가로 세로의 +2 씩 계산

---

### Java 구현 코드

```java
class Solution {
    public int[] solution(int brown, int yellow) {
        int[] answer = new int[2];
        
        for(int i = 1; i <= yellow/2 + 1; i++){
            
            // 세로: i
            if(yellow % i == 0){
                int row = yellow / i; // 가로: yellow / i
                
                int checkBrown = (row * 2) + (i * 2) + 4;
                if(checkBrown == brown){
                    answer[0] = row + 2;
                    answer[1] = i + 2;
                    break;
                }
            }
        }
        return answer;
    }
}
```

### 삽질 기록 🧠
1. i 반복 돌 때, i < yellow / n 라고 하여 yellow가 1인 경우 반복을 돌지 못하였음. 또한 <= 로 해야 1부터 시작하는 반복문이 제대로 실행되기에 i <= yellow +1 로 수정함.
