### 문제 설명 (요약)
- Leo가 본 카펫에서 갈색 격자의 수 (brown), 노란색 격자의 수 (yellow)를 바탕으로 카펫의 가로,세로 크기를 구하는 문제
- 갈색 격자는 테두리를 의미하며, 노란색 격자는 내부의 격자를 의미한다.
- 전체 카펫의 크기는 brown + yellow이다.

### 입력
- int brown : 갈색 격자의 수
- int yellow : 노란색 격자의 수

### 출력
- 카펫의 가로, 세로 크기를 순서대로 배열에 담아 출력
<hr>

### 핵심 풀이 전략
1. brown과 yellow를 더해 카펫의 총 크기를 알아낸다.
2. 반복문을 돌며 가로 세로의 약수쌍을 구한 뒤, yellow값을 찾는다
3. yellow의 값은 항상 가로 세로 길이에서 상하좌우로 1씩 모자르기 때문에 이를 활용하여 yellow 값을 계산한다.
4. yellow 값이 도출된 시점의 가로, 세로 길이를 반환한다.
---

```java
class Solution {
    public int[] solution(int brown, int yellow) {
        int[] answer = new int[2];
        int sum = brown + yellow;
        
        for(int i = 1; i <= sum; i++){
            int height = i;
            
            //정수로 떨어지지 않는다면 확인하지 않음
            if(sum % height !=0){
                continue;
            }
            //가로 구하기
            int weight = sum / height;
            
            //yellow를 구하는 코드, weight -2, height-2한 값이 내부 칸 수
            if(((weight-2) * (height-2)) == yellow){
               answer[0] = weight;
               answer[1] = height;
               break;
            }          
        }
        return answer;
    }
}
```
