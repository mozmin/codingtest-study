### 문제 설명
송전탑이 `n`개 있고, 전선 정보가 `wires`로 주어진다. <br>
전선 하나를 끊어서 전력망을 둘로 나눴을 때,  
두 전력망에 속한 송전탑 개수 차이의 최솟값을 구하는 문제이다. <br>

각 전선은 트리 구조로 연결되어 있으므로,  
전선 하나를 제거하면 정확히 두 개의 연결 요소로 나뉘게 된다. <br>

### 입력
- `n` : 송전탑의 개수
- `wires` : 연결된 두 송전탑 번호가 담긴 2차원 배열

### 출력
- 전선 하나를 끊었을 때 나뉘는 두 전력망의 송전탑 수 차이의 최솟값

### 핵심 풀이 전략
모든 전선을 하나씩 끊어보면서,  
끊어진 상태에서 한 쪽 전력망의 크기를 BFS로 구한 뒤  
나머지 전력망과의 크기 차이를 계산했다. <br>

1. `connect[a][b] = 1` 형태의 인접 행렬로 전력망 연결 상태를 저장
2. 각 전선을 하나씩 제거
3. 제거된 상태에서 한 송전탑을 시작점으로 BFS 수행
4. BFS로 연결된 송전탑 개수 `count` 계산
5. 다른 한 쪽 전력망의 크기는 `n - count`
6. 두 전력망의 차이 `Math.abs(n - 2 * count)`를 구해 최솟값 갱신
7. 다음 경우를 위해 제거했던 전선을 다시 복구

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(int n, int[][] wires) {
        int answer = n;
        int[][] connect = new int[n+1][n+1];
        for(int i = 0 ; i < wires.length; i++){
            int a = wires[i][0];
            int b = wires[i][1];
            connect[a][b] = 1;
            connect[b][a] = 1;
        }
        for(int i = 0 ; i < wires.length; i++){
            int a = wires[i][0];
            int b = wires[i][1];
            connect[a][b] = 0;
            connect[b][a] = 0;
            
            int count = bfs(connect, a, n);
            int diff = Math.abs(n - 2*count); // count - (n - count)
            answer = Math.min(answer, diff);
            
            connect[a][b] = 1;
            connect[b][a] = 1;
        }  
        return answer;
    }
    
    int bfs(int[][] connect, int a, int n){
        int count = 1;
        int next = -1;
        
        boolean[] visited = new boolean[n+1];
        Queue<Integer> queue = new LinkedList<>();
        visited[a] = true;
        queue.add(a);
        
        while(!queue.isEmpty()){
            int cur = queue.poll();
            
            for(int i = 0 ; i < connect.length; i++){
                if(connect[cur][i] == 1 && visited[i] != true){
                    next = i;
                    visited[i] = true;
                    queue.add(next);
                    count++;
                }
                else continue;          
            }
        }
        return count;     
    }
}

### 삽길 기록 🧠
- 처음에는 두 전력망의 차이를 Math.abs(n - count)로 계산했는데, 이건 한 쪽 전력망 크기만 뺀 값이라 틀렸다.
- 실제로는 한 쪽 크기가 count이면 다른 쪽은 n - count이므로, 차이는 Math.abs(count - (n - count))로 계산해야 했다.
- 그래서 최종적으로 Math.abs(n - 2 * count)로 수정했다.
- BFS 자체는 맞게 구현했는데, 차이 계산식 하나 때문에 결과가 계속 틀렸었다.
    


### 배운 점 && 보완할 점
- 인접 행렬로 풀어서 구현은 단순했지만, 노드 수가 더 커진다면 인접 리스트 방식이 더 효율적일 것 같다.
