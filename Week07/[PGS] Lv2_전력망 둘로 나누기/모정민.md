### 문제 설명 (요약)
n개의 송전탑이 트리 형태로 연결된 전력망 네트워크가 있다. 이 전력망의 전선(간선) 중 하나를 끊어서 전체 네트워크를 2개로 분할하려고 한다.

이때, 나누어진 두 전력망이 갖게 되는 **송전탑의 개수 차이를 최소화**해야 한다.

- 간선을 하나씩 순차적으로 끊어본다.
- 나누어진 두 전력망 중 하나의 송전탑 개수를 탐색(BFS/DFS)하여 센다.
- 두 전력망의 크기 차이의 최솟값을 갱신한다.

### 입력
- n: 송전탑의 개수 (2 이상 100 이하)
- wires: 전선 정보가 담긴 2차원 정수 배열 (형태: `[[v1, v2], ...]`)

### 출력
전선을 하나 끊어서 두 전력망으로 나누었을 때, 두 전력망이 가지고 있는 송전탑 개수의 차이(절댓값) 중 최솟값을 정수로 반환

---

### 핵심 풀이 전략
1. 2차원 boolean 배열 선언 후, wires 정보 저장
2. wires 각 반복마다 연결을 false로 끊고 dfss() 호출
3. 시작 노드는 끊은 wires 중 하나를 index로 넘김
4. index를 기준으로 0~n-1 까지 반복을 돌며 true인지 확인 -> true라면 방문 처리 후 dfs() 재귀 호출
5. dfs로 구한 값과 송전탑의 개수(n)을 활용하여 차이 값 계산
6. wires 각 반복마다 min 값 갱신

---

### Java 구현 코드

```java
import java.util.*;

class Solution {
    
    int diff;
    int min;
    public int solution(int n, int[][] wires) {
        diff = 0;
        min = Integer.MAX_VALUE;
        
        // 연결된 와이어 저장
        boolean[][] list = new boolean[n][n];
        for(int i = 0; i < wires.length; i++){
            int from = wires[i][0] - 1;
            int to = wires[i][1] - 1;
            
            list[from][to] = true;
            list[to][from] = true;
        }
        
        // 각 wire 반복마다 하나씩 끊어가며 dfs() 참조
        for(int i = 0; i < wires.length; i++){
            int from = wires[i][0] - 1;
            int to = wires[i][1] - 1;
            
            list[from][to] = false;
            list[to][from] = false;
            
            boolean[] visited = new boolean[n];
            int count = dfs(list, from, visited);
                
            diff = Math.abs(count - (n - count));
            min = Math.min(min, diff);
            
            list[from][to] = true;
            list[to][from] = true;
        }
        
        return min;
    }
    
    public int dfs(boolean[][] list, int index, boolean[] visited){
        
        int count = 1;
        visited[index] = true;
        for(int i = 0; i < list.length; i++){
            
            if(list[index][i] && !visited[i]){
                count += dfs(list, i, visited);
            }
        }
        
        return count;
    }
}

```

### 삽질 기록 🧠
1. 시작 노드를 정하지 못하여서 dfs()에 이중 for문을 사용하였으나 더 복잡해져서 AI의 도움을 받음.
2. dfs()에서 count를 0으로 하는 바람에 밑단 노드부터 return count가 항상 0으로 고정되는 문제가 생겨버림.
