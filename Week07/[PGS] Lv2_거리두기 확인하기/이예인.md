### 문제 설명
5개의 대기실이 주어지고, 각 대기실은 5×5 크기의 문자열 배열로 표현된다. <br>
각 칸은 다음 셋 중 하나이다. <br>

- P : 응시자가 앉아 있는 자리
- O : 빈 테이블
- X : 파티션

각 대기실마다 모든 응시자가 거리두기 규칙을 지키고 있는지 확인해야 한다. <br>

거리두기 규칙은 다음과 같다. <br>

- 맨해튼 거리 2 이하에 다른 응시자가 있으면 안 된다.
- 단, 사이에 파티션이 적절히 있으면 허용된다.

각 대기실이 거리두기를 지키면 1, 아니면 0을 결과 배열에 담아 반환한다. <br>


### 입력
- places : 5개의 대기실 정보가 담긴 2차원 문자열 배열
- 각 대기실은 길이 5의 문자열 5개로 구성됨
- 각 문자열의 길이는 5

### 출력
- 각 대기실별 거리두기 준수 여부를 담은 정수 배열
- 준수하면 1, 위반이면 0

### 핵심 풀이 전략
각 대기실을 5×5 정수 배열로 변환한 뒤, 모든 P 위치를 기준으로 맨해튼 거리 2 이하의 좌표를 검사했다. <br>

1. 현재 위치가 P이면 주변 좌표 (dx, dy)를 순회
2. 맨해튼 거리 |dx| + |dy| <= 2 인 칸만 확인
3. 그 위치에도 P가 있다면, 두 사람 사이가 거리두기 위반인지 check() 함수로 판별

check()에서는 경우를 3가지로 나누어 처리했다. <br>

- 거리 1인 경우 : 무조건 위반
- 대각선 거리 2인 경우 : 두 중간 칸 중 하나라도 파티션이 아니면 위반
- 일직선 거리 2인 경우 : 중간 칸이 파티션이 아니면 위반

대기실 하나라도 위반이 발견되면 바로 0 처리하고 다음 대기실로 넘어갔다. <br>

### 제출 코드
```java
import java.util.*;

class Solution {
    public int[] solution(String[][] places) {
        int[] answer = new int[places.length];
        for(int i = 0; i < answer.length; i++){
            answer[i] = -1;
        }
        int[][] place = new int[places[0].length][places[0][0].length()];
        
        for(int i = 0 ; i < places.length; i++){
            // P -> 1, O -> 0, X -> -1
            for(int j = 0; j < places[0].length; j++){
                for(int k = 0; k < places[0][0].length(); k++){
                    if(places[i][j].charAt(k) == 'P') place[j][k] = 1;
                    else if (places[i][j].charAt(k) == 'O') place[j][k] = 0;
                    else place[j][k] = -1;                 
                }
            }
            
            outer: for(int j = 0; j < place.length; j++){
                for(int k = 0 ; k < place[0].length; k++){
                    // P를 만났을 때
                    if(place[j][k] == 1){
                        for(int dx = -2 ; dx <= 2 ; dx++){
                            for(int dy = -2; dy <= 2; dy++){
                                if(dx == 0 && dy == 0) continue;
                                // dx + dy 맨해튼 거리 2 이하인지 확인
                                if(Math.abs(dx) + Math.abs(dy) <= 2) {
                                    // 배열 안 위치인지 확인
                                    if(j + dx >= 0 && k + dy >= 0 && j + dx < place.length 
                                       && k + dy < place[0].length){
                                        if(place[j+dx][k+dy] == 1){
                                            boolean checkRes = check(place, j, k, dx, dy);
                                            if(checkRes == false){
                                                answer[i] = 0;
                                                break outer;
                                            }
                                        }
                                    }                                    
                                }
                            }
                        }
                    }
                }
            }            
        }
             
        for(int i = 0 ; i < answer.length; i++){
            if(answer[i] == -1) answer[i] = 1;
        }
        return answer;
    }
    
    boolean check(int[][] place, int j, int k, int dx, int dy){
        // case1 : 거리가 1인 경우 -> 무조건 return 0
        if(Math.abs(dx) + Math.abs(dy) == 1) return false;
         
        // case2: abs(dx) == 1 && abs(dy) == 1인 경우 -> 사이에 X 있는지 확인 (대각선인 경우)
        if(Math.abs(dx) == 1 && Math.abs(dy) == 1){
            if(place[j+dx][k] != -1 || place[j][k+dy] != -1) return false;
        }
        
        // case3: abs(dx) == 2 || abs(dy) == 2인 경우 -> 사이에 X 있는지 확인 (일직선 거리 2)
        if(Math.abs(dx) == 2 && dy == 0){
            if(place[j + dx/2][k] != -1) return false;
        }
        if(Math.abs(dy) == 2 && dx == 0){
            if(place[j][k + dy/2] != -1) return false;
        }  
        
        return true;
    }
}
```

### 삽길 기록 🧠
- 대각선 검사에서 `||` 을 사용해야하는데 처음에 `&&`을 사용해서 틀렸다. (머리로 생각은 대각선 두개를 다 통과해야한다고 생각만 하고 &&를 사용했다)
- 자기자신(`dx==0 && dy ==0`)을 포함해서 검사했었다.
- 인덱스 예외가 터져서 잘못된 부분 찾는라 시간을 많이 썼음..
