### 문제 설명 (요약)
직사각형 격자 형태의 공원에서 로봇 강아지가 산책을 한다.  
공원은 다음 문자로 구성된다.

- **S** : 시작 위치
- **O** : 이동 가능한 길
- **X** : 장애물

산책 경로는 `"방향 거리"` 형태의 명령으로 주어지며, 순서대로 수행된다.

이동 명령을 수행하기 전에 다음을 확인한다.
- 이동 경로가 **공원을 벗어나는지**
- 이동 중 **장애물을 만나는지**

위 두 조건 중 하나라도 만족하면 **해당 명령은 무시하고 다음 명령을 수행한다.**  
모든 명령 수행 후 **최종 위치 좌표 [row, col]을 반환한다.**

### 입력
- park: 공원을 나타내는 문자열 배열
- routes: 이동 명령 배열  
  예: `["E 2", "S 2", "W 1"]`

### 출력
모든 명령 수행 후의 위치를 `[세로 좌표, 가로 좌표]` 형태로 반환

---

### 핵심 풀이 전략
1. **시작 위치 찾기**  
   공원 배열을 순회하며 `'S'` 위치를 찾아 현재 위치 `(x, y)`로 저장한다.

2. **방향 벡터 설정**  
   방향에 따라 이동 좌표를 매핑한다.
   - `N` → `(-1, 0)`
   - `S` → `(1, 0)`
   - `W` → `(0, -1)`
   - `E` → `(0, 1)`

3. **명령 수행 전 이동 가능 여부 체크**  
   한 번에 이동하지 않고 **한 칸씩 이동하면서** 아래를 검사한다.
   - 범위를 벗어나는지
   - 장애물(`X`)을 만나는지

4. **이동 가능하면 최종 위치 갱신**  
   명령 수행 도중 문제가 없을 때만 현재 위치를 갱신한다.

### Java 구현 코드

```java
import java.util.*;

class Solution {
    
    String dir = "NSWE";
    int[] moveX = new int[]{0, 0, -1, 1};
    int[] moveY = new int[]{-1, 1, 0, 0};
    
    public int[] solution(String[] park, String[] routes) {
        
        int[] curPosition = new int[2];
        
        outer: for(int i = 0; i < park.length; i++){
            for(int j = 0; j < park[0].length(); j++){
                char c = park[i].charAt(j);
                
                if(c == 'S'){
                    curPosition[0] = i; // Y
                    curPosition[1] = j; // X
                    break outer;
                }
            }
        }
        
        for(int i = 0; i < routes.length; i++){
        
            String op = routes[i].substring(0, 1);
            int n = Integer.parseInt(routes[i].substring(2));
            
            curPosition = check(op, n, park, curPosition);
            
        }
        
        
        return curPosition;
    }
    
    public int[] check(String op, int n, String[] park, int[] curPosition){
        
        int index = dir.indexOf(op);
        int newY = curPosition[0];
        int newX = curPosition[1];
        
        for(int i = 0; i < n; i++){
            newY += moveY[index];
            newX += moveX[index];
        
            if(newY >= park.length || newY < 0
              || newX >= park[0].length() || newX < 0 || park[newY].charAt(newX) == 'X'){
                return curPosition;
            }
        }
        
        return new int[]{newY, newX};
    }
}
```

### 삽질 기록 🧠
1. x, y 위치 뒤바뀜
- 자료구조상에서는 Y가 앞에, X가 뒤에 온다. 하지만 처음 풀이할 때, X를 앞에 Y를 뒤에 두어서 잘못 탐색하게 되었다.
 
2. park[0]은 String 문자열이다. 즉 length 가 아닌 length()로 작성해야한다.
