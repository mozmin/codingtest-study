### 문제 설명 (요약)
- 공원에서 로봇 강아지가 명령에 따라 산책을 한 후 최종적으로 놓인 위치를 구하는 문제
- 동서남북 주어진 방향으로 이동할 때 공원을 벗어나거나 장애물을 만난다면 이동할 수 없음

### 입력
- String[] park : 공원을 나타내는 문자열 배열
- String[] routes : 로봇 강아지가 수행할 명령이 담긴 문자열 배열

### 출력
- 로봇 강아지가 모든 명령을 수행한 후 놓인 위치[세로 방향 좌표, 가로 방향 좌표]를 리턴
<hr>

### 핵심 풀이 전략
1. 공원을 순회하면서 시작 위치인 'S'를 찾아 (x, y) 좌표로 저장
2. routes 배열을 순회하며 각 명령을 방향(dir)과 이동 거리(count)로 분리
3. 현재 위치를 바로 변경하지 않고 temporaryX, temporaryY를 사용하여 이동 가능 여부를 먼저 확인
4. 이동 거리만큼 반복하면서 공원 범위를 벗어나는지 확인 장애물('X')이 있는지 확인, 하나라도 걸리면 해당 이동은 무효 처리
5. 모든 검사를 통과한 경우에만 현재 위치 (x, y)를 임시 좌표로 업데이트
6. 모든 이동이 끝난 후 최종 좌표를 [y, x] 형태로 반환

---

```java
import java.util.*;
class Solution {
    Map<String, Integer> map = new HashMap<>();
    public int[] solution(String[] park, String[] routes) {
        //정답을 반환할 정수 배열
        int [] answer = new int[2];
        //시작점 찾기
        findStartPoint(park);
        
        //map에 저장된 x,y 값을 꺼내어 저장
        int x = map.get("x");
        int y = map.get("y");
        
        for(int i = 0; i < routes.length; i++){
            //(E 2, W 1) 동서남북과 이동한 거리를 분리하기 위한 코드
            String [] route = routes[i].split(" ");
            String dir = route[0];
            int count = Integer.parseInt(route[1]);
            
            //현재 위치인 x,y값을 사용하게 되면 값을 증가시키고 조건에 맞지 않을 경우 다시 되돌리는 것이 어렵기 때문에 임시 변수를 사용
            int temporaryX = x;
            int temporaryY = y;
            
            //움직일 수 있는지 확인하는 boolean 변수
            boolean canMove = true;
            
            //동서남북 방향에 따라 값 조정
            for(int j = 0; j < count; j++){
                if (dir.equals("E")) {
                    temporaryX++;
                } else if (dir.equals("W")) {
                    temporaryX--;
                } else if (dir.equals("S")) {
                    temporaryY++;
                } else if (dir.equals("N")) {
                    temporaryY--;
                }
                
            //만약 공원의 상하좌우 범위를 벗어난다면 false   
            if (temporaryY < 0 || temporaryY >= park.length || temporaryX < 0 || temporaryX >= park[0].length()) {
                    canMove = false;
                    break;
                }
            
            //만약 이동해야 할 위치에 방해물(X)가 있다면 false
            if (park[temporaryY].charAt(temporaryX) == 'X') {
                    canMove = false;
                    break;
                }                   
            }
            
            //위 두 조건을 만족한다면 실제로 현재 위치를 이동
            if (canMove) {
                x = temporaryX;
                y = temporaryY;
            }
        }
        answer[0] = y;
        answer[1] = x;
        return answer;
    }
    //시작점을 찾는 메서드, 이중 for문을 돌면서 S를 만나게 되면 해당 x,y값을 map에 대입
    public void findStartPoint(String[] park){
        outer:
        for(int y = 0; y < park.length; y++){  
            String [] row = park[y].split("");
            for(int x = 0; x < row.length; x++){
                if(row[x].equals("S")){
                    map.put("x", x);
                    map.put("y", y);
                    break outer;
                }
            }
        }  
    } 
}
```

### 배운 점 && 보완할 점
- 현재 코드에서 시작점을 찾을 때 map에 저장하고 다시 꺼내는 방식인데, 굳이 map을 사용하지 않고 int[]배열에 x,y값을 저장하고 꺼내는 것이 성능적으로 더 효율적이라는 사실을 알게되었다.
- 시작점을 찾는 메서드에서 관성적으로 split("")을 활용하여 문자를 찾고 있었는데, charAt()를 사용하는 것이 성능적으로 더 유리할 것 같다.
- 또한 이중 반복문을 탈출하기 위해 outer: 라벨을 활용하여 만들었는데, 다시보니 그냥 조건문에서 return 해도 됐을 것 같다.
