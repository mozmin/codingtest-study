### 문제 설명 (요약)
- 문자열로 주어진 다트 결과를 해석해 최종 점수를 계산하는 문제이다.
- 각 기회는 점수 + 보너스 + 옵션 구조로 이루어져있다.
- 점수: `0` ~ `10`
- 보너스: `S`: 1제곱, `D`: 2제곱, `T`: 3제곱
- 옵션: `*`: 현재 점수와 직전 점수를 2배, `#`: 현재 점수를 음수로 변경

### 입력
- String dartResult : 점수, 보너스, 옵션이 담겨있는 문자열

### 출력
- 최종 점수의 합계

<hr>

### 핵심 풀이 전략
1. 모든 문자열을 한 글자씩 queue에 넣는다.
2. queue에서 문자를 하나씩 꺼내며 숫자인지 확인한다.
3. 숫자가 1이고, 다음 문자가 0이면 10으로 처리하고 그렇지 않으면 한 자리 숫자로 처리한다.
4. 다음 문자를 꺼내 S,D,T에 따라 제곱 계산을 한다.
5. 다음 문자가 숫자가 아니라면 옵션 *, #인지 확인하고 처리한다.
6. 최종적으로 socres 배열의 합을 반환한다.

---

```java
import java.util.*;

class Solution {

    public int solution(String dartResult) {
        int[] scores = new int[3];
        int scoreCount = -1;

        Deque<Character> queue = new ArrayDeque<>();

        for (int i = 0; i < dartResult.length(); i++) {
            queue.offer(dartResult.charAt(i));
        }

        while (!queue.isEmpty()) {
            char firstCheck = queue.poll();

            if (Character.isDigit(firstCheck)) {

                String number = String.valueOf(firstCheck);

                if (firstCheck == '1' && !queue.isEmpty() && queue.peek() == '0') {
                    number = "10";
                    queue.poll();
                }

                scoreCount++;
                scores[scoreCount] = Integer.parseInt(number);

                char secondCheck = queue.poll();

                if (secondCheck == 'D') {
                    scores[scoreCount] *= scores[scoreCount];
                } else if (secondCheck == 'T') {
                    scores[scoreCount] *= scores[scoreCount] * scores[scoreCount];
                }

                if (!queue.isEmpty() && !Character.isDigit(queue.peek())) {
                    char thirdCheck = queue.poll();

                    if (thirdCheck == '#') {
                        scores[scoreCount] = -scores[scoreCount];
                    } else if (thirdCheck == '*') {
                        scores[scoreCount] *= 2;

                        if (scoreCount > 0) {
                            scores[scoreCount - 1] *= 2;
                        }
                    }
                }
            }
        }

        return scores[0] + scores[1] + scores[2];
    }
}
```

### 삽질 기록 🧠
- Character.isDigit() 메서드가 생각나지 않아서 찾아보게 되었다. 추후 숫자인지 판별할 수 있는 유용한 메서드이니 꼭 기억해야겠다.
