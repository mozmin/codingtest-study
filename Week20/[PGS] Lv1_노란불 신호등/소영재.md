### 문제 설명 (요약)
- 여러 개의 신호등이 있고, 각 신호등은 초록불 -> 노란불 -> 빨간불 순서로 반복된다.
- 각 신호등마다 초록불 시간, 노란불 시간, 빨간불 시간이 주어질 때
- 모든 신호등이 동시에 노란불이 되는 가장 빠른 시간을 구하는 문제이다.
- 만약 그런 시간이 없다면 -1을 반환한다.

### 입력
- int[][] signals : 각 신호등의 초록불, 노란불, 빨간불 시간이 담긴 2차원 배열

### 출력
- 모든 신호등이 동시에 노란불이 되는 가장 빠른 시간

<hr>

### 핵심 풀이 전략
1. 각 신호등의 한 주기를 구한다.
2. 모든 신호등 패턴이 다시 처음으로 돌아오는 시점은 각 주기의 최소공배수이다.
3. 1초부터 totalCycle까지 하나씩 확인한다.
4. 각 시간 t마다 모든 신호등을 확인한다.
5. 현재 시간 t가 해당 신호등의 주기에서 몇 번째 초인지 구한다.
6. 노란불 구간인지 확인하고, 모든 신호등이 노란불이면 해당 시간 t를 반환한다.
7. 끝까지 찾지 못하면 -1을 반환한다.

---

```java
class Solution {
    public int solution(int[][] signals) {
        int totalCycle = 1;

        for (int i = 0; i < signals.length; i++) {

            int green = signals[i][0];
            int yellow = signals[i][1];
            int red = signals[i][2];

            int cycle = green + yellow + red;

            totalCycle = cal(totalCycle, cycle);
        }

        for (int t = 1; t <= totalCycle; t++) {

            boolean allYellow = true;

            for (int i = 0; i < signals.length; i++) {

                int green = signals[i][0];
                int yellow = signals[i][1];
                int red = signals[i][2];

                int cycle = green + yellow + red;

                int pos = t % cycle;

                if(pos == 0){
                    pos = cycle;
                }

                if (!(green < pos && pos <= green + yellow)) {
                    allYellow = false;
                    break;
                }
            }

            if (allYellow) {
                return t;
            }
        }

        return -1;
    }

    //최소 공배수를 구하는 메서드
    public int cal(int a, int b) {
        int x = a;
        int y = b;

        //유클리드 호제법을 활용하여 최소 공약수 구하기
        while (y != 0) {
            int temp = y;
            y = x % y;
            x = temp;
        }

        return a / x * b;
    }
}
```
