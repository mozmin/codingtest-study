### 문제 설명 (요약)
- 2차원 보드에서 같은 문자로 이루어진 2x2 블록을 찾아 제거하는 문제이다.
- 블록이 제거되면 위에 있던 블록들이 아래로 떨어지고, 다시 2x2 블록을 검사한다.
- 더 이상 제거할 수 있는 2x2 블록이 없을 때까지 반복하며, 최종적으로 제거된 블록의 개수를 반환한다.

### 입력
- int m : 보드의 높이
- int n : 보드의 너비
- String[] board : 문자열 배열 형태의 초기 보드 형태

### 출력
- 제거된 블록의 총 개수

<hr>

### 핵심 풀이 전략
1. 문자열 배열 board를 다루기 쉽게 char[][] map으로 변환한다.
2. 매 라운드마다 boolean[][] check를 새로 만들어 제거할 블록을 표시한다.
3. 모든 좌표를 돌면서 현재 위치 기준 오른쪽, 아래, 오른쪽 아래가 같은 문자인지 확인한다.
4. 같은 2x2 블록이면 해당 4칸을 check = true로 표시한다.
5. 탐색이 끝난 뒤 check가 true인 칸은 공백 ''으로 바꾸고 제거 개수를 센다.
6. 제거된 블록이 없다면 반복을 종료한다.
7. 제거된 블록이 있다면 각 열마다 아래에서 위로 탐색하며 블록을 아래로 떨어뜨린다.
8. 더 이상 제거할 블록이 없을 때까지 반복한다.


---

```java
class Solution {
    public int solution(int m, int n, String[] board) {
        int answer = 0;
        char[][] map = new char[m][n];

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                map[i][j] = board[i].charAt(j);
            }
        }

        while (true) {
            boolean[][] check = new boolean[m][n];
            for (int i = 0; i < m - 1; i++) {
                for (int j = 0; j < n - 1; j++) {
                    char block = map[i][j];

                    if (block == ' ') continue;

                    if (map[i][j + 1] == block && map[i + 1][j] == block && map[i + 1][j + 1] == block) {

                        check[i][j] = true;
                        check[i][j + 1] = true;
                        check[i + 1][j] = true;
                        check[i + 1][j + 1] = true;
                    }
                }
            }
            
            int count = 0;
            for (int i = 0; i < m; i++) {
                for (int j = 0; j < n; j++) {
                    if (check[i][j]) {
                        map[i][j] = ' ';
                        count++;
                    }
                }
            }
            
            if (count == 0) break;

            answer += count;
            
            for (int j = 0; j < n; j++) {
                int row = m - 1;

                for (int i = m - 1; i >= 0; i--) {
                    if (map[i][j] != ' ') {
                        map[row][j] = map[i][j];
                        if (row != i) {
                            map[i][j] = ' ';
                        }
                        row--;
                    }
                }
            }
        }

        return answer;
    }
}
```
