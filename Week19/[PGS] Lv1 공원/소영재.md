### 문제 설명 (요약)
- 공원은 2차원 문자열 배열 park로 주어지며, 빈 공간은 "-1"로 표시된다.
- 각 돗자리는 정사각형 형태이며, mats 배열에는 사용할 수 있는 돗자리의 한 변 길이가 주어진다.
- 공원 안에서 사람이나 물건이 없는 빈 공간에 돗자리를 펼칠 수 있을 때, 펼칠 수 있는 돗자리 중 가장 큰 크기를 반환해야 한다.
- 만약 어떤 돗자리도 펼칠 수 없다면 -1을 반환한다.

### 입력
- int mats[] : 돗자리 한 변의 길이가 담긴 정수 배열
- String park[][] : 공원의 상태를 나타내는 2차원 문자열 배열

### 출력
- 펼칠 수 있는 가장 큰 돗자리의 한 변 길이, 없다면 -1을 반환

<hr>

### 핵심 풀이 전략
1. 돗자리를 큰 크기부터 확인하기 위해 mats 배열을 오름차순 정렬한 뒤, 뒤에서부터 순회한다.
2. 공원의 모든 위치를 순회하면서 현재 위치가 빈 공간("-1")이면, 해당 위치를 돗자리의 왼쪽 위 시작점으로 보고 돗자리를 놓을 수 있는지 확인한다.
3. 돗자리가 공원의 범위를 벗어나지 않는지 확인하고, checkMat x checkMat 크기의 모든 칸이 "-1"이닞 확인한다. (돗자리를 놓을 수 있는지)
4. 모든 칸이 비어 있다면 현재 돗자리 크기로 갱신한다.
---

```java
import java.util.*;
class Solution {
    int MaxValue = -1;
    public int solution(int[] mats, String[][] park) {
        
        Arrays.sort(mats);
        for(int i = 0; i < park.length; i++){
            for(int j = 0; j < park[0].length; j++){
                if(park[i][j].equals("-1")){
                    checkPark(mats, park, i, j);
                }
            }
        }
        return MaxValue;
    }
    
    public void checkPark(int[] mats, String[][] park, int x, int y){
        for(int i = mats.length -1; i >= 0; i--){
            int checkMat = mats[i];
            if (x + checkMat > park.length || y + checkMat > park[0].length) {
                continue;
            }
            boolean canMat = true;
            for(int j = 0; j < checkMat; j++){
                for(int k = 0; k < checkMat; k++){ 
                    if(!park[x+j][y+k].equals("-1")){
                    canMat = false;
                    break;
                    }
                }
            }
            if(canMat){
                MaxValue = Math.max(MaxValue, checkMat);
                return;
            }
        }
    }
}
```

### 삽질 기록 🧠
- 공원의 빈자리에 돗자리 길이를 넣을 때 공원의 길이를 넘어서는지 확인을 하지 못해 ArrayIndexOutOfBoundsException 에러가 났었다.
