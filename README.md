# Commit Convention

- feat: 풀이 추가 / 기능성 변경
- refactor: 풀이 개선(로직/가독성) + 결과 동일
- fix: 버그 수정(오답/예외 케이스 해결)
- docs: README/문서 수정
- chore: 설정/폴더 정리
