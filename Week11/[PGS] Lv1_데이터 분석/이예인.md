### 문제 설명
주어진 `data`에서 `ext`에 해당하는 값이 `val_ext`보다 작은 데이터만 추출한 뒤,  <br>
`sort_by`에 해당하는 값을 기준으로 오름차순 정렬하여 반환하는 문제.

### 입력
- `data`: `[code, date, maximum, remain]` 형식의 2차원 배열
- `ext`: 필터링 기준이 되는 문자열
- `val_ext`: 비교 기준 값
- `sort_by`: 정렬 기준이 되는 문자열

### 출력
- 조건을 만족하는 행들만 모아 `sort_by` 기준으로 정렬한 2차원 배열

---

### 핵심 풀이 전략
- 문자열인 `ext`를 실제 열 인덱스로 변환한다.
- `data`를 순회하면서 `ext` 기준 값이 `val_ext`보다 작은 행의 인덱스를 저장한다.
- 저장한 인덱스를 바탕으로 조건을 만족하는 행만 `answer` 배열에 옮긴다.
- `sort_by`도 열 인덱스로 변환한 뒤, `Arrays.sort()`와 람다식을 사용해 해당 열 기준으로 오름차순 정렬한다.

---

### 제출 코드
```java
import java.util.*;
class Solution {
    public int[][] solution(int[][] data, String ext, int val_ext, String sort_by) {
        List<Integer> list = new ArrayList<>();
        int idx = 0;
        if(ext.equals("code")) idx = 0;
        else if (ext.equals("date")) idx = 1;
        else if (ext.equals("maximum")) idx = 2;
        else idx = 3;
        
        for(int i = 0 ; i < data.length; i++){
            if(data[i][idx] < val_ext){
                list.add(i);
            }
        }
        
        int[][] answer = new int[list.size()][4];
        for(int i = 0 ; i < list.size(); i++){
            answer[i] = data[list.get(i)];
        }
        
        int sortIdx = 0;
        if(sort_by.equals("code")) sortIdx = 0;
        else if (sort_by.equals("date")) sortIdx = 1;
        else if (sort_by.equals("maximum")) sortIdx = 2;
        else sortIdx = 3;
        
        final int sIdx = sortIdx;
        Arrays.sort(answer, (a, b) -> Integer.compare(a[sIdx], b[sIdx]));
        
        return answer;
    }
}
```
--- 
### 삽길 기록 🧠 
- 처음에는 answer[i] = list.get(i);처럼 넣으려고 했는데, answer[i]는 int[]이고 list.get(i)는 Integer라서 타입이 맞지 않았다.
- 따라서 list에 저장한 인덱스를 활용해서 data[list.get(i)] 형태로 실제 행 전체를 넣어야 했다.
- 또 정렬할 때 필터 기준 인덱스인 idx를 그대로 쓰면 sort_by 기준이 아니라 ext 기준으로 정렬되는 실수를 했다. 그래서 정렬용 인덱스 sortIdx를 따로 구분했다.
- Arrays.sort()에서 람다식 내부에 사용할 인덱스는 final 또는 사실상 final이어야 해서 final int sIdx = sortIdx;로 따로 선언했다.

### 배운 점 && 보완할 점
```
// 오름차순 정렬
Arrays.sort(answer, (a, b) -> Integer.compare(a[sIdx], b[sIdx]));
```
