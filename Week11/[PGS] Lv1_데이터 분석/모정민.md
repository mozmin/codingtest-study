### 문제 설명 (요약)
주어진 데이터 `data`에서 조건에 맞는 데이터를 뽑아내고, 지정된 기준에 따라 정렬하여 반환하는 문제.
데이터의 각 행은 `[코드 번호(code), 제조일(date), 최대 수량(maximum), 현재 수량(remain)]`으로 구성된다.

- **data**: 정렬되지 않은 2차원 정수 배열 (1 <= data의 길이 <= 500)
- **ext**: 뽑아낼 정보의 기준이 되는 문자열
- **val_ext**: 뽑아낼 정보의 기준값 (정수)
- **sort_by**: 정렬 기준이 되는 문자열
- **조건**: `ext` 값이 `val_ext`보다 작은 데이터만 뽑아낸 후, `sort_by` 값을 기준으로 오름차순 정렬하여 반환한다.

---

### 핵심 풀이 전략
1. 컬럼명(`code`, `date`, `maximum`, `remain`)을 배열의 인덱스(0, 1, 2, 3)와 매핑할 수 있도록 `HashMap`을 선언하고 초기화한다.
2. 매핑된 맵을 활용하여 조건 기준 열의 인덱스(`extIdx`)와 정렬 기준 열의 인덱스(`sortIdx`)를 찾는다.
3. `data` 배열을 순회하며, 각 행의 `extIdx`번째 값이 `val_ext`보다 작은 경우에만 동적 배열(`ArrayList`)에 추가한다.
4. `ArrayList`에 담긴 필터링된 배열들을 `sortIdx`번째 값을 기준으로 오름차순 정렬한다. (람다식 활용)
5. 정렬된 `ArrayList`를 다시 2차원 배열로 변환하여 반환한다.

---

### Java 구현 코드

```java
import java.util.*;

class Solution {
    public int[][] solution(int[][] data, String ext, int val_ext, String sort_by) {
        
        Map<String, Integer> map = new HashMap<>();
        map.put("code", 0);
        map.put("date", 1);
        map.put("maximum", 2);
        map.put("remain", 3);
        
        int[][] answer = Arrays.stream(data)
            .filter(a -> a[map.get(ext)] < val_ext)
            .sorted(Comparator.comparingInt(a -> a[map.get(sort_by)]))
            .toArray(int[][]::new);
        
        return answer;
    }
}
```
