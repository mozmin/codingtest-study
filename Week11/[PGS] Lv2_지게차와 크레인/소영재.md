### 문제 설명 (요약)
- 창고(storage)에는 알파벳으로 표시된 컨테이너가 격자 형태로 쌓여 있다.
- 요청(requests)이 순서대로 주어지며, 각 요청에 따라 컨테이너를 제거한다.
- 요청 종류는 1. 문자열 길이가 1인 경우 -> 지게차 : 외부와 연결된 컨테이너만 제거 가능, 문자열 길이가 2인 경우 -> 크레인 : 해당 종류의 컨테이너를 위치와 관계 없이 모두 제거
- 모든 요청을 처리한 뒤, 남아있는 컨테이너 개수를 구하는 문제이다.

### 입력
- String[] storage : 창고 상태 (2차원 격자)
- String[] requests : 제거 요청 목록

### 출력
- 모든 요청 처리 후 남아있는 컨테이너의 개수를 반환

<hr>

### 핵심 풀이 전략
1. 기존 storage 배열을 (row+2) x (col+2)크기로 확장하고, 바깥은 ' '으로 채움 이를 통해 (0,0)을 실제 외부 공간으로 간주하고 BFS를 시작할 수 있도록 설계
2. requests 하나씩 처리하며 map 상태를 지속적으로 갱신
3. 크레인 요청은 해당 문자와 동일한 모든 컨테이너를 전체 순회하며 즉시 제거
4. 지게차 요청은 BFS를 활용하여 외부 (0,0)에서 시작해 ' '(빈칸)영역을 따라 탐색, 탐색 중 인접한 target 문자를 발견하면 제거 대상에 저장
5. ' '이 아닌 칸의 개수를 세어 반환

---

```java
import java.util.*;

class Solution {
    int[] dx = {-1, 0, 1, 0};
    int[] dy = {0, 1, 0, -1};

    public int solution(String[] storage, String[] requests) {
        int row = storage.length;
        int col = storage[0].length();

        // 바깥을 한 겹 확장한 배열
        char[][] map = new char[row + 2][col + 2];

        // 전체를 빈칸으로 초기화
        for (int i = 0; i < row + 2; i++) {
            Arrays.fill(map[i], ' ');
        }

        // 실제 storage를 (1,1)부터 채움
        for (int i = 0; i < row; i++) {
            for (int j = 0; j < col; j++) {
                map[i + 1][j + 1] = storage[i].charAt(j);
            }
        }

        for (String request : requests) {
            char target = request.charAt(0);

            // 크레인
            if (request.length() > 1) {
                crane(target, map);
            }
            // 지게차
            else {
                jigae(target, map, row + 2, col + 2);
            }
        }

        // 남은 컨테이너 개수 세기
        int answer = 0;
        for (int i = 1; i <= row; i++) {
            for (int j = 1; j <= col; j++) {
                if (map[i][j] != ' ') {
                    answer++;
                }
            }
        }

        return answer;
    }

    // 크레인: 해당 문자 전체 제거
    public void crane(char target, char[][] map) {
        for (int i = 0; i < map.length; i++) {
            for (int j = 0; j < map[i].length; j++) {
                if (map[i][j] == target) {
                    map[i][j] = ' ';
                }
            }
        }
    }

    // 지게차: 외부에서 접근 가능한 target만 제거
    public void jigae(char target, char[][] map, int row, int col) {
        Deque<int[]> queue = new ArrayDeque<>();
        boolean[][] visited = new boolean[row][col];
        List<int[]> removeList = new ArrayList<>();

        // 외부 시작점
        queue.offer(new int[]{0, 0});
        visited[0][0] = true;

        while (!queue.isEmpty()) {
            int[] cur = queue.poll();
            int x = cur[0];
            int y = cur[1];

            for (int d = 0; d < 4; d++) {
                int nx = x + dx[d];
                int ny = y + dy[d];

                if (nx < 0 || nx >= row || ny < 0 || ny >= col) {
                    continue;
                }

                if (visited[nx][ny]) {
                    continue;
                }

                // 빈칸이면 계속 외부 공간 탐색
                if (map[nx][ny] == ' ') {
                    visited[nx][ny] = true;
                    queue.offer(new int[]{nx, ny});
                }
                // target이면 제거 대상으로만 표시
                else if (map[nx][ny] == target) {
                    visited[nx][ny] = true;
                    removeList.add(new int[]{nx, ny});
                }
            }
        }

        // BFS 끝난 후 한 번에 제거
        for (int[] pos : removeList) {
            map[pos[0]][pos[1]] = ' ';
        }
    }
}
```

### 삽질 기록 🧠
```java
import java.util.*;
class Solution {
    int[] dx = {-1, 0, 1, 0};
    int[] dy = {0, 1, 0, -1};
    
    public int solution(String[] storage, String[] requests) {
        int row = storage.length;
        int height = storage[0].length();
        
        char[][] map = new char[row][height];
        
        for(int i = 0; i < row; i++){
            for(int j = 0; j < height; j++){
                map[i][j] = storage[i].charAt(j);
            }
        }
        
        for(int i = 0; i < requests.length; i++){
            Deque<int[]> queue = new ArrayDeque<>();
            boolean[][] visited = new boolean[row][height];
            List<int[]> removeList = new ArrayList<>();
            
            String target = requests[i];
            
            if(target.length() > 1){
                crane(target, map);
                continue;
            }
            
            char jigaeTarget = target.charAt(0);
            
            queue.offer(new int[]{0,0});
            visited[0][0] = true;
            
            while(!queue.isEmpty()){
               int[] currentBlock = queue.poll();
               int x = currentBlock[0];
               int y = currentBlock[1];
               
                for(int d = 0; d < 4; d++){
                    int nx = x + dx[d];
                    int ny = y + dy[d];
                    
                    if(nx < 0 || nx >= row || ny < 0 || ny >= height){
                        continue;
                    }
                    if(visited[nx][ny]){
                        continue;
                    }
                    
                    if(map[nx][ny] == ' '){
                        visited[nx][ny] = true;
                        queue.offer(new int[]{nx, ny});
                    }
                    
                    else if (map[nx][ny] == jigaeTarget) {
                        visited[nx][ny] = true;
                        removeList.add(new int[]{nx,ny});
                    }
                }  
            }
            for(int[] pos : removeList){
                map[pos[0]][pos[1]] = ' ';
            }
        }
        
        int answer = 0;
        for (int i = 0; i < row; i++){
            for(int j = 0; j < height; j++){
                if(map[i][j] != ' '){
                    answer++;
                }
        }
    }
        return answer;
    }
    
    public void crane(String target, char[][] map){
        char craneTarget = target.charAt(0);
        for(int i = 0; i < map.length; i++){
            for(int j = 0; j < map[i].length; j++){
                if (map[i][j] == craneTarget) {
                    map[i][j] = ' ';
                }
            }
        }
    }
}
```
- 처음에는 BFS를 (0,0)부터 탐색을 시작하였다. 하지만 (0,0)은 외부 공간이 아니라 창고 내부의 첫 번째 칸이라는 점을 놓쳤다.
- 외부 공간을 따로 표현하지 않고 내부 좌표 (0,0)을 기준으로 탐색을 시작하면서 실제로는 외부에서 접근 가능한 영역을 제대로 탐색하지 못했다.
