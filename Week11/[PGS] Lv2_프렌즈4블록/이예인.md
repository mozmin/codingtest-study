### 문제 설명
2x2 형태로 같은 블록 4개가 모이면 해당 블록들을 제거하고, 제거된 자리에 위의 블록들이 아래로 떨어진다.  <br>
이 과정을 더 이상 제거할 블록이 없을 때까지 반복한 뒤, 최종적으로 제거된 블록 수를 구하는 문제이다.<br>

### 입력
- `m`: 보드의 높이
- `n`: 보드의 너비
- `board`: 길이 `m`의 문자열 배열

### 출력
- 최종적으로 제거된 블록의 개수

---

### 핵심 풀이 전략
- 문자열 배열을 다루기 편하도록 `char[][]` 형태의 2차원 배열로 변환한다.
- 매 라운드마다 전체 보드를 순회하면서 2x2 같은 문자 블록이 있는지 검사한다.
- 바로 삭제하지 않고 `boolean[][]` 배열에 삭제 대상만 표시한다.
  - 겹쳐서 삭제되는 블록이 있을 수 있으므로 표시만 먼저 해두는 것이 중요하다.
- 탐색이 끝난 뒤 삭제 대상 블록을 `'.'`으로 바꾸고 개수를 센다.
- 이후 각 열마다 아래에서 위로 보면서 빈칸(`'.'`) 위에 있는 블록을 내려준다.
- 더 이상 삭제할 블록이 없으면 반복을 종료한다.

---

### 제출 코드
```java
class Solution {
    public int solution(int m, int n, String[] board) {
        int count = 0;
        // boardC : board를 char array로 변경
        char[][] boardC = new char[m][n];
        for(int i = 0; i < m; i++){
            for(int j = 0 ; j < n; j++){
                boardC[i][j] = board[i].charAt(j);
            }
        }
        boolean finish;
        
        while(true){
            // boardB : 지울 블록 표시. true면 지워야 하는 블록
            boolean[][] boardB = new boolean[m][n];
            finish = false;

            for(int i = 0; i < m-1; i++){
                for(int j = 0 ; j < n-1; j++){
                    if(boardC[i][j] == '.') continue;
                    if(check(i, j, boardC, boardB)){
                        finish = true;
                    }
                }
            }
            
            // 지운 블록이 없는 경우 break! 
            if(finish == false) break;

            // 지울 블록은 .으로 표시
            for(int i = 0 ; i < m; i++){
                for(int j = 0; j < n; j++){
                    if(boardB[i][j] == true){
                        count++;
                        boardC[i][j] = '.';
                    }
                }
            }
            
            // 블록 아래로 내리기
            for(int j = 0; j < n; j++){
                for(int i = m - 1; i >= 0; i--){
                    if(boardC[i][j] == '.'){
                        int k = i - 1;
                        while(k >= 0 && boardC[k][j] =='.') k--;
                        if(k >= 0){
                            boardC[i][j] = boardC[k][j];
                            boardC[k][j] = '.';
                        }                 
                    }
                }
            }  
        }    
        
        return count;
    }
    
    boolean check(int i, int j, char[][] boardC, boolean[][] boardB){
        
        char bb = boardC[i][j];
        if(boardC[i][j+1]== bb && boardC[i+1][j] == bb && boardC[i+1][j+1] == bb){                         
            boardB[i][j] = true;
            boardB[i][j+1] = true;                
            boardB[i+1][j] = true;                   
            boardB[i+1][j+1] = true;
            return true;
        }
        
        return false;     
    }
}
```

### 삽길 기록 🧠
- 처음에는 finish를 check() 메서드의 매개변수로 넘겨서 내부에서 true로 바꾸려고 했다.
- 하지만 자바의 기본형은 값 복사로 전달되기 때문에, 메서드 내부에서 바꿔도 바깥에는 반영되지 않았다.
- 그래서 check()가 boolean을 반환하도록 수정하고, 삭제할 블록이 하나라도 발견되면 finish = true가 되도록 바꿨다.
- boardB를 while문 바깥에서 한 번만 생성하면 이전 라운드의 삭제 표시가 남는 문제가 있었다. 따라서 매 라운드마다 새로 생성하도록 수정했다.
- char 배열에서 x, y 변수 위치가 헷갈렸다.
