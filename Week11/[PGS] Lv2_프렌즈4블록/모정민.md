### 문제 설명 (요약)
카카오프렌즈 캐릭터 블록이 2x2 형태로 4개가 모이면 지워지는 게임이다.
같은 블록 4개가 모인 곳을 찾아 지우고, 빈 공간을 채우기 위해 위에 있는 블록들이 아래로 떨어진다. 
블록이 지워지고 떨어지는 과정이 더 이상 일어날 수 없을 때까지 반복했을 때, 총 몇 개의 블록이 지워졌는지 구하는 문제.

- **m, n**: 판의 높이와 폭 (2 ≦ n, m ≦ 30)
- **board**: 블록의 배치 정보가 담긴 1차원 문자열 배열
- 겹치는 2x2 블록들도 한 번에 지워질 수 있다. (예: 2x3 직사각형 형태로 같은 블록이 모여 있다면 6개가 모두 지워짐)

---

### 핵심 풀이 전략
1. **배열 초기화**: 다루기 쉽도록 1차원 문자열 배열 `board`를 2차원 문자열 배열(`String[][] map`)로 변환한다. 빈 공간은 빈 문자열(`""`)로 처리한다.
2. **반복 구조 (do-while)**: 지워질 블록의 좌표를 담는 `ArrayList`가 탐색 후에도 비어있을 때까지(더 이상 지워질 블록이 없을 때까지) 아래 과정을 반복한다.
3. **블록 파괴 조건 확인 (탐색 및 저장)**:
    - 배열을 순회(`0 ~ m-2`, `0 ~ n-2`)하며 현재 기준 2x2 영역의 문자열이 모두 같고 빈 칸(`""`)이 아니라면, 해당 4곳의 좌표를 모두 `ArrayList`에 추가한다.
    - (이때 겹치는 블록의 좌표가 중복으로 리스트에 들어갈 수 있다.)
4. **블록 지우기 및 카운트**:
    - `ArrayList`에 수집된 좌표들을 순회한다.
    - 겹치는 블록 때문에 동일한 좌표가 여러 번 불릴 수 있으므로, 현재 맵의 해당 좌표가 이미 지워진 상태(`""`)가 아닐 때만 정답 카운트를 1 증가시키고 해당 칸을 `""`로 변경한다.
5. **중력 적용 (블록 내리기)**:
    - `drop` 메서드를 호출하여 빈 공간을 채운다.
    - 배열의 맨 아래 칸들부터 역순(`i = m-1` 부터 `0`까지)으로 탐색한다.
    - 빈 공간(`""`)을 발견하면, 해당 위치에서 위쪽 열(`nextY`)을 순차적으로 탐색하여 가장 처음 만나는 문자열 블록을 현재 빈 공간으로 끌어내리고, 원래 있던 자리는 `""`로 만든다.

### Java 구현 코드

```java
import java.util.*;

class Solution {
    public int solution(int m, int n, String[] board) {
        int answer = 0;
        
        // 배열 초기화
        String[][] map = new String[m][n];
        for(int i = 0; i < map.length; i++){
            for(int j = 0; j < map[0].length; j++){
                map[i][j] = String.valueOf(board[i].charAt(j));
            }
        }
        
        // 4개인 그룹 찾기
        // 2x2 이므로 length-1 만큼만 탐색
        ArrayList<int[]> list = new ArrayList<>();
        
        do{
            list.clear();
            
            for(int i = 0; i < map.length-1; i++){
                for(int j = 0; j < map[0].length-1; j++){
                    
                    String value = map[i][j];
                    if(!value.equals("")
                      && map[i][j+1].equals(value)
                      && map[i+1][j].equals(value)
                      && map[i+1][j+1].equals(value)){
                        list.add(new int[]{i, j});
                        list.add(new int[]{i, j+1});
                        list.add(new int[]{i+1, j});
                        list.add(new int[]{i+1, j+1});
                    }
                }
            }
            
            // 4개인 그룹 카운트 및 제거
            for(int[] l : list){
                int y = l[0];
                int x = l[1];
                
                if(!map[y][x].equals("")){
                    map[y][x] = "";
                    answer++;
                }
            }
            
            // 아래 블록 떨구기
            drop(m, n, map);
            
        }while(!list.isEmpty());
        
        return answer;
    }
    
    public void drop(int m, int n, String[][] map){
        
        // 아래부터 탐색
        for(int i = m-1; i >= 0; i--){
            for(int j = n-1; j >= 0; j--){
                
                // 만약 빈 곳이라면 위에 값을 현재 위치로 이동
                if(map[i][j].equals("")){
                    int nextY = i-1;
                    while(nextY >= 0){
                        if(nextY >=0 && !map[nextY][j].equals("")){
                            map[i][j] = map[nextY][j];
                            map[nextY][j] = "";
                            break;
                        }
                        nextY--;
                    }
                }
            }
        }
    }
}

```

### 삽질 기록 🧠
1. 빈칸("")도 블록으로 인식
- 처음에 "" 대신 null로 했더니 nullpointException이 발생함 -> 그래서 "" 사용
- value = ""인 상태에서도 2x2 조건이 만족됨
- 제거 다 끝난 뒤에도 계속 list가 채워짐
- !value.equals("") 조건 필수
  
2. drop()에서 무한루프 발생
while(i >= 0)에서 i가 변하지 않음
