### 문제 설명 (요약)
- m x n 크기의 보드가 주어지고, 각 칸에는 알파벳 대문자가 하나씩 들어 있다.
- 보드에서 같은 문자가 2x2 형태로 모여 있는 경우, 해당 블록을 제거할 수 있다.
- 블록을 제거하면 그 자리는 빈칸이 되고, 위에 있던 블록들은 아래로 떨어진다.
- 이 과정에서 새로운 2x2 블록이 생길 수 있으며, 더 이상 제거할 수 있는 블록이 없을 때까지 반복한다.
- 최종적으로 제거된 블록의 총 개수를 구하는 문제이다.

### 입력
- int m : 보드의 세로 길이
- int n : 보드의 가로 길이
- String[] board : 블록이 담긴 문자열 배열

### 출력
- 제거된 블록의 총 개수를 반환 (int)
<hr>

### 핵심 풀이 전략
1. 보드를 2차원 배열로 변환한 후, 각 칸을 2x2 블록의 시작점으로 두고 같은 문자가 네 칸에 모두 존재하는지 확인
2. 삭제 대상을 바로 지우지 않고 check 배열에 표시한 뒤, 탐색이 끝난 후 한 번에 삭제 (2x2 블록이 서로 겹칠 수 있기 때문)
3. 삭제가 끝나면 각 열을 기준으로 아래에서부터 다시 채우는 방식으로 블록을 떨어트리고, 더 이상 삭제할 블록이 없을 때까지 과정을 반복
4. 각 반복에서 제거된 블록 수를 누적하여 최종 정답을 반환
---

```java
class Solution {
    public int solution(int m, int n, String[] board) {
        int answer = 0;
        char[][] map = new char[m][n];
        
        //지도 그리기
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                map[i][j] = board[i].charAt(j);
            }
        }
        
        
        while (true) {
            //
            boolean[][] check = new boolean[m][n];
            
            //2x2 블록이 완성되면 check 배열에 표시
            for (int i = 0; i < m - 1; i++) {
                for (int j = 0; j < n - 1; j++) {
                    char block = map[i][j];

                    if (block == ' ') continue;

                    if (map[i][j + 1] == block &&
                        map[i + 1][j] == block &&
                        map[i + 1][j + 1] == block) {

                        check[i][j] = true;
                        check[i][j + 1] = true;
                        check[i + 1][j] = true;
                        check[i + 1][j + 1] = true;
                    }
                }
            }
            
            // check가 true인 대상 블록 카운트 세기
            int count = 0;
            for (int i = 0; i < m; i++) {
                for (int j = 0; j < n; j++) {
                    if (check[i][j]) {
                        map[i][j] = ' ';
                        count++;
                    }
                }
            }
            
            //더 이상 지울 블록이 남지 않았다면 종료
            if (count == 0) break;

            answer += count;
            
            // 각 열마다 블록을 아래로 떨어트리기
            for (int j = 0; j < n; j++) {
                
                // 현재 열에서 채워 넣을 위치 (맨 아래부터 시작)
                int row = m - 1;
                
                // 아래에서 위로 올라가면서 블록을 하나씩 확인
                for (int i = m - 1; i >= 0; i--) {
                    
                    //현재 칸이 빈칸이 아니라면 (블록이 있다면)
                    if (map[i][j] != ' ') {
                        
                        // 현재 블록을 아래쪽 채울 위치(row)에 복사
                        map[row][j] = map[i][j];
                        
                        // 만약 현재 위치 (i)와 채울 위치(row)가 다르다면
                        if (row != i) {
                            
                            //원래 있던 위치를 빈칸으로 변경
                            map[i][j] = ' ';
                        }
                        
                        // 다음 블록은 그 위 칸에 채워야 하므로 채울 위치를 한 칸 위로 이동
                        row--;
                    }
                }
            }
        }

        return answer;
    }
}
```

### 배운 점 && 보완할 점
- 블록을 아래로 떨어트리는 과정은 빈칸을 하나씩 채운다라고 생각하기보다 현재 블록들을 아래쪽으로 압축한다라는 관점으로 접근해야 단순해진다는 것을 배웠다.
