# [Lv2] 미로 탈출

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/159993)

`#구현` `#BFS`

### 제출 일자

### 문제 설명 (요약)
- 문자열 배열로 주어진 미로에서 시작 지점('S')에서 출발하여 먼저 레버('L')를 작동시킨 뒤, 출구('E')까지 도달하는 최소 이동 거리를 구하는 문제
- 이동은 상,하,좌,우 네 방향으로만 가능하며, 벽 X가 있는 칸은 지나갈 수 없다.
### 입력
- String[] maps : 미로 정보가 담긴 문자열 배열, ('S') : 시작 지점, ('L') : 레버 위치, ('E') : 출구, ('X') : 벽, ('O') : 이동 가능한 통로

### 출력
- 시작 지점 ('S')에서 레버 ('L')를 거쳐 출구 ('E')까지 이동하는 최소 거리를 반환
- 만약 레버에 도달할 수 없거나, 레버를 작동한 뒤 출구까지 갈 수 없다면 -1을 반환
<hr>

### 핵심 풀이 전략
1. 문자열 배열 maps를 순회하며 시작 지점 ('S') 좌표를 찾고, char[][] 형태 2차원 배열로 전체 미로를 그린다.
2. BFS를 사용하여 찾은 시작 지점('S')에서 레버 ('L')까지의 최단 거리를 탐색한다.
3. 만약 레버('L')에 도달하면 해당 위치부터 다시 BFS를 사용하여 레버 위치에서 출구 ('E')까지의 최단거리를 구한다.

---

```java
import java.util.*;
class Solution {
    int [] dx = {-1, 0, 1, 0};
    int [] dy = {0, 1, 0 , -1};

    public int solution(String[] maps) {
        int row = maps[0].length();
        int column = maps.length;

        boolean[][] visited = new boolean[column][row];
        char[][] arrayMap = new char[column][row];

        int startX = 0;
        int startY = 0;

        for(int i = 0; i < column; i++){
            for(int j = 0; j < row; j++){
                char c =  maps[i].charAt(j);
                arrayMap[i][j] = c;
                if(c == 'S'){
                    startX = i;
                    startY = j;
                }
            }
        }

        Deque<int[]>queue = new ArrayDeque<>();

        queue.offer(new int[]{startX, startY, 0});

        visited[startX][startY] = true;

        while(!queue.isEmpty()){
            int[] arr = queue.poll();
            int currentX = arr[0];
            int currentY = arr[1];
            int dist = arr[2];
            
            //레버를 만났다면 해당 위치, 거리를 매개변수로 다시 탐색 시작
            if(arrayMap[currentX][currentY] == 'L'){
                return lever(currentX, currentY, dist, arrayMap);
            }

            for(int i = 0; i < 4; i++){
                int nx = currentX + dx[i];
                int ny = currentY + dy[i];
                
                //미로의 범위를 벗어난다면 탐색 불가
                if(nx < 0 || ny < 0 || nx >= column || ny >= row){
                    continue;
                }
                
                //이미 방문한 곳이면 탐색 불가
                if(visited[nx][ny] == true){
                    continue;
                }
                
                //벽에 막혔다면 탐색 불가
                if(arrayMap[nx][ny] == 'X'){
                    continue;
                }
                
                //모든 조건을 통과하면 방문처리 후 다음 위치로 다시 탐색
                visited[nx][ny] =  true;
                queue.offer(new int[]{nx, ny, dist+1});
            }
        }
        //레버를 찾지 못했다면 -1 반환
        return -1;
    }
    
    //레버를 만났을 때 해당 위치부터 다시 탐색하는 메서드
    public int lever (int currentX, int currentY, int dist, char [][] arrayMap){

        int column = arrayMap.length;
        int row = arrayMap[0].length;

        boolean[][] visited = new boolean[column][row];
        visited[currentX][currentY] = true;

        Deque<int[]>queue = new ArrayDeque<>();
        queue.offer(new int[]{currentX, currentY, dist});

        while(!queue.isEmpty()){
            int[] arr = queue.poll();
            int currentLeverX = arr[0];
            int currentLeverY = arr[1];
            int leverDist = arr[2];

            //출구를 만났다면 해당 거리 반환
            if(arrayMap[currentLeverX][currentLeverY] == 'E'){
                return leverDist;
            }

            for(int i = 0; i < 4; i++){
                int nx = currentLeverX + dx[i];
                int ny = currentLeverY + dy[i];
                
                if(nx < 0 || ny < 0 || nx >= column || ny >= row){
                    continue;
                }

                if(visited[nx][ny] == true){
                    continue;
                }
                if(arrayMap[nx][ny] == 'X'){
                    continue;
                }
                visited[nx][ny] =  true;
                queue.offer(new int[]{nx, ny, leverDist+1});
            }
        }
        //레버를 찾았으나, 출구를 찾지 못했다면 -1 반환
        return -1;
    }
}
```

### 삽질 기록 🧠
- 레버 메서드 안에서 visited[][]를 재사용해서 에러가 났었다.
- 단순하게 S -> L -> E 형식으로 생각해서 visited를 재사용 했었는데, S → L 탐색 과정에서 이미 방문 처리된 경로를 L → E 탐색에서도 다시 사용해야 하는 경우가 존재한다는 것을 알게 되었다.
