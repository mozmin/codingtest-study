### 문제 설명 (요약)
미로를 나타내는 문자열 배열 `maps`가 주어진다. 출발 지점(`S`)에서 레버(`L`)를 당긴 후, 출구(`E`)로 나가는 최소 시간을 구하는 문제.
- **maps**: 미로의 상태 (가로, 세로 길이 5 ~ 100)
- `S`: 시작 지점, `E`: 출구, `L`: 레버, `O`: 통로, `X`: 벽
- 레버를 당기지 않으면 출구로 나갈 수 없으며, 레버가 있는 칸으로 이동해야만 문이 열린다. 출구로 나갈 수 없는 경우 `-1`을 반환한다.

---

### 핵심 풀이 전략
1. **경유지 분할**: 레버를 반드시 먼저 거쳐야 하므로 탐색 구간을 두 부분으로 나눈다.
   - 1구간: 출발 지점(`S`) -> 레버(`L`)
   - 2구간: 레버(`L`) -> 출구(`E`)
2. **BFS(너비 우선 탐색)**: 가중치가 없는 2차원 배열의 최단 경로를 구하는 문제이므로 BFS가 적합하다.
3. 코드 중복을 피하기 위해, 시작점과 도착점을 파라미터로 받는 공통 `bfs` 메서드를 구현한다.
4. 1구간(`S` -> `L`)의 최소 거리를 구하고 도달 불가능하면 `-1`을 반환한다.
5. 1구간에 도달했다면, 방문 배열(`visited`)을 새로 초기화하고 2구간(`L` -> `E`)의 최소 거리를 구한다.
6. 두 구간의 거리를 합산하여 반환한다.

---

### Java 구현 코드

```java
import java.util.*;

class Solution {
    
    int[] dy = {0, 0, 1, -1};
    int[] dx = {1, -1, 0, 0};
    public int solution(String[] maps) {
        int answer = 0;
        boolean[][] visited = new boolean[maps.length][maps[0].length()];
        
        int[] startPoint = new int[2];
        int[] leverPoint = new int[2];
        int[] exitPoint = new int[2];
        
        // 배열 초기화
        for(int i = 0; i < maps.length; i++){
            for(int j = 0; j < maps[0].length(); j++){
                switch(maps[i].charAt(j)){
                    case 'S':
                        startPoint[0] = i;
                        startPoint[1] = j;
                        break;
                    case 'L':
                        leverPoint[0] = i;
                        leverPoint[1] = j;
                        break;
                    case 'E':
                        exitPoint[0] = i;
                        exitPoint[1] = j;
                        break;
                    case 'X':
                        visited[i][j] = true;
                        break;       
                }
            }
        }
        
        // start -> Lever
        int leverCost = bfs(visited.clone(), startPoint[0], startPoint[1], leverPoint[0], leverPoint[1]);

        // Lever -> Exit
        int exitCost = bfs(visited.clone(), leverPoint[0], leverPoint[1], exitPoint[0], exitPoint[1]);
        
        if(leverCost == -1 || exitCost == -1) return -1;
        else return leverCost + exitCost;
    }
    
    public int bfs(boolean[][] visited, int y, int x, int endY, int endX){
        
        // visited 배열 깊은 복사
        boolean[][] copy = new boolean[visited.length][];
        for (int i = 0; i < visited.length; i++) {
            copy[i] = visited[i].clone();
        }
        
        // 가로, 세로 길이
        int m = copy.length;
        int n = copy[0].length;
        
        // 큐 초기화
        Deque<int[]> que = new ArrayDeque<>();
        que.offer(new int[]{y, x, 0});
        copy[y][x] = true;
        
        while(!que.isEmpty()){
            int[] poll = que.poll();
            int curY = poll[0];
            int curX = poll[1];
            int curTime = poll[2];
            
            if(curY == endY && curX == endX) return curTime;
            
            for(int i = 0; i < 4; i++){
                int nextY = curY + dy[i];
                int nextX = curX + dx[i];
                
                if(nextY >= 0 && nextY < m && nextX >= 0 && nextX < n && !copy[nextY][nextX]){
                    que.offer(new int[]{nextY, nextX, curTime+1});
                    copy[nextY][nextX] = true;
                }
            }
        }
        return -1;
    }
}
```

### 삽질 기록 🧠
1. visited 배열 그냥 넘기기
- visited 배열을 bfs() 파라미터로 그냥 넘겼는데, 배열은 복사본을 넘기는게 아니라 참조값을 넘겨주는 것이므로 모든 bfs()에서 `원본 visited` 배열을 사욯안다.
- 이 때문에 `start -> lever`를 갈 때 썼던 visited 배열이 그대로 `lever -> exit` bfs()에 영향이 갔다.
- 따라서 bfs()안에 boolean[][] copy 라는 복사본 배열을 만들어 사용하였다.

### 배운 점 && 보완할 점
1. 2차원 배열 복사에 대한 이해 부족
- clone()이 얕은 복사라는 점을 놓쳐 의도치 않은 참조 공유 발생
- **clone()은 `1차 배열`에만 유효한 것임을 알게 되었다.**
- 이후에는 구조(1차원 vs 2차원)에 따라 복사 방법을 구분해야 함
