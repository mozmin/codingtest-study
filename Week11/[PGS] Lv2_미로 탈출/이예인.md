### 문제 설명
맵에서 시작점 `S`에서 레버 `L`를 먼저 당긴 뒤, 출구 `E`까지 이동해야 하는 문제.  
벽 `X`는 지나갈 수 없으며, 각 칸은 상하좌우로 한 칸씩 이동할 수 있다.  
레버를 당기지 않고는 출구로 가더라도 탈출할 수 없기 때문에,  
반드시 `S -> L -> E` 순서의 최단거리를 구해야 한다.

### 입력
- `maps`: 미로의 상태가 문자열 배열로 주어짐
  - `S`: 시작 지점
  - `L`: 레버
  - `E`: 출구
  - `O`: 통로
  - `X`: 벽

### 출력
- 시작점에서 레버를 거쳐 출구까지 도달하는 최소 이동 횟수
- 이동할 수 없다면 `-1`

---

### 핵심 풀이 전략
- 먼저 문자열 배열 `maps`를 2차원 문자 배열 `table`로 변환하면서 시작점 `S`, 레버 `L`, 출구 `E`의 위치를 찾는다.
- 이 문제는 한 번의 BFS로 해결하는 것이 아니라,
  - `S -> L`
  - `L -> E`
  두 번의 BFS를 수행해야 한다.
- BFS에서는 큐에 현재 좌표와 현재까지의 이동 거리(`dist`)를 함께 저장한다.
- 목표 지점에 처음 도달했을 때의 거리가 곧 최단거리이므로 바로 반환한다.
- 두 구간 중 하나라도 도달할 수 없으면 최종적으로 `-1`을 반환한다.

---

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(String[] maps) {
        int[] start = new int[2];
        int[] lever = new int[2];
        int[] exit = new int[2];
        char[][] table = new char[maps.length][maps[0].length()];
        for(int i = 0 ; i < maps.length; i++){
            for(int j = 0 ; j < maps[0].length(); j++){
                table[i][j] = maps[i].charAt(j);
                if(maps[i].charAt(j) == 'S') start = new int[] {i, j};
                if(maps[i].charAt(j) == 'L') lever = new int[] {i, j};
                if(maps[i].charAt(j) == 'E') exit = new int[] {i, j};
            }
        }
        
        int toLever = bfs(start, lever, table);
        if(toLever == -1) return -1;
        
        int toExit = bfs(lever, exit, table);
        if(toExit ==  -1) return -1;
        
        return toLever + toExit;
    }
    
    int bfs(int[] start, int[] target, char[][] table){
        boolean[][] visited = new boolean[table.length][table[0].length];
        int[] dx = {1, -1, 0, 0};
        int[] dy = {0, 0, -1, 1};
        
        Queue<int[]> queue = new LinkedList<>();
        // queue : x좌표, y좌표, dist
        queue.offer(new int[]{start[0], start[1], 0});
        visited[start[0]][start[1]] = true;
        
        while(!queue.isEmpty()){
            int[] now = queue.poll();
            int x = now[0];
            int y = now[1];
            int dist = now[2];
            
            if(x == target[0] && y == target[1]){
                return dist;
            }
            
            for(int i = 0 ; i < 4; i++){
                int nx = x + dx[i];
                int ny = y + dy[i];
                
                if(nx < 0 || ny < 0 || nx >= table.length || ny >= table[0].length) continue;
                if(visited[nx][ny] == true || table[nx][ny] == 'X') continue;
                
                visited[nx][ny] = true;
                queue.offer(new int[]{nx, ny, dist+1});
            }
        }
        return -1;
    }
}
```

### 삽길 기록 🧠 
- 오랜만에 BFS 문제를 풀어서 풀이 방법이 잘 기억이 안났다. 
- 처음에는 dist를 안넣고, 칸 옮길때마다 count를 늘려서 틀렸었다. 
- 또한 if(Arrays.equals(lever, next)) break;처럼 작성하면 for문만 종료되고 BFS 전체가 끝나지 않아 원하는 시점에 탐색을 멈출 수 없다는 점도 놓쳤다.
- 이후 큐에 (x, y, dist) 형태로 거리 정보를 함께 넣어 BFS를 구성하면서 최단거리를 올바르게 구할 수 있었다.
