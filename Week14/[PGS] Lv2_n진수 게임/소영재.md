### 문제 설명 (요약)
- 0부터 시작해서 숫자를 차례대로 N진수로 변환한 뒤, 참가자들이 한 글자씩 순서대로 말한다.
- 이때 튜브가 말해야 하는 순서의 문자 개수를 구하는 제이다.

### 입력
- int n : 사용할 진법
- int t : 튜브가 말해야 하는 숫자의 개수
- int m : 게임에 참가하는 인원
- int p : 튜브의 순서

### 출력
- 튜브가 말해야 하는 문자들을 이어 붙인 문자열을 반환

<hr>

### 핵심 풀이 전략
1. 0~num까지를 모두 N진수 문자열로 변환하고 문자가 변경될 경우를 생각하여 대문자로 변환하여 하나의 문자열로 이어 붙인다.
2. 하나의 문자열로 이어붙인 부분에서 튜브가 말해야 하는 위치의 문자만 추출한다.
3. 최종적으로 튜브가 말해야 하는 문자열을 반환한다.

---

```java
class Solution {
    public String solution(int n, int t, int m, int p) {
        StringBuilder sb = new StringBuilder();
        StringBuilder answer = new StringBuilder();
        
        int num = 0;
        while(num < t * m){
            //num을 n진수로 변경하는 메서드 (반환 타입은 String)
            sb.append(Integer.toString(num, n).toUpperCase());
            num++;
        }
        
        int count = 0;
        for(int i = p-1; count < t; i+=m){
            answer.append(sb.charAt(i));
            count++;
        }
        
        return answer.toString();
    }
}
```

### 배운 점 && 보완할 점
- 숫자를 N진수로 변환하는 메서드에 대해 배웠다.
- Integer.toString(타겟 숫자, 변환할 진수); 형태로 사용한다면 숫자를 N진수로 변환하여 String으로 반환한다는 점을 배웠다.
- 또한 Integer.toString()은 기본적으로 소문자를 반환하기 때문에 toUpperCase()를 활용해야 한다는 점도 배웠다!
