### 문제 설명
총 `n`개의 노드가 있고, 노드들은 서로 연결되어 있다.  <br>
각 연결에는 파이프 타입이 존재하며, 파이프 타입은 `1`, `2`, `3` 중 하나이다.<br>

처음 감염된 노드는 `infection`이다.  <br>
이후 총 `k`번 동안 파이프 타입을 하나씩 선택할 수 있다. <br>

한 번 파이프 타입을 선택하면, 현재 감염된 모든 노드에서 해당 타입의 파이프를 따라 이동할 수 있는 노드들이 모두 감염된다. <br>

이때 `k`번의 선택을 통해 감염시킬 수 있는 노드 수의 최댓값을 구하는 문제이다. <br>


### 입력
- `n`: 전체 노드의 개수
- `infection`: 처음 감염된 노드 번호
- `edges`: 노드 간 연결 정보
  - `edges[i][0]`: 시작 노드
  - `edges[i][1]`: 도착 노드
  - `edges[i][2]`: 파이프 타입
- `k`: 파이프 타입을 선택할 수 있는 횟수

### 출력
- 최대 감염 가능 노드 수

---

### 핵심 풀이 전략
- 그래프를 인접 행렬로 저장한다.
- 처음 감염된 노드를 `visited` 처리한다.
- 매 단계마다 선택할 수 있는 파이프 타입은 `1`, `2`, `3`이다.
- 따라서 DFS로 가능한 모든 파이프 선택 순서를 탐색한다.
- 각 선택마다 BFS를 사용해 해당 파이프 타입으로 갈 수 있는 모든 노드를 감염시킨다.
- 감염 상태는 선택 경로마다 달라지므로 `visited.clone()`을 사용해 복사한다.
- DFS의 모든 경로에서 감염된 노드 수의 최댓값을 갱신한다.

---

### 제출 코드
```java
import java.util.*;

class Solution {
    public int solution(int n, int infection, int[][] edges, int k) {
        int answer = 0;
        int[][] table = new int[n+1][n+1];
        for(int i = 0 ; i < edges.length; i++){
            int a = edges[i][0];
            int b = edges[i][1];
            int c = edges[i][2];
            table[a][b] = c;
            table[b][a] = c;
        }
        
        boolean[] visited = new boolean[n+1];
        visited[infection] = true;
        
        answer = dfs(0, k, n, table, visited);
        
        return answer;
    }
    
    int dfs(int depth, int k, int n, int[][] table, boolean[] visited){
        int max = count(n, visited);
        
        if(depth == k) return max;
        
        for(int i = 1; i <= 3; i++){
            boolean[] next = visited.clone();
            
            bfs(n, table, next, i);
            
            int result = dfs(depth+1, k, n, table, next);
            
            max = Math.max(max, result);
        }
        return max;
    }
    
    void bfs(int n , int[][] table, boolean[] visited, int pipeType){
        Queue<Integer> queue = new LinkedList<>();
        
        for(int i = 1; i <= n; i++){
            if(visited[i]) queue.add(i);
        }
        
        while(!queue.isEmpty()){
            int current = queue.poll();
            
            for(int j =1 ; j <= n; j++){
                if(table[current][j] == pipeType && !visited[j]){
                    visited[j] = true;
                    queue.add(j);
                }
            }
        }
    }
    
    int count (int n, boolean[] visited){
        int count = 0;
        
        for(int i = 1; i <= n; i++){
            if(visited[i]) count++;
        }
        
        return count;
    }
    
}
```
