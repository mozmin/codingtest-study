### 문제 설명
택배 상자가 가로 길이 `w`인 창고에 지그재그 형태로 쌓여 있다.<br>

상자는 1번부터 `n`번까지 차례대로 쌓이며, 한 줄이 가득 차면 다음 줄로 올라간다.  <br>
단, 각 층마다 상자가 놓이는 방향이 번갈아 바뀐다.<br>

예를 들어 한 층은 왼쪽에서 오른쪽으로 쌓이고, 다음 층은 오른쪽에서 왼쪽으로 쌓이는 방식이다.<br>

이때 꺼내려는 상자 번호가 `num`일 때, 해당 상자를 꺼내기 위해 위에 쌓인 상자까지 포함해서 총 몇 개의 상자를 꺼내야 하는지 구하는 문제이다.<br>


### 입력
- `n`: 전체 택배 상자의 개수
- `w`: 한 층에 놓을 수 있는 상자의 개수
- `num`: 꺼내려는 택배 상자의 번호

### 출력
- `num`번 상자를 꺼내기 위해 제거해야 하는 상자의 개수

---

### 핵심 풀이 전략
- 상자를 실제 배열로 만들지 않고, 각 상자의 **층 번호**와 **위치**만 계산한다.
- 전체 상자의 마지막 층을 `n_level`로 계산한다.
- 꺼내려는 상자의 층을 `num_level`로 계산한다.
- `n`과 `num`이 같은 층에 있다면, 위에 쌓인 상자가 없으므로 바로 `1`을 반환한다.
- 각 층이 정방향인지 역방향인지 확인한다.
- 마지막 상자의 위치와 꺼낼 상자의 위치를 비교해서, 맨 위층에 해당 열의 상자가 존재하는지 판단한다.
- 존재한다면 `n_level - num_level + 1`개를 꺼내야 하고, 존재하지 않는다면 `n_level - num_level`개를 꺼내면 된다.

---

### 제출 코드
```java
class Solution {
    public int solution(int n, int w, int num) {
        int n_level = n / w;
        if(n % w > 0) n_level++;
        
        int num_level = num / w;
        if(num % w > 0) num_level++;
        
        if(n_level == num_level) return 1;
        
        boolean isTopReverse = n_level % 2 == 0;
        boolean isNumReverse = num_level % 2 == 0;
        
        int num_place = num % w;
        if(num_place == 0){
            if(!isNumReverse) num_place = 1;
            else num_place = w;
        }
        
        int n_place = n % w;
        
        if(isTopReverse == isNumReverse){
            if(n_place == 0 || n_place >= num_place) return n_level - num_level + 1;
            else return n_level - num_level;
        }
        else{
            if(n_place == 0 || w - n_place <= num_place) return n_level - num_level + 1;
            else return n_level - num_level;
        }
        
    }
}
```
