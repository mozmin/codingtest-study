### 문제 설명 (요약)
- 두 문자열을 각각 두 글자씩 끊어서 다중집합으로 만든다.
- 단, 두 글자가 모두 알파벳인 경우만 사용하고, 대소문자는 구분하지 않는다.
- 그 후 두 다중집합의 자카드 유사도를 구한다. (자카드 유사도 = 교집합 크기 / 합집합 크기)
- 마지막에 자카드 유사도에 65536을 곱한 뒤, 소수점 아래를 버린 값을 반환한다.

### 입력
- String str1 : 첫 번째 문자열
- String str2 : 두 번째 문자열

### 출력
- 두 문자열의 자카드 유사도 * 65336 값을 정수로 반환

<hr>

### 핵심 풀이 전략
1. 문자열을 모두 대문자로 변환한다.
2. 각 문자열을 앞에서부터 두 글자씩 자른 뒤, 두 글자가 모두 알파벳인 경우에만 리스트에 저장한다.
3. list1의 원소를 하나씩 선택하면서 list2에 같은 값이 있는지 확인한다.
4. 이미 교집합으로 사용한 list2 원소는 boolean[] checkString으로 다시 사용하지 않게 막는다.
5. 교집합 크기와 합집합 크기를 구한 뒤 계산한다.

---

```java
import java.util.*;
class Solution {
    public int solution(String str1, String str2) {
        
        List<String> list1 = new ArrayList<>();
        List<String> list2 = new ArrayList<>();
        
        Set<String> set = new HashSet<>();
         
        for(int i = 0; i < str1.length() -1; i++){
            String checkStr1 = str1.substring(i, i+2).toUpperCase();
            if(checkAlphabet(checkStr1.charAt(0)) && checkAlphabet(checkStr1.charAt(1))){
                list1.add(checkStr1);
            }
        }
        
        for(int j = 0; j < str2.length() -1; j++){
            String checkStr2 = str2.substring(j, j+2).toUpperCase();
            if(checkAlphabet(checkStr2.charAt(0)) && checkAlphabet(checkStr2.charAt(1))){
                list2.add(checkStr2);
            }
        }
        boolean[] checkString = new boolean[list2.size()];
        
        int same = 0;
        
        for(int i = 0; i < list1.size(); i++){
            String compareString = list1.get(i);
            for(int j = 0; j < list2.size(); j++){
                if(!checkString[j] && compareString.equals(list2.get(j))){
                    same++;
                    checkString[j] = true;
                    break;
                }
            }
        }
        
        int hap = list1.size() + list2.size() - same;
        
        if(hap == 0){
            return 65536;
        }
        
        return (int)((double)same/hap * 65536);
    }
    
    public boolean checkAlphabet(char c){
        return c >= 'A' && c <= 'Z';
    }
}
```
