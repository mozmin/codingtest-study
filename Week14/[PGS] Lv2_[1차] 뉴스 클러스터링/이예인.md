### 문제 설명
두 문자열 str1, str2가 주어졌을 때, 각 문자열을 두 글자씩 끊어서 다중집합을 만든다.<br>
단, 두 글자가 모두 영문자인 경우만 원소로 인정한다.<br>
이후 두 다중집합의 자카드 유사도를 구한다.<br>

자카드 유사도는 교집합 크기 / 합집합 크기이다.<br>
문제에서는 이 값에 65536을 곱한 뒤, 소수점 아래를 버린 정수를 반환해야 한다.<br>
두 집합이 모두 공집합인 경우에는 자카드 유사도를 1로 정의하므로 65536을 반환한다. <br>

### 입력
- str1: 첫 번째 문자열
- str2: 두 번째 문자열

### 출력
- 두 문자열의 자카드 유사도에 65536을 곱한 후 소수점 아래를 버린 값

---

### 핵심 풀이 전략
- 문자열을 모두 대문자로 변환한다.
- 각 문자열을 두 글자씩 끊는다.
- 두 글자가 모두 영문자인 경우에만 다중집합에 추가한다.
- 일반 Set이 아니라 Map<String, Integer>를 사용한다.
- 이유는 이 문제의 집합이 중복을 허용하는 다중집합이기 때문이다.
- 각 문자열 조각이 몇 번 등장했는지 Map에 저장한다.
- 두 Map의 모든 key를 하나의 Set에 모아 순회한다.
- 교집합은 각 key의 등장 횟수 중 작은 값이다.
- 합집합은 각 key의 등장 횟수 중 큰 값이다.
- 합집합이 0이면 두 집합이 모두 공집합이므로 65536을 반환한다.
- 그렇지 않으면 (교집합 / 합집합) * 65536을 계산한다.

---

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(String str1, String str2) {
        int answer = 0;
        str1 = str1.toUpperCase();
        str2 = str2.toUpperCase();
        
        Map<String, Integer> map1 = new HashMap<>();
        Map<String, Integer> map2 = new HashMap<>();
        Set<String> set = new HashSet<>();
        
        for(int i = 0 ; i < str1.length() - 1; i++){
            char a = str1.charAt(i);
            char b = str1.charAt(i+1);
            if(!Character.isAlphabetic(a) || !Character.isAlphabetic(b)) continue;
            
            StringBuilder sb = new StringBuilder(); 
            sb.append(a); 
            sb.append(b); 
            map1.put(sb.toString(), map1.getOrDefault(sb.toString(), 0) + 1);
            set.add(sb.toString()); 
        }
        
        for(int i = 0 ; i < str2.length() - 1; i++){
            char a = str2.charAt(i);
            char b = str2.charAt(i+1);
            if(!Character.isAlphabetic(a) || !Character.isAlphabetic(b)) continue;
            
            StringBuilder sb = new StringBuilder(); 
            sb.append(a); 
            sb.append(b); 
            map2.put(sb.toString(), map2.getOrDefault(sb.toString(), 0) + 1);
            set.add(sb.toString()); 
        }
        int num1 = 0; // 교집합
        int num2 = 0; // 합집합
        
        for(String key : set){
            int count1 = map1.getOrDefault(key, 0);
            int count2 = map2.getOrDefault(key, 0);
            
            num1 += Math.min(count1, count2);
            num2 += Math.max(count1, count2);
        }
        
        if(num2 == 0) return 65536;
        answer = (int)(((double) num1 / num2) * 65536);
        
        return answer;
    }
}
```
