### 문제 설명 (요약)
- 두 문자열 X, Y에서 공통으로 등장하는 숫자들을 뽑아 가장 큰 정수를 만드는 문제이다.
- 같은 숫자가 여러 번 등장하면, 두 문자열에 공통으로 등장한 회소 횟수만큼 사용할 수 있다.

### 입력
- String X : 첫 번째 문자열
- String Y : 두 번째 문자열

### 출력
- 공통으로 만들 수 있는 가장 큰 수를 반환
- 공통으로 만들 수 있는 숫자가 없다면 -1을 반환

<hr>

### 핵심 풀이 전략
1. X,Y값에 등장하는 숫자를 담을 int 배열을 각각 선언한다.
2. X, Y에 각각 숫자가 몇 번 등장했는지 저장한다.
3. 가장 큰 수를 만들어야 하므로 9부터 0까지 내림차순으로 결과 문자열에 추가한다.
4. 만약 결과 문자열이 비어있다면 -1을 반환하고, 결과 문자열의 첫 글자가 0이면 모든 숫자가 0이라는 뜻이므로 0을 반환한다.
---

```java
import java.util.*;
class Solution {
    public String solution(String X, String Y) {
        int[] arrayX = new int[10];
        int[] arrayY = new int[10];
        StringBuilder sb = new StringBuilder();
        
        for(int i = 0; i < X.length(); i++){
            char numberX1 = X.charAt(i);
            int numberX2 = Integer.valueOf(numberX1 - '0');
            arrayX[numberX2]++;
            
        }
        
        for(int j = 0; j < Y.length(); j++){
            char numberY1 = Y.charAt(j);
            int numberY2 = Integer.valueOf(numberY1 - '0');
            arrayY[numberY2]++;
        }
        
        for(int k = 9; k >= 0; k--){
            int numberCount = Math.min(arrayX[k], arrayY[k]);
            for(int l = 0; l < numberCount; l++){
                sb.append(k);
            }
        }
        
        if (sb.length() == 0) {
            return "-1";
        }

        if (sb.charAt(0) == '0') {
            return "0";
        }
        
        return sb.toString();
    }
}
```

### 삽질 기록 🧠
```java
import java.util.*;
class Solution {
    public String solution(String X, String Y) {
        List<Integer> list = new ArrayList<>();
        StringBuilder sb = new StringBuilder();
        
        for(int i = 0; i < X.length(); i++){
            char x = X.charAt(i);
            for(int j = 0; j < Y.length(); j++){
                char y = Y.charAt(j);
                if(x == y){
                    list.add(Integer.valueOf(y - '0'));
                    break;
                }
            }
        }
        if(list.size()== 0){
            return "-1";
        }
        
        
        for(int k = 0; k < list.size(); k++){
            sb.append(list.get(k));
        }
        
        return sb.toString();
    }
}

```
- 처음에는 리스트에 같은 숫자가 나오면 저장하고 stream으로 중복을 제거한 뒤, 정렬을 하려고 했다.
- 하지만 문제에서 공통으로 나타나는 정수 즉, X와 Y가 같은 값을 가진다고 저장하는 것이 아닌 그 중에서 최소의 짝(?)만 저장하는 문제였다.
- 따라서 해당 풀이 방법을 모두 취소하게 되었다.

### 정답 풀이에서의 삽질 기록
- if (sb.charAt(0) == '0') { return "0"; } 이라는 조건을 추가하지 않아서 오류가 났었다.
- 이유는 만약 공통된 문자열의 개수가 예를 들어 2개라면 "00"이 반환되기 때문이다.
- 따라서 만약 공통된 문자열이 없고, 첫 문자열 시작이 0이라면 0을 반환하도록 수정했다.
