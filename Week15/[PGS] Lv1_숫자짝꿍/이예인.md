- K: 공통으로 사용 가능한 숫자의 개수

### 문제 설명
두 문자열 X와 Y가 주어졌을 때, 두 문자열에 공통으로 등장하는 숫자들을 이용해 만들 수 있는 가장 큰 수를 구하는 문제이다. <br>

각 숫자는 X와 Y에 등장한 횟수만큼만 사용할 수 있으며, 같은 숫자가 여러 번 등장하는 경우 두 문자열에서의 최소 등장 횟수만큼 사용할 수 있다. <br>
 
공통으로 사용할 수 있는 숫자가 없다면 `-1`을 반환하고, 공통 숫자가 모두 `0`이라면 `0`을 반환한다. <br>

### 입력
- 문자열 `X`
- 문자열 `Y`

### 출력
- X와 Y의 공통 숫자로 만들 수 있는 가장 큰 수
- 공통 숫자가 없으면 `-1`
- 결과가 0으로만 이루어져 있으면 `0`

---

### 핵심 풀이 전략
- 문자열 X와 Y를 각각 순회하면서 각 숫자의 등장 횟수를 `HashMap`에 저장한다.
- 동시에 등장한 숫자들을 `HashSet`에 저장하여 이후 공통 숫자 후보로 사용한다.
- `set`을 순회하면서 X와 Y에 모두 존재하는 숫자만 확인한다.
- 두 문자열에서 해당 숫자가 등장한 횟수 중 더 작은 값을 사용 가능 개수로 판단한다.
- 사용 가능한 개수만큼 리스트에 숫자를 추가한다.
- 가장 큰 수를 만들어야 하므로 리스트를 내림차순 정렬한다.
- 리스트가 비어 있으면 공통 숫자가 없으므로 `-1`을 반환한다.
- 내림차순 정렬 후 첫 번째 값이 0이면 모든 값이 0이라는 뜻이므로 `0`을 반환한다.
- 그 외에는 `StringBuilder`를 이용해 숫자들을 문자열로 이어 붙여 반환한다.

---

### 제출 코드
```java
import java.util.*;

class Solution {
    public String solution(String X, String Y) {
        String answer = "";
        HashMap<Integer, Integer> mapX = new HashMap<>();
        HashMap<Integer, Integer> mapY = new HashMap<>();
        HashSet<Integer> set = new HashSet<>();
        for(int i = 0 ; i < X.length(); i++){
            int xx = X.charAt(i) - '0';
            mapX.put(xx, mapX.getOrDefault(xx, 0) + 1);
            set.add(xx);
        }
        
        for(int i = 0 ; i < Y.length(); i++){
            int yy = Y.charAt(i) - '0';
            mapY.put(yy, mapY.getOrDefault(yy, 0) + 1);
            set.add(yy);
        }
        List<Integer> list = new ArrayList<>();
        for(int s : set){
            int countX = mapX.getOrDefault(s, 0);
            int countY = mapY.getOrDefault(s, 0);
            if(countX == 0 || countY == 0) continue;
            int count = Math.min(countX, countY);
            for(int i = 0; i < count; i++){
                list.add(s);
            }
        }
        
        Collections.sort(list, Collections.reverseOrder());
        
        if(list.isEmpty()) return "-1";
        if(list.get(0) == 0) return "0";
        
        StringBuilder sb = new StringBuilder();
        for(int li : list){
            sb.append(String.valueOf(li));
        }
        
        return answer = sb.toString();
    }
}
```

### 삽길 기록 🧠
- 처음에는 문자열을 `Integer.parseInt()`로 숫자로 변환해서 처리하려고 했다.
  - 하지만 X와 Y의 길이가 길 경우 `int` 범위를 초과해 `NumberFormatException`이 발생할 수 있다.
  - 따라서 숫자로 변환하지 않고 문자열을 한 글자씩 순회해야 했다.

 - 결과값이 0인지 확인할 때도 `Integer.parseInt(answer) == 0`으로 검사하려고 했다.
  - 이 역시 결과 문자열이 매우 길 경우 런타임 에러가 발생할 수 있다.
  - 내림차순 정렬 후 첫 번째 숫자가 `0`이면 전체가 0이라는 점을 이용해 `list.get(0) == 0`으로 처리했다.
