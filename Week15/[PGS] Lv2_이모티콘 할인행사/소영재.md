### 문제 설명 (요약)
- 각 이모티콘에 대해 10%, 20%, 30%, 40% 중 하나의 할인율을 적용ㅇ할 수 있다.
- 사용자는 자신이 원하는 최소 할인율 이상의 이모티콘만 구매하며, 구매 금액의 합이 일정 기준 이상이 되면 이모티콘 플러스 서비스에 가입한다.
- 모든 할인율 조합을 탐색하여 이모티콘 플러스 가입자 수를 최대로 만들고, 가입자 수가 같다면 매출액을 최대로 만드는 최적의 할인 정책을 찾아야 한다.

### 입력
- int[][] users : 최소 할인율, 플러스 가입 기준 금액이 담겨있는 2차원 정수 배열
- int[] emoticons : 각 이모티콘의 가격이 담겨있는 1차원 정수 배열

### 출력
- 플러스 가입자 수, 매출액이 담긴 1차원 정수 배열

<hr>

### 핵심 풀이 전략
1. DFS를 이용하여 각 이모티콘의 할인율 조합을 모두 생성한다.
2. 하나의 할인율 조합이 완성될 때마다 모든 사용자의 구매 금액을 계산한다.
3. 구매 금액이 사용자의 가입 기준 금액 이상이면 플러스 가입자로 처리한다.
4. 가입하지 않은 사용자의 구매 금액은 매출액에 더한다.
5. 가입자 수를 우선 비교하고, 가입자 수가 같으면 매출액을 비교하여 최적의 값을 갱신한다.

---

```java (해설 코드 참고)
import java.util.*;

class Solution {
    int[] discount = {10, 20, 30, 40};
    int[] selected;

    int[][] users;
    int[] emoticons;

    int maxJoin = 0;
    int maxSales = 0;

    public int[] solution(int[][] users, int[] emoticons) {
        this.users = users;
        this.emoticons = emoticons;
        this.selected = new int[emoticons.length];

        dfs(0);

        return new int[]{maxJoin, maxSales};
    }

    public void dfs(int depth) {
        if (depth == emoticons.length) {
            calculate();
            return;
        }

        for (int d : discount) {
            selected[depth] = d;
            dfs(depth + 1);
        }
    }

    public void calculate() {
        int joinCount = 0;
        int sales = 0;

        for (int[] user : users) {
            int userDiscount = user[0];
            int userLimit = user[1];

            int total = 0;

            for (int i = 0; i < emoticons.length; i++) {
                if (selected[i] >= userDiscount) {
                    total += emoticons[i] * (100 - selected[i]) / 100;
                }
            }

            if (total >= userLimit) {
                joinCount++;
            } else {
                sales += total;
            }
        }

        if (joinCount > maxJoin) {
            maxJoin = joinCount;
            maxSales = sales;
        } else if (joinCount == maxJoin && sales > maxSales) {
            maxSales = sales;
        }
    }
}
```
