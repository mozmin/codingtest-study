### 성능 요약
메모리: KB, 시간: ms  
시간복잡도: O(4^E × U × E)

- E: 이모티콘 개수
- U: 사용자 수
- 각 이모티콘마다 할인율 4가지(10%, 20%, 30%, 40%)를 모두 시도한다.
- 하나의 할인 조합이 완성될 때마다 모든 사용자와 모든 이모티콘을 확인한다.

### 제출 일자

### 문제 설명
카카오톡에서는 이모티콘 할인 행사를 진행하려고 한다.<br>

각 사용자마다 다음과 같은 기준이 있다.<br>

- 일정 할인율 이상인 이모티콘만 구매한다.
- 구매 금액의 합이 사용자의 기준 금액 이상이면, 이모티콘 구매를 취소하고 이모티콘 플러스 서비스에 가입한다.

목표는 다음 우선순위에 따라 최적의 할인율 조합을 찾는 것이다. <br>

1. 이모티콘 플러스 가입자 수를 최대화한다.
2. 가입자 수가 같다면 이모티콘 판매액을 최대화한다.


### 입력
- `users`: 사용자들의 구매 기준 정보
  - `users[i][0]`: 사용자가 구매를 고려하는 최소 할인율
  - `users[i][1]`: 이 금액 이상 구매하게 되면 이모티콘 플러스에 가입하는 기준 금액
- `emoticons`: 이모티콘의 정가 배열

### 출력
- `[이모티콘 플러스 가입자 수, 이모티콘 판매액]`

---

### 핵심 풀이 전략
- 이 문제는 이모티콘마다 적용할 수 있는 할인율이 10%, 20%, 30%, 40%로 정해져 있다.
- 이모티콘의 개수가 많지 않기 때문에, 가능한 모든 할인율 조합을 DFS로 생성한 뒤 각 조합마다 결과를 계산한다.
- 각 할인 조합이 완성되면 모든 사용자를 확인한다.
- 사용자별로 현재 할인 조합에서 구매 가능한 이모티콘의 합계를 구한다.
  - 이모티콘 할인율이 사용자의 기준 할인율 이상이면 구매한다.
  - 구매 합계가 사용자의 기준 금액 이상이면 플러스 가입자 수를 증가시킨다.
  - 기준 금액 미만이면 해당 금액을 판매액에 더한다.
- 이후 현재 조합의 결과가 기존 정답보다 좋은지 비교한다.
- 비교 기준은 다음과 같다.
  - 플러스 가입자 수가 더 많으면 정답 갱신
  - 플러스 가입자 수가 같고 판매액이 더 크면 정답 갱신

### 제출 코드
```java
import java.util.*;
class Solution {
    
    int[] answer = new int[2];
    int[] discounts = {10, 20, 30, 40};
    int[] selectedDiscounts;
    
    public int[] solution(int[][] users, int[] emoticons) {
        // 이모티콘 개수만큼 할인율을 저장할 배열 생성
        selectedDiscounts = new int[emoticons.length];
        
        dfs(0, users, emoticons);
        
        return answer;
        
    }
    
    void dfs(int depth, int[][] users, int[] emoticons){
        // depth 는 현재 할인율을 정할 이모티콘의 인덱스
        // depth가 emoticons.length가 되면 모든 이모티몬의 할인율을 정한 상태
        if(depth == emoticons.length){
            calculate(users, emoticons);
            return;
        }
        
        for(int discount: discounts){
            selectedDiscounts[depth] = discount;
            dfs(depth + 1, users, emoticons);
        }
    }

    // 완성된 할인율 조합으로 가입자 수와 판매액 계산
    void calculate(int[][] users, int[] emoticons){
        int plusCount = 0; // 플러스 가입자
        int totalSales = 0; // 총 판매액
        
        for(int[] user : users){
            int userMinDiscount = user[0];
            int userLimitPrice = user[1];
            
            int sum = 0; // 현재 사용자가 구매하는 이모티콘들의 총 금액
            
            for(int i = 0 ; i < emoticons.length; i++){
                int discount = selectedDiscounts[i];
                
                if(discount >= userMinDiscount){
                    int discountedPrice = emoticons[i] * (100 - discount) / 100;
                    sum += discountedPrice;
                }
            }

            // 구매 금액이 기준 금액 이상이면 플러스 가입
            if(sum >= userLimitPrice){
                plusCount++;
            }
            else{
                totalSales += sum;
            }
        }

        // 1순위 : 플러스 가입자 수가 더 많으면 정답 갱신
        if(plusCount > answer[0]){
            answer[0] = plusCount;
            answer[1] = totalSales;
        }
        // 2순위: 가입자 수가 같다면 판매액이 더 큰 경우 정답 갱신
        else if(plusCount == answer[0] && totalSales > answer[1]){
            answer[1] = totalSales;
        }
    }
}
```
