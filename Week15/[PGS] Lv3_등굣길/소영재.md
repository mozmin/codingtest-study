### 문제 설명 (요약)
- 집에서 학교까지 가는 최단 경로의 개수를 구하는 문제이다.
- 오른쪽과 아래쪽으로만 이동할 수 있으며, 물웅덩이가 있는 칸은 지나갈 수 없다.

### 입력
- int m : 격자의 가로 길이
- int n : 격자의 세로 길이
- int[][] puddles : 물웅덩이의 위치가 담긴 2차원 정수 배열

### 출력
- 집에서 학교까지 갈 수 있는 최단 경로의 수를 1,000,000,007로 나눈 나머지를 반환

<hr>

### 핵심 풀이 전략
1. dp[i][j]를 (j,i) 위치까지 도달할 수 있는 경우의 수로 정의한다.
2. 물웅덩이는 -1로 표시하고 시작점을 1로 초기화를 한다,
3. 현재 칸까지 오는 방법은 위쪽 칸과 왼쪽 칸에서 오는 경우이므로 dp[i][j] = dp[i-1][j] + dp[i][j-1]로 값을 채운다.
4. 물웅덩이 칸은 지나갈 수 없으므로 0으로 바꾼 뒤 계산하지 않는다.
5. 최종적으로 dp[n][m]을 반환한다.

---

```java
class Solution {
    public int solution(int m, int n, int[][] puddles) {

        int[][] dp = new int[n + 1][m + 1];
        
        for(int[] puddle : puddles){
            dp[puddle[1]][puddle[0]] = -1;
        }

        dp[1][1] = 1;

        for(int i = 1; i <= n; i++){
            for(int j = 1; j <= m; j++){
                if(i == 1 && j == 1){
                    continue;
                }
                if(dp[i][j] == -1){
                    dp[i][j] = 0;
                    continue;
                }
                dp[i][j] = (dp[i-1][j] + dp[i][j-1]) % 1000000007;
            }
        }

        return dp[n][m];
    }
}
```
