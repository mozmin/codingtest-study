### 문제 설명
집에서 학교까지 가는 최단 경로의 개수를 구하는 문제이다.<br>

격자는 `m x n` 크기이며, 이동은 오른쪽과 아래쪽으로만 가능하다.<br>

단, 물에 잠긴 지역인 `puddles`는 지나갈 수 없다.<br>

최종적으로 집 `(1, 1)`에서 학교 `(m, n)`까지 갈 수 있는 최단 경로의 개수를 구해야 한다.<br>

경로의 수가 커질 수 있으므로 결과는 `1,000,000,007`로 나눈 나머지를 반환한다.<br>

### 입력
- `m`: 격자의 가로 길이
- `n`: 격자의 세로 길이
- `puddles`: 물에 잠긴 지역의 좌표 배열
  - 각 좌표는 `[x, y]` 형태로 주어진다.

### 출력
- 집에서 학교까지 갈 수 있는 최단 경로의 개수
- 결과는 `1,000,000,007`로 나눈 나머지를 반환한다.

---

### 핵심 풀이 전략
- 이 문제는 오른쪽과 아래쪽으로만 이동할 수 있다.

- 따라서 어떤 칸 `(y, x)`에 도착하는 방법은 두 가지뿐이다.

1. 위쪽 칸 `(y - 1, x)`에서 내려오는 경우
2. 왼쪽 칸 `(y, x - 1)`에서 오른쪽으로 오는 경우

- 즉, 현재 칸까지 오는 경로의 수는 다음과 같이 계산할 수 있다.
```
dp[y][x] = dp[y - 1][x] + dp[y][x - 1]
```
- 단, 현재 칸이 물웅덩이라면 지나갈 수 없으므로 경로 수를 0으로 처리한다.

- 시작점인 집 (1, 1)은 출발 위치이므로 dp[1][1] = 1로 초기화한다.

- 이후 (1, 1)부터 (n, m)까지 순서대로 탐색하면서 각 칸까지 도달 가능한 경로 수를 누적한다.
---

### 제출 코드
```java
class Solution {
    public int solution(int m, int n, int[][] puddles) {
        int answer = 0;
        int num = 1000000007;
        int[][] dp = new int[n+1][m+1];
        dp[1][1] = 1;
        boolean[][] isPuddle = new boolean[n+1][m+1];
        for(int i = 0 ; i < puddles.length; i++){
            isPuddle[puddles[i][1]][puddles[i][0]] = true;
        }
        
        for(int y = 1; y <= n ; y++){
            for(int x = 1; x <= m; x++){
                if(isPuddle[y][x]){
                    dp[y][x] = 0;
                    continue;
                }
                if(y > 1) dp[y][x] = (dp[y][x] + dp[y-1][x]) % num;
                if(x > 1) dp[y][x] = (dp[y][x] + dp[y][x-1]) % num;
            }
        }
        
        
        return dp[n][m];
    }
}
```
