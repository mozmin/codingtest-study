### 문제 설명
보드 위에 로봇의 시작 위치 `R`, 목표 위치 `G`, 장애물 `D`, 빈 공간 `.`이 주어진다.<br>

로봇은 상하좌우 한 방향으로 이동할 수 있지만, 한 칸씩 멈추는 것이 아니라 장애물이나 벽을 만나기 전까지 계속 미끄러진다.  <br>
즉, 이동을 시작하면 장애물 `D` 또는 보드 경계 바로 앞 칸에서만 멈출 수 있다.<br>

시작 위치 `R`에서 목표 위치 `G`까지 도달하는 데 필요한 최소 이동 횟수를 구해야 한다.  <br>
도달할 수 없다면 `-1`을 반환한다.<br>

### 입력
- 문자열 배열 `board`
  - `R`: 로봇의 시작 위치
  - `G`: 목표 위치
  - `D`: 장애물
  - `.`: 빈 공간

### 출력
- `R`에서 `G`까지 이동하는 최소 횟수
- 도달할 수 없다면 `-1`

---

### 핵심 풀이 전략
- 최소 이동 횟수를 구하는 문제이므로 BFS를 사용한다.
- 먼저 board를 순회하면서 시작 위치 `R`을 찾는다.
- BFS 큐에는 현재 위치와 이동 횟수를 함께 저장한다.
- 일반적인 BFS처럼 한 칸씩 이동하는 것이 아니라, 한 방향을 정하면 벽이나 장애물을 만나기 전까지 계속 이동해야 한다.
- 따라서 각 방향마다 while(true)를 사용해 다음 위치를 계속 확인한다.
- 반복문이 종료되면 nx, ny는 해당 방향으로 미끄러진 뒤 최종적으로 멈출 수 있는 위치가 된다.
- 방문하지 않은 위치라면 방문 처리 후 큐에 넣는다.
- BFS 특성상 처음으로 G에 도착했을 때의 이동 횟수가 최소 이동 횟수이다.
- 큐가 빌 때까지 G에 도달하지 못하면 -1을 반환한다.

### 제출 코드
```java
import java.util.*;
class Solution {
    public int solution(String[] board) {
        int answer = 0;
        int[] R = new int[2];
        int[] G = new int[2];
        Character[][] table = new Character[board.length][board[0].length()];
        for(int i = 0 ; i < board.length; i++){
            for(int j = 0 ; j < board[0].length(); j++){
                table[i][j] = board[i].charAt(j);
                if(table[i][j] == 'R'){
                    R = new int[]{i, j};
                }
            }
        }
        
        int[] dx = {1, -1, 0, 0};
        int[] dy = {0, 0, 1, -1};
        
        boolean[][] visited = new boolean[table.length][table[0].length];
        Queue<int[]> queue = new LinkedList<>();
        queue.add(new int[]{R[0], R[1], 0});
        visited[R[0]][R[1]] = true;
        
        while(!queue.isEmpty()){
            int[] cur = queue.poll();
            int x = cur[0];
            int y = cur[1];
            int count = cur[2];
            
            if(table[x][y] == 'G') return count;
            
            
            for(int i = 0 ; i < 4; i++){
                int nx = x;
                int ny = y;
                
                while(true){
                    int nextX = nx + dx[i];
                    int nextY = ny + dy[i];
                    
                    if(nextX < 0 || nextX >= table.length || nextY < 0 || nextY >= table[0].length) break;
                    
                    if(table[nextX][nextY] == 'D') break;
                    
                    nx = nextX;
                    ny = nextY;
                }
                
                if(!visited[nx][ny]){
                    visited[nx][ny] = true;
                    queue.offer(new int[]{nx, ny, count+1});
                }
            }
        }
        
        
        return -1;
    }
}
```
