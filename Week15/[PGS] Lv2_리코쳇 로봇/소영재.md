### 문제 설명 (요약)
- 로봇이 R 위치에서 출발해서 목표 지점 G까지 도달하는 최소 이동 횟수를 구하는 문제이다.
- 단, 로봇은 한 칸씩 이동하는 것이 아니라 한 방향으로 움직이면서 장애물 D를 만나거나 보드 끝에 닿기 전까지 계속 미끄러진다.

### 입력
- String[] board : 보드가 담겨있는 String 배열, "R" : 시작 위치, "G" : 목표 위치, "D" : 장애물, "." : 빈 공간

### 출력
- R에서 G까지 도달할 수 있는 최소 이동 횟수, 도달할 수 없다면 -1

<hr>

### 핵심 풀이 전략
1. String[] board를 char[][] map 형태의 이차원 char 배열로 변환한다.
2. R의 시작 좌표를 찾는다.
3. BFS를 통해 최소 이동 횟수를 구한다.
4. 상하좌우 4방향을 확인하며, 한 방향을 선택하면 D 또는 벽을 만나기 전까지 계속 이동한다.
5. 이동이 끝난 위치를 아직 방문하지 않았다면 큐에 넣는다.
6. G에 도착하면 현재 이동 횟수를 반환한다.
7. 끝까지 탐색해도 G에 도착하지 못하면 -1을 반환한다.
---

```java
import java.util.*;
class Solution {
    int [] dx = {-1, 0, 1, 0};
    int [] dy = {0, 1, 0, -1};
    public int solution(String[] board) {
        char[][] map = new char[board.length][board[0].length()];
        
        for(int i = 0; i < board.length; i++){
            for(int j = 0; j < board[0].length(); j++){
                map[i][j] = board[i].charAt(j);
            }
        }
        
        int startX = 0;
        int startY = 0;
        
        for(int i = 0; i < map.length; i++){
            for(int j = 0; j < map[0].length; j++){
                if(map[i][j] == 'R'){
                    startX = i;
                    startY = j;
                }
            }
        }
        
        return bfs(map, startX, startY);
        
    }
    public int bfs (char[][] map, int startX, int startY){
        Queue<int[]> queue = new ArrayDeque<>();
        boolean[][] visited = new boolean[map.length][map[0].length];
        
        
        queue.offer(new int[]{startX, startY, 0});
        visited[startX][startY] = true;
        
        while(!queue.isEmpty()){
            int[] currentMap = queue.poll();
            
            int x = currentMap[0];
            int y = currentMap[1];
            int count = currentMap[2];
            
            if(map[x][y] == 'G'){
                return count;
            }
            
            for(int i = 0; i < 4; i++){
                int nx = x;
                int ny = y;
                
                while(true) {
                    int nextX = nx + dx[i];
                    int nextY = ny + dy[i];
                    
                    if(nextX < 0 || nextY < 0 || nextX >= map.length || nextY >= map[0].length || map[nextX][nextY] == 'D'){
                        break;
                    }
                    
                    nx = nextX;
                    ny = nextY;
                }
                
                if(!visited[nx][ny]) {
                    visited[nx][ny] = true;
                    queue.offer(new int[]{nx, ny, count+1});
                }
            }
        }
        return -1;
    }
}
```
