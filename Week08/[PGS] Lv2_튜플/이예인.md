### 문제 설명
문자열 `s`가 여러 집합의 형태로 주어질 때, 이를 바탕으로 원래 튜플을 이루는 숫자의 순서를 구하는 문제이다.

입력 문자열에서 집합들의 순서는 섞여 있을 수 있고, 집합 내부 숫자의 순서도 일정하지 않을 수 있다.  
하지만 집합의 크기를 기준으로 보면 튜플의 순서를 복원할 수 있다.

### 입력
- `s` : 튜플을 표현한 문자열

### 출력
- 원래 튜플을 이루는 숫자를 순서대로 담은 배열

### 핵심 풀이 전략
이 문제는 문자열을 파싱한 뒤, 집합 크기 순서대로 확인하면서 처음 등장한 숫자만 answer에 넣는 방식으로 해결할 수 있다.

1. 문자열의 가장 바깥 `{{`, `}}`를 제거한다.
2. `"},{"`를 기준으로 집합 문자열들을 분리한다.
3. 각 집합을 원소 개수 기준으로 구분하기 위해 `map`에 저장한다.
4. 길이가 1인 집합부터 차례대로 꺼낸다.
5. 각 집합 안 숫자들을 하나씩 확인하면서, 아직 등장하지 않은 숫자만 `answer`에 추가한다.
6. 이미 등장한 숫자인지는 `HashSet`으로 관리한다.

즉, 작은 집합부터 차례대로 보면서 새롭게 등장한 숫자를 하나씩 answer에 넣으면 된다.

### 제출 코드
```java
import java.util.*;
class Solution {
    public int[] solution(String s) {
        HashMap<Integer, String> map = new HashMap<>();
        HashSet<Integer> set = new HashSet<>();
        
        // 끝에 {{ }} 삭제
        s = s.substring(2, s.length() - 2);
        
        // ss는 집합 단위
        String[] ss = s.split("\\},\\{");
        int[] answer = new int[ss.length];

        for(int i = 0; i < ss.length; i++){
            String[] sss = ss[i].split(",");
            // map의 key에는 집합의 길이, value에는 집합 내용이 들어있는 String
            map.put(sss.length, ss[i]);
        }
        
        int idx = 0;
        
        // 길이 1 집합부터 확인
        for(int i = 1 ; i <= ss.length; i++){
            String str = map.get(i); // str = i길이 집합에 있는 집합내용
            // strSplit은 원소 모아둔 배열
            String[] strSplit = str.split(","); // 원소로 쪼개기
            
            for(int j = 0 ; j < strSplit.length; j++){
                int num = Integer.parseInt(strSplit[j]);
                
                if(!set.contains(num)){
                    set.add(num);
                    answer[idx] = num;
                    idx++;
                }
            }         
        } 
        return answer;
    }
}
```
