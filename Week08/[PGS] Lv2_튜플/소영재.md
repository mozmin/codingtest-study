### 문제 설명 (요약)
- 주어진 문자열을 분석하여 집합으로 표현된 튜플의 원소 순서를 구하는 문제

### 입력
- String s : 특정 튜플을 표현하는 집합이 담긴 문자열 ex) "{{2},{2,1},{2,1,3},{2,1,3,4}}"

### 출력
- s가 표현하는 튜플을 int[]에 담아 반환
<hr>

### 핵심 풀이 전략
1. 문자열 s를 가공하기 쉬운 형태로 변환한다.
2. 튜플의 집합의 개수가 적은 순서부터 정렬한다.
3. 정렬된 배열에서 중복을 제거하며 튜플의 원소를 추가한다.

---

```java
import java.util.*;
class Solution {
    public int[] solution(String s) {
        List<Integer> result = new ArrayList<>();
        
        //1. "{{", "}}" 공백으로 처리 ex) "2},{2,1},{2,1,3},{2,1,3,4" 형태로 변경
        s = s.replace("{{", "").replace("}}", "");
        
        //2. "}," 기준으로 잘라서 새로운 배열 생성 ex) ["2", "{2,1", "{2,1,3", "{2,1,3,4"] 
        String[] arr = s.split("},");
        
        //3. "{" 처리 ex) ["2", "2,1", "2,1,3", "2,1,3,4"]
        for (int i = 0; i < arr.length; i++) {
             arr[i] = arr[i].replace("{", "");
        }
        
        // arr의 a,b의 값을 비교하여 음수면 a가 b보다 앞으로, 양수면 b가 a보다 앞으로 정렬됨
        Arrays.sort(arr, (a, b) -> a.split(",").length - b.split(",").length);
        
        for (String str : arr) {
            String[] nums = str.split(",");

            for (String num : nums) {
                int n = Integer.parseInt(num);

                //아직 result에 없는 값만 추가
                if (!result.contains(n)) {
                    result.add(n);
                }
            }
        }

        //List → int[] 변환
        int[] answer = new int[result.size()];
        for (int i = 0; i < result.size(); i++) {
            answer[i] = result.get(i);
        }

        return answer;
        
    }
}
```
