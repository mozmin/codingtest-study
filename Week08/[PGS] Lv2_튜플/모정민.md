### 문제 설명 (요약)
특정 튜플을 표현하는 '집합의 집합'이 문자열로 주어짐
예를 들어 튜플이 (2, 1, 3, 4)라면 집합으로는 {{2}, {2, 1}, {2, 1, 3}, {2, 1, 3, 4}}와 같이 표현

단, 집합 내의 원소 순서와 집합들의 나열 순서는 바뀔 수 있음
주어진 문자열 s를 보고 원래의 튜플을 배열에 담아 반환

- **n** : 숫자의 범위 (1 ~ $n$)
- **q** : 시도한 5개의 숫자 조합들 (2차원 배열)
- **ans** : 각 시도에서 비밀 코드와 일치한 숫자의 개수 (1차원 배열)

### 입력
- s: 집합이 담긴 문자열 (예: "{{20,111},{111}}" )

### 출력
- 원래 튜플을 담은 정수 배열

---

### 핵심 풀이 전략

1. 규칙성 찾기
- 집합의 크기가 1개인 집합의 원소는 튜플의 첫 번째 숫자입니다.
- 집합의 크기가 2개인 집합에서 새로운 원소는 튜플의 두 번째 숫자입니다.
- 즉, 집합의 길이 순서대로 정렬한 뒤, 이전 집합에 없던 숫자를 하나씩 추가하면 그것이 원래 튜플의 순서가 됩니다.

2. 문자열 파싱 (가장 중요!)
- 양 끝의 대괄호 {{ 와 }} 를 제거합니다.
- },{ 를 기준으로 문자열을 자르면 각 집합("2", "2,1" 등)을 얻을 수 있습니다.

3. 중복 제거 (Set 활용)
- 집합을 순회하며 이미 튜플에 추가한 숫자인지 확인하기 위해 HashSet을 사용합니다.

---

### Java 구현 코드

```java
import java.util.*;

class Solution {
    public int[] solution(String s) {

        String[] groups = s.substring(2, s.length() - 2).split("\\}\\,\\{");

        // 길이를 기준으로 오름차순 정렬
        Arrays.sort(groups, (a, b) -> a.length() - b.length());

        int[] answer = new int[groups.length];
        Set<Integer> set = new HashSet<>();
        int index = 0;

        // 각 집합을 순회하며 새로운 숫자를 answer에 담기
        for (String group : groups) {
            String[] nums = group.split(",");
            for (String numStr : nums) {
                int num = Integer.parseInt(numStr);
                
                // set에 없다면
                if (!set.contains(num)) {
                    set.add(num);
                    answer[index++] = num;
                    break;
                }
            }
        }

        return answer;
    }
}

```

### 삽질 기록 🧠

1. 정규표현식(Regex)의 벽
- split("},{")를 그냥 쓰면 에러가 난다. {와 }는 정규표현식에서 특수문자이므로 이스케이프(\\) 처리를 해줘야 한다는 점을 다시 기억해냈다.

2. String 파싱 순서
- 처음에는 replace를 써서 모든 {, }를 제거하고 숫자의 빈도수로 풀까 생각했지만, 문제의 본질(집합의 크기 순서)을 이용하는 것이 훨씬 직관적이라 판단하여 substring 방식을 선택했다.
