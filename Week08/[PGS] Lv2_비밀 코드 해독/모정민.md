### 문제 설명 (요약)
1부터 $n$까지의 숫자 중 서로 다른 5개로 구성된 **비밀 코드**를 알아내야 함
여러 차례의 시도(`q`)와 그에 따른 결과(`ans`, 비밀 코드와 겹치는 숫자의 개수)가 주어질 때, 모든 조건을 만족하는 가능한 비밀 코드의 총 개수를 구하는 문제

- **n** : 숫자의 범위 (1 ~ $n$)
- **q** : 시도한 5개의 숫자 조합들 (2차원 배열)
- **ans** : 각 시도에서 비밀 코드와 일치한 숫자의 개수 (1차원 배열)

### 입력
- `n`: 숫자의 범위 ($10 \le n \le 30$)
- `q`: 시도한 코드 배열 (최대 10개)
- `ans`: 맞힌 개수 배열

### 출력
- 모든 힌트를 만족하는 비밀 코드 조합의 수

---

### 핵심 풀이 전략
1. **완전 탐색 (조합 생성)** $n$개의 숫자 중 5개를 뽑는 경우의 수는 최대 $_{30}C_5 = 142,506$. 이는 1초 내에 모든 경우를 확인하기에 매우 충분한 수치

2. **DFS를 활용한 백트래킹** 1부터 $n$까지의 숫자를 하나씩 확인하며 **'선택하지 않는 경우'**와 **'선택하는 경우'** 두 가지 분기로 나누어 탐색.
   - `count == 5`: 5개를 모두 뽑았을 때만 모든 힌트(`q`, `ans`)와 대조.
   - `num > n`: 모든 숫자를 탐색했음에도 5개를 채우지 못했다면 종료

3. **조건 검증 (Check)** 선택된 5개의 숫자(`picked`)와 각 질문 `q[i]`를 비교하여 겹치는 숫자의 개수가 `ans[i]`와 정확히 일치하는지 확인. 모든 질문을 통과해야만 정답 카운트 증가

---

### Java 구현 코드

```java
import java.util.*;

class Solution {
    private int answer = 0;
    private int n;
    private int[][] queries;
    private int[] expectedAnswers;
    private int[] picked = new int[5]; // 현재 선택한 5개의 숫자 저장

    public int solution(int n, int[][] q, int[] ans) {
        this.n = n;
        this.queries = q;
        this.expectedAnswers = ans;

        // DFS 시작: 1번 숫자부터 탐색, 현재 0개 선택함
        dfs(1, 0);

        return answer;
    }

    private void dfs(int num, int count) {
        // 1. 기저 조건: 5개를 모두 선택한 경우
        if (count == 5) {
            if (check()) answer++;
            return;
        }

        // 2. 기저 조건: n 범위를 벗어나서 더 이상 고를 수 없는 경우
        if (num > n) return;
        
        // 3. 현재 숫자(num)를 선택하지 않고 다음 숫자로 넘어가는 경우
        dfs(num + 1, count);

        // 4. 현재 숫자(num)를 선택하고 다음 숫자로 넘어가는 경우
        picked[count] = num;
        dfs(num + 1, count + 1);
    }

    // 모든 힌트 조건을 만족하는지 검사
    private boolean check() {
        for (int i = 0; i < queries.length; i++) {
            int matchCount = 0;
            
            // 현재 조합(picked)과 질문(queries[i]) 간의 중복 숫자 개수 확인
            for (int p : picked) {
                for (int target : queries[i]) {
                    if (p == target) {
                        matchCount++;
                        break;
                    }
                }
            }
            
            // 힌트의 개수와 일치하지 않으면 오답
            if (matchCount != expectedAnswers[i]) {
                return false;
            }
        }
        return true;
    }
}

```
