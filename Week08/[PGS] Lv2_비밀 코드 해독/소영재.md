### 문제 설명 (요약)
- 1부터 n까지의 정수 중 서로 다른 5개를 오름차순으로 선택한 비밀 코드가 있다.
- 여러 번의 입력 시도 q와 그에 대한 시스템 응답 ans가 주어질 때 비밀 코드의 가능한 조합 개수를 구하는 문제
### 입력
- int n : 정수의 최대값
- int[][] q : 입력한 정수가 담겨있는 2차원 정수 배열
- int[] ans : 시스템의 응답이 담긴 1차원 정수 배열

### 출력
- 비밀 코드로 가능한 정수 조합 개수를 반환
<hr>

### 핵심 풀이 전략
1. n개의 숫자로 만들 수 있는 모든 조합을 생성한다.
2. 각 조합에 대해 모든 q[i]를 검사하면서 해당 조합과 일치하는 숫자의 개수가 ans[]와 같다면 비밀코드로 가능한 조합이다.
3. 마지막으로 가능한 조합의 숫자를 반환한다.
---

```java
class Solution {
    public int solution(int n, int[][] q, int[] ans) {
        
        int count = 0;
        /**
            모든 조합을 만드는 반복문
            a = 첫 번째 숫자 b = 두 번째 숫자 ..
        **/
        for (int a = 1; a <= n - 4; a++) {
            for (int b = a + 1; b <= n - 3; b++) {
                for (int c = b + 1; c <= n - 2; c++) {
                    for (int d = c + 1; d <= n - 1; d++) {
                        for (int e = d + 1; e <= n; e++) {
                            int[] selected = {a, b, c, d, e};

                            if (check(selected, q, ans)) {
                                count++;
                            }
                        }
                    }
                }
            }
        }

        return count;
    }

    //만들어진 조합에 대해 모든 q[i]를 검사하면서 해당 조합과 일치하는 숫자의 개수가 ans[i]와 같은지 검사
    public boolean check(int[] selected, int[][] q, int[] ans) {
        for (int i = 0; i < q.length; i++) {
            int match = 0;
            for (int j = 0; j < 5; j++) {
                for (int k = 0; k < 5; k++) {
                    if (selected[j] == q[i][k]) {
                        match++;
                    }
                }
            }
            //일치하는 개수가 다르다면 false
            if (match != ans[i]) {
                return false;
            }
        }

        return true;
    }
}
```

### 삽질 기록 🧠
- 처음에는 int[][]q 에 해당하는 숫자를 꺼낸 다음 모든 경우의 수를 탐색하며 q[][]랑 ans[] 짝짝쿵해서 빈도가 높은 숫자들로 조합을 찾으려고 했지만
- 문제가 안 풀려서 처음부터 다시 문제를 파악하게 됐다.
