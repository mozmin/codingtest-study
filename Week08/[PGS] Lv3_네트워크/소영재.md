`#DFS`

### 문제 설명 (요약)
- 컴퓨터들이 서로 연결된 네트워크의 개수를 구하는 문제

### 입력
- String n : 컴퓨터의 개수가 담긴 문자열
- int[][] computers : 연결에 대한 정보가 담긴 2차원 배열

### 출력
- 컴퓨터 간 연결되어 있는 네트워크의 총 개수를 반환
<hr>

### 핵심 풀이 전략
1. 방문하지 않은 컴퓨터를 발견하고 DFS를 시작
2. DFS 안에서 해당 컴퓨터와 조건에 따라 연결된 모든 컴퓨터를 방문한다.
3. DFS 호출이 종료되면 네트워크의 개수를 증가시킨다. 
---

```java
import java.util.*;
class Solution {
    boolean[] visited;
    public int solution(int n, int[][] computers) {
        visited = new boolean[n];
        int count = 0;
        
        //방문하지 않은 컴퓨터를 시작점으로 선정
        for (int i = 0; i < n; i++){
            if(!visited[i]){
                //방문하지 않았다면, 해당 시작점을 기준으로 연결된 네트워크 탐색
                dfs(i, computers);
                count++;
            }
        }
           
        return count;
    }
    
    public void dfs(int computer, int[][] computers){
        //방문처리
        visited[computer] = true;
        
        for(int i = 0; i < computers.length; i++){
            if(computers[computer][i] == 1 && !visited[i]){
                dfs(i, computers);
            }
        }
    }
}
```
