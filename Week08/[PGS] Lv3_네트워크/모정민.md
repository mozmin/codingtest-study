### 문제 설명 (요약)
컴퓨터의 개수 $n$과 연결 정보가 담긴 2차원 배열 `computers`가 주어질 때, 연결된 컴퓨터들의 묶음인 **네트워크의 개수**를 구하는 문제

- **n** : 컴퓨터의 개수 (1 ~ 200)
- **computers** : 연결 정보 (i번과 j번이 연결되어 있으면 `computers[i][j]`는 1)
- **네트워크** : 직접 혹은 간접적으로 연결된 컴퓨터들의 집합

### 입력
- `n`: 컴퓨터 개수
- `computers`: 연결 상태를 나타내는 2차원 배열

### 출력
- 네트워크의 총 개수 (정수)

---

### 핵심 풀이 전략
1. BFS를 한 번 수행할 때마다 하나의 네트워크(덩어리)를 완전히 탐색

2. **전체 순회와 방문 체크** - 0번부터 $n-1$번 컴퓨터까지 반복문을 돌며 방문하지 않은(`!visited[i]`) 컴퓨터 탐색
   - 새로운 컴퓨터를 발견하면 `answer++`를 하고 해당 컴퓨터를 시작점으로 BFS 탐색을 시작

3. **BFS 탐색 로직** - `Deque<Integer> que = new ArrayDeque<>()`를 사용하여 큐를 구현
   - 큐에서 현재 컴퓨터를 꺼내고, 그와 연결되어 있으면서 아직 방문하지 않은 다른 컴퓨터들을 모두 큐에 넣으며 방문 처리(`visited[j] = true`)

---

### Java 구현 코드

```java
import java.util.*;

class Solution {
    public int solution(int n, int[][] computers) {
        int answer = 0;
        boolean[] visited = new boolean[n];

        for (int i = 0; i < n; i++) {
            // 아직 방문하지 않은 컴퓨터를 발견하면 새로운 네트워크의 시작
            if (!visited[i]) {
                bfs(i, n, computers, visited);
                answer++; // 네트워크 개수 증가
            }
        }

        return answer;
    }

    private void bfs(int start, int n, int[][] computers, boolean[] visited) {
        // 성능이 좋은 ArrayDeque 사용
        Deque<Integer> que = new ArrayDeque<>();
        
        que.offer(start);
        visited[start] = true;

        while (!que.isEmpty()) {
            int curNetwork = que.poll();

            for (int j = 0; j < n; j++) {
                // 1. 두 컴퓨터가 연결되어 있고(1) 
                // 2. 아직 방문하지 않았으며
                // 3. 자기 자신이 아닌 경우 (사실 computers[i][i]는 1이므로 체크 필수)
                if (computers[curNetwork][j] == 1 && !visited[j] && j != curNetwork) {
                    que.offer(j);    
                    visited[j] = true;
                }
            }
        }
    }
}

```
